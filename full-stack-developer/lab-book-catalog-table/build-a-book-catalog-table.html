Создайте таблицу каталога книг

Создайте приложение, функционально похожее на этот пример проекта. Старайтесь не копировать пример проекта, придайте ему свой собственный стиль.

Цель: Выполнить приведенные ниже пользовательские истории и добиться успешного прохождения всех тестов для завершения лабораторной работы.

Пользовательские истории:

  1. Вам следует создать элемент table, в котором будет отображаться информация о книге.
  2. В вашей таблице должен быть элемент заголовка таблицы, содержащий одну строку.
  3. В строке элемента заголовка таблицы должно быть четыре элемента заголовка таблицы с текстом Название, Автор, Жанр и Год публикации в указанном порядке.
  4. Ваша таблица должна содержать элемент table body, включающий не менее пяти строк.
  5. Каждая строка в основной части таблицы должна содержать четыре элемента данных, отображающие название книги, автора, жанр и год издания.
  6. В вашей таблице должен быть элемент нижнего колонтитула, содержащий одну строку.
  7. В строке нижнего колонтитула вашей таблицы должен быть элемент данных таблицы, занимающий четыре столбца и содержащий текст Всего книг: N, где N следует заменить количеством книг в вашей таблице.

Тесты

  1. Вам потребуется один элемент table.
  2. Внутри элемента table должен быть один элемент thead.
  3. Внутри элемента thead должен быть один элемент tr.
  4. В строке элемента thead должно быть четыре элемента th.
  5. Все элементы th должны содержать следующие слова: Название, Автор, Жанр и Год издания, именно в таком порядке.
  6. Внутри элемента table должен быть один элемент tbody.
  7. Элемент tbody должен содержать не менее пяти строк.
  8. Каждая строка элемента tbody должна содержать ровно четыре дочерних элемента td.
  9. Каждый элемент td в теле таблицы должен содержать текст с информацией о книге.
  10. Внутри элемента table должен быть один элемент tfoot.
  11. В элементе tfoot должен быть ровно один элемент tr.
  12. У элемента td в строке элемента tfoot атрибут colspan должен быть установлен на значение 4.
  13. В строке элемента td элемента tfoot должен быть текст Всего книг: N, где N — количество книг в вашей таблице.
