Создайте страницу с подборкой видео

Создайте приложение, функционально похожее на этот пример проекта. Постарайтесь не копировать пример проекта, а придайте ему свой собственный стиль.

Цель: реализовать приведенные ниже пользовательские истории и пройти все тесты, необходимые для завершения лабораторной работы.

Истории пользователей:

  1. Элемент main должен быть единственным дочерним элементом элемента body.
  2. У вас должен быть элемент h1 с темой вашей страницы.
  3. Под элементом h1 должен быть абзац, описывающий тему вашей страницы.
  4. Под первым абзацем должно быть три элемента section.
  5. Каждый раздел должен содержать элемент h2, абзац и элемент iframe в указанном порядке.
  6. Три элемента iframe должны иметь атрибут src, соответствующий доступному видео.
  7. Каждый элемент iframe также должен иметь атрибут title для описания встроенного содержимого, а также атрибуты height и width для установки правильного размера элемента.

Тесты

  1. Внутри элемента body должен быть элемент main.
  2. Ваш элемент main должен быть единственным дочерним элементом элемента body.
  3. Внутри элемента main должен быть элемент h1 с темой вашей страницы.
  4. Под элементом h1 должен быть абзац, описывающий тему вашей страницы.
  5. Под первым элементом p должно быть три элемента section.
  6. Каждый элемент section должен начинаться с элемента h2, который служит заголовком этого раздела.
  7. Каждый элемент section должен содержать элемент p для представления видеоконтента в качестве своего второго дочернего элемента.
  8. Каждый элемент section должен содержать элемент iframe в качестве третьего дочернего элемента.
  9. Ваш первый элемент iframe должен иметь атрибут src, указывающий на доступное видео.
  10. Ваш первый элемент iframe должен иметь атрибут title для описания встроенного содержимого.
  11. Ваш первый элемент iframe должен иметь атрибут height.
  12. Ваш первый элемент iframe должен иметь атрибут width.
  13. Ваш второй элемент iframe должен иметь атрибут src, указывающий на доступное видео.

  17. Третий элемент iframe должен иметь атрибут src, указывающий на доступное видео.
  18. Третий элемент iframe должен иметь атрибут title для описания встроенного содержимого.
