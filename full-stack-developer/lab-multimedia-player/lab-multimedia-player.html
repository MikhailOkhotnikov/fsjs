Создание мультимедийного проигрывателя

Создайте приложение, функционально похожее на этот пример проекта. Старайтесь не копировать проект целиком, придайте ему свой собственный стиль.

В предыдущих уроках вы познакомились с работой с аудио и видео элементов. В этой лабораторной работе вы создадите мультимедийный плеер, который будет отображать аудиодорожку и видео с текстовым описанием.

Для элемента audio вам потребуется указать элемент source, который используется для определения используемого медиафайла.

Вот пример: 

    
      <audio controls aria-label="описательный ярлык находится здесь">
        <source src="ссылка на аудио находится здесь" type="audio/mpeg">
      </audio>

Элемент source также можно использовать в элементе video следующим образом: 

    
      <video controls width="600" aria-label="описательный ярлык находится здесь">
        <source src="ссылка на mp4 находится здесь" type="video/mp4">
        <!-- Оставшийся код находится здесь -->  
      </video>

Цель: Выполнить приведенные ниже пользовательские истории и добиться успешного прохождения всех тестов для завершения лабораторной работы.

Истории пользователей:

  1. Для основного заголовка страницы следует использовать элемент h1.
  2. Вам следует включить три элемента section.
  3. Внутри первого элемента section должен находиться элемент h2, содержащий название воспроизводимой песни.
  4. Под элементом h2 должен располагаться элемент audio с атрибутами controls и aria-label.
  5. Внутри элемента audio должен быть элемент source с атрибутом src, указывающим на аудиофайл, и атрибутом type. Вы можете использовать этот URL-адрес аудиофайла, если хотите: https://cdn.freecodecamp.org/curriculum/js-music-player/sailing-away.mp3
  6. Внутри второго элемента section должен находиться элемент h2, содержащий заголовок воспроизводимого видео.
  7. Под элементом h2 должен располагаться элемент video с атрибутами controls, width и aria-label.
  8. Внутри элемента video должен находиться элемент source с атрибутом src, указывающим на видеофайл, и атрибутом type. Вы можете использовать следующую ссылку на видео: https://cdn.freecodecamp.org/curriculum/labs/what-is-the-map-method-and-how-does-it-work.mp4
  9. Ниже элемента source должен располагаться элемент track с атрибутом src, указывающим на файл субтитров, а также атрибутами kind, srclang и label. При желании вы можете использовать следующий URL-адрес файла субтитров: https://cdn.freecodecamp.org/curriculum/labs/what-is-the-map-method-and-how-does-it-work.vtt
  10. Внутри третьего элемента section должен быть элемент h2 для заголовка раздела, например, Транскрипт.
  11. Под элементом h2 должен располагаться элемент p с текстом видео.

Тесты

  1. Вам нужен элемент h1.
  2. В заголовке h1 должен быть текст, представляющий собой основной заголовок страницы.
  3. Вам следует включить три элемента section.
  4. Первый элемент section должен содержать элемент h2.
  5. В заголовке h2 должен отображаться текст, представляющий название воспроизводимой песни.
  6. Первый элемент section должен содержать элемент audio.
  7. Ваш элемент audio должен иметь атрибут controls.
  8. Ваш элемент audio должен иметь атрибут aria-label с текстом, описывающим аудио.
  9. Ваш элемент audio должен содержать элемент source.
  10. Элемент source должен иметь атрибут src, указывающий на аудиофайл.
  11. Элемент source должен иметь атрибут type со значением, указывающим тип аудиофайла.
  12. Второй section должен содержать элемент h2.
  13. В заголовке h2 должен отображаться текст, представляющий название воспроизводимого видео.
  14. Второй section должен содержать элемент video.
  15. Ваш элемент video должен иметь атрибут controls.
  16. Ваш элемент video должен иметь атрибут width со значением, указывающим ширину видео.
  17. Ваш элемент video должен содержать элемент source.
  18. Элемент source должен иметь атрибут src, указывающий на видеофайл.
  19. Элемент source должен иметь атрибут type со значением, указывающим тип видео.
  20. Ваш элемент video должен содержать элемент track.
  21. Элемент track должен иметь атрибут src, указывающий на файл субтитров.
  22. Элемент track должен иметь атрибут kind с текстом, описывающим тип файла.
  23. Элемент track должен иметь атрибут srclang с текстом, описывающим язык субтитров.
  24. Элемент track должен иметь атрибут label с текстом, описывающим язык субтитров.
  25. В третьем section должен содержаться элемент h2.
  26. В заголовке h2 должен быть текст, представляющий собой заголовок раздела.
  27. В третьем section должен содержаться элемент p, обозначающий текст транскрипта.
