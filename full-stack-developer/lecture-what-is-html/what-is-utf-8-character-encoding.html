Что такое кодировка символов UTF-8 и зачем она нужна?

UTF-8, или формат преобразования UCS 8, — это стандартизированная кодировка символов, широко используемая в интернете. Кодировка символов — это метод, с помощью которого компьютеры хранят символы в виде данных. По сути, весь текст на веб-странице представляет собой последовательность символов, хранящихся в виде одного или нескольких байтов. В вычислительной технике байт — это единица данных, состоящая из 8 бит, или двоичных цифр. UTF-8 поддерживает все символы из набора символов Unicode, включая буквы и знаки всех письменностей, языков и технические символы. Вот пример использования элемента meta с атрибутом charset для установки кодировки символов UTF-8: 

    
      <meta charset="UTF-8" />

Если установить кодировку символов UTF-8, то символ с диакритическими знаками "e"(é) будет правильно отображаться на странице. Вот расширенный пример кода с использованием кодировки символов UTF-8: 

    
      <!DOCTYPE html>
      <html lang="en">
        <head>
          <meta charset="UTF-8" />
          <meta name="viewport" content="width=device-width, initial-scale=1.0" />
          <title>Примеры кодировки UTF-8</title>
        </head>
        <body>
          <p>Café</p>
        </body>
      </html>

Для каждого нового проекта, который вы создаёте, вы должны включать этот элемент meta с атрибутом charset со значением UTF-8.
