Что такое атрибуты и как они работают?

Посмотрите видео или прочтите стенограмму и ответьте на вопросы ниже.

Стенограмма

Атрибут — это значение, помещаемое внутрь открывающего тега HTML-элемента. Атрибуты предоставляют дополнительную информацию об элементе или определяют, как должен вести себя элемент. Вот базовый синтаксис атрибута: 

    
      <element attribute="value"></element>

После имени атрибута следует знак равенства (=) и значение в кавычках. Значение может быть строкой или числом, в зависимости от атрибута.

Давайте рассмотрим несколько примеров распространённых HTML-атрибутов. Первый пример — атрибут href, который используется для указания URL-адреса ссылки: 

    
      <a href="https://www.example-website.com">Посетите наш веб-сайт</a>

Без этого атрибута ссылка не будет работать, потому что не будет целевого URL-адреса. Поэтому вы должны добавить этот href атрибут, чтобы ссылка работала. Другие распространённые атрибуты — это src или источник, и alt или альтернатива, которые используются для указания источника изображения и предоставления альтернативного описательного текста для изображения соответственно: 

    
      <img src="image.jpg" alt="Красивое изображение" />

Как и атрибут href, атрибут src является обязательным, поскольку он указывает файл изображения, который будет отображаться. Атрибут alt не является обязательным, но рекомендуется для обеспечения доступности. Доступность означает, что все, включая людей с ограниченными возможностями, могут пользоваться и понимать такие вещи, как веб-сайты, приложения и физические пространства. Вы узнаете больше о доступности на следующих лекциях.

Некоторые атрибуты немного отличаются по синтаксису, например, атрибут checked, показанный здесь: 

    
      <input type="checkbox" checked />

В приведённом примере у нас есть элемент input с атрибутом type со значением checkbox. Входные данные используются для сбора информации от пользователей, а атрибут type определяет тип входных данных. В данном случае это флажок. Подробнее о том, как работают входные данные, вы узнаете на следующих лекциях. Атрибут checked используется для указания того, что флажок должен быть установлен по умолчанию. Атрибут checked не требует значения. Если он присутствует, флажок будет установлен по умолчанию. Если атрибут отсутствует, флажок будет снят. Это называется логическим атрибутом. Вы узнаете больше о логических значениях в целом, когда перейдете к разделу JavaScript. Существует несколько распространенных логических атрибутов, с которыми вы столкнетесь в HTML, таких как disabled, readonly и required. Эти атрибуты используются для указания состояния элемента, например, является ли он отключенным, доступным только для чтения или обязательным.

В HTML есть множество атрибутов, которые можно использовать для настройки поведения и внешнего вида элементов на веб-странице. Понимание того, как использовать атрибуты, необходимо для создания интерактивного и доступного веб-контента.
