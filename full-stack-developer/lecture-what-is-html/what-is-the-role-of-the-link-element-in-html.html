Какова роль элемента Link в HTML и как его можно использовать для ссылок на внешние таблицы стилей?

Посмотрите видео или прочтите стенограмму и ответьте на вопросы ниже.

Стенограмма

Давайте узнаем об элементе link и о том, как использовать его для ссылки на внешние таблицы стилей.

Элемент link используется для ссылок на внешние ресурсы, такие как таблицы стилей и значки сайта. Вот базовый синтаксис для использования элемента link для внешнего CSS-файла: 

    
      <link rel="stylesheet" href="./styles.css" />

Атрибут rel используется для указания связи между связанным ресурсом и HTML-документом. В данном случае нам нужно указать, что этот связанный ресурс является stylesheet.

Считается, что лучше всего разделять HTML и CSS в разных файлах. Разработчики будут использовать элемент link для внешнего CSS-файла вместо того, чтобы писать всё в HTML-документе.

Атрибут href используется для указания местоположения URL-адреса внешнего ресурса.

В этом примере dot(точка) с косой чертой в конце указывает компьютеру искать файл styles.css в текущей папке или каталоге.

Элемент link должен быть помещён внутрь элемента head, как показано в следующем примере: 

    
      <head>
        <meta charset="UTF-8" />
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
        <title>Пример элемента link (ссылка)</title>
        <link rel="stylesheet" href="./styles.css" />
      </head>

Часто в профессиональных кодовых базах можно увидеть несколько элементов link со ссылками на разные таблицы стилей, шрифты и иконки. Вот пример использования элемента link для ссылки на внешний шрифт Google под названием Playwright Cuba: 

    
      <link rel="preconnect" href="https://fonts.googleapis.com" />
      <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin />
      <link href="https://fonts.googleapis.com/css2?family=Playwrite+CU:wght@100..400&display=swap" rel="stylesheet" />

Google Fonts — это набор бесплатных пользовательских шрифтов с открытым исходным кодом, которые вы можете использовать в любом проекте. Вы можете выбрать, какие шрифты вы хотите использовать, и Google предоставит вам необходимый код HTML и CSS. В этом примере значение preconnect для атрибута rel указывает браузеру на необходимость создать предварительное соединение со значением, указанным в атрибуте href. Это делается для ускорения загрузки внешних ресурсов.

Другой распространённый вариант использования элемента link — создание ссылок на значки. Вот пример ссылки на значок: 

    
      <link rel="icon" href="favicon.ico" />

favicon, что означает «любимый значок», — это небольшой значок, который обычно отображается на вкладке браузера рядом с названием сайта. Многие сайты используют favicon для отображения значка своего бренда.
