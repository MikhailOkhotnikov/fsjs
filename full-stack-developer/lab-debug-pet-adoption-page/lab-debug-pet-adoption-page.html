Отладка страницы Pet Adoption

Салли, владелица магазина по приему домашних животных, создала свою первую веб-страницу, но есть несколько проблем.

Ваша задача — исправить все ошибки, чтобы Салли могла продолжить работу над своей страницей.

Цель: реализовать приведенные ниже пользовательские истории и пройти все тесты, необходимые для завершения лабораторной работы.

Истории пользователей:

  1. Салли хочет использовать изображение кошек, но оно отображается некорректно. Вам нужно исправить следующее в элементе img:
    • Замените атрибут href на правильный атрибут источника изображения.
    • Замените атрибут att правильным атрибутом, представляющим собой краткий описательный текст для изображений.
    • Удалите закрывающий тег </img>, поскольку элементы img являются пустым элементом и не имеют закрывающих тегов.
  2. Салли хочет использовать ссылки для перенаправления пользователей на страницы о собаках и кошках. Но ссылки работают некорректно. Вам нужно исправить следующее в элементах a:
    • Замените оба атрибута src правильными атрибутами, используемыми для указания URL-адресов.
Файл index.html 
      
        <h1>Добро пожаловать в XYZ Pet Adoption!</h1>
        <p>Подумайте о том, чтобы завести питомца уже сегодня. У нас есть кошки, собаки, кролики и другие.</p>

        <h2>Посмотрите на наших кошек!</h2>
        <img href="https://cdn.freecodecamp.org/curriculum/cat-photo-app/cats.jpg" att="Пять кошек осматривают поле."></img>

        <h2>Возьмите кошку из приюта!</h2>
        <a src="/cats">Посетите страницу кошек</a>

        <h2>Возьмите собаку из приюта!</h2>
        <a src="/dogs">Посетите страницу собак</a>

Тесты

  1. Ваш элемент img должен иметь атрибут src вместо атрибута href.
  2. Ваш элемент img должен иметь атрибут alt вместо несуществующего атрибута att.
  3. Элемент img не должен иметь закрывающего тега </img>.
  4. Ваш элемент a с текстом Посетите страницу кошек должен иметь атрибут href вместо атрибута src.
  5. Ваш элемент a с текстом Посетите страницу собак должен иметь атрибут href вместо атрибута src.
