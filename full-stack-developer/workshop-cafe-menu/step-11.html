Шаг 11

Теперь, когда у вас есть CSS-код в файле styles.css, удалите элемент style и всё его содержимое. После его удаления текст, который был центрирован, снова сместится влево.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8" />
            <title>Меню кафе</title>
            <style>
              h1, h2, p {
                text-align: center;
              }              
            </style>
          </head>
          <body>
            <main>
              <h1>КАФЕ НА КОЛЕСАХ</h1>
              <p>Основано в 2020 г.</p>
              <section>
                <h2>Кофе</h2>
              </section>
            </main>
          </body>
        </html>
Файл styles.css 
      
        h1, h2, p {
          text-align: center;
        }
