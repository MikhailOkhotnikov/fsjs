Шаг 10

Вы оформили три элемента, написав CSS-стили внутри тегов style. Это работает, но поскольку стилей будет гораздо больше, лучше поместить все стили в отдельный файл и создать на него ссылку.

Создайте отдельный файл styles.css. Вы можете переключаться между файлами с помощью вкладок в верхней части редактора.

Для начала перепишите стили, которые вы создали, в файл styles.css. Не забудьте исключить открывающие и закрывающие теги style.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8" />
            <title>Меню кафе</title>
            <style>
              h1, h2, p {
                text-align: center;
              }              
            </style>
          </head>
          <body>
            <main>
              <h1>КАФЕ НА КОЛЕСАХ</h1>
              <p>Основано в 2020 г.</p>
              <section>
                <h2>Кофе</h2>
              </section>
            </main>
          </body>
        </html>
