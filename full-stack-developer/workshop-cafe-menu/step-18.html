Комментарии в CSS выглядят так:

Пример кода 
      
        /* комментарий здесь */

В вашей таблице стилей закомментируйте строку, содержащую свойство background-color и его значение, чтобы увидеть эффект стилизации только элемента #menu. Это снова сделает фон белым.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Меню кафе</title>
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div id="menu">
              <main>
                <h1>КАФЕ НА КОЛЕСАХ</h1>
                <p>Основано в 2020 г.</p>
                <section>
                  <h2>Кофе</h2>
                </section>
              </main>
            </div>
          </body>
        </html>
Файл styles.css 
      
        body {
          background-color: burlywood;
        }
        h1, h2, p {
          text-align: center;
        }
        #menu {
          width: 300px;
        }
