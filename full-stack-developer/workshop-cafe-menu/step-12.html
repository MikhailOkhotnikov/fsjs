Шаг 12

Теперь необходимо связать файл styles.css, чтобы стили применялись снова. Внутри элемента head добавьте элемент link. Присвойте ему атрибут rel со значением stylesheet и атрибут href со значением styles.css.

Примечание: элемент link является пустым элементом, то есть у него нет закрывающего тега. Пустые элементы следует обозначать как <link>, а не <link></link>.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8" />
            <title>Меню кафе</title>

          </head>
          <body>
            <main>
              <h1>КАФЕ НА КОЛЕСАХ</h1>
              <p>Основано в 2020 г.</p>
              <section>
                <h2>Кофе</h2>
              </section>
            </main>
          </body>
        </html>
Файл styles.css 
      
        h1, h2, p {
          text-align: center;
        }
