Шаг 24

Поскольку основным товаром кафе является кофе, в качестве фона страницы можно использовать изображение кофейных зерен.

Удалите комментарий и его содержимое из селектора типа body. После этого добавьте свойство background-image и установите его значение равным url(https://cdn.freecodecamp.org/curriculum/css-cafe/beans.jpg).

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Меню кафе</title>
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div class="menu">
              <main>
                <h1>КАФЕ НА КОЛЕСАХ</h1>
                <p>Основано в 2020 г.</p>
                <section>
                  <h2>Кофе</h2>
                </section>
              </main>
            </div>
          </body>
        </html>
Файл styles.css 
      
        body {

          background-color: burlywood;

        }
