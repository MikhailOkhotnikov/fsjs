Шаг 13

Чтобы оформление страницы на мобильных устройствах выглядело одинаково как на настольном компьютере или ноутбуке, необходимо добавить элемент meta со специальным атрибутом content.

Вы уже изучили мета-элемент viewport в предыдущих уроках.

Вот пример:

Пример кода 
      
        <meta name="viewport" content="width=device-width, initial-scale=1.0" />
Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8" />
            
            <title>Меню кафе</title>
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <main>
              <h1>КАФЕ НА КОЛЕСАХ</h1>
              <p>Основано в 2020 г.</p>
              <section>
                <h2>Кофе</h2>
              </section>
            </main>
          </body>
        </html>
Файл styles.css 
      
        h1, h2, p {
          text-align: center;
        }
