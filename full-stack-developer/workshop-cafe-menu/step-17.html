Теперь задача состоит в том, чтобы div не занимал всю ширину страницы. Свойство width в CSS идеально подходит для этого.

С помощью селектора id можно выбрать конкретный элемент, имеющий атрибут id.

В предыдущих уроках вы научились работать с селектором id следующим образом:

Пример кода 
      
        #cat {
          width: 250px;
        }

Используйте селектор #menu, чтобы задать ширину элемента в 300 пикселей(300px).

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Меню кафе</title>
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div id="menu">
              <main>
                <h1>КАФЕ НА КОЛЕСАХ</h1>
                <p>Основано в 2020 г.</p>
                <section>
                  <h2>Кофе</h2>
                </section>
              </main>
            </div>
          </body>
        </html>
Файл styles.css 
      
        body {
          background-color: burlywood;
        }
        h1, h2, p {
          text-align: center;
        }
