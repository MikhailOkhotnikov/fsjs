Шаг 36

Это сработало, но справа от цены всё ещё остаётся небольшое пустое пространство.

Вы можете продолжать экспериментировать с различными процентами для ширины. Вместо этого используйте клавишу Backspace, чтобы переместить элемент p с классом price рядом с элементом p с классом flavor так, чтобы они находились на одной строке в редакторе. Убедитесь, что между двумя элементами нет пробела.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Меню кафе</title>
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div class="menu">
              <main>
                <h1>КАФЕ НА КОЛЕСАХ</h1>
                <p>Основано в 2020 г.</p>
                <section>
                  <h2>Кофе</h2>
                  <article class="item">
                    <p class="flavor">Французская ваниль</p>
                    <p class="price">130</p>
                  </article>
                  <article>
                    <p>Карамельный маккиато</p>
                    <p>150</p>
                  </article>
                  <article>
                    <p>Тыквенный пряный</p>
                    <p>135</p>
                  </article>
                  <article>
                    <p>Фундуковый</p>
                    <p>140</p>
                  </article>
                  <article>
                    <p>Мокко</p>
                    <p>145</p>
                  </article>
                </section>
              </main>
            </div>
          </body>
        </html>
Файл styles.css 
      
        body {
          background-image: url(https://cdn.freecodecamp.org/curriculum/css-cafe/beans.jpg);
        }

        h1, h2, p {
          text-align: center;
        }

        .menu {
          width: 80%;
          background-color: burlywood;
          margin-left: auto;
          margin-right: auto;
        }

        .item p {
          display: inline-block;
        }

        .flavor {
          text-align: left;
          width: 49%;
        }

        .price {
          text-align: right;
          width: 49%;          
        }
