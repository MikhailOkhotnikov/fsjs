Шаг 9

Теперь у вас есть три селектора типа с одинаковым стилем. Вы можете добавить одну и ту же группу стилей ко многим элементам, создав список селекторов. Каждый селектор разделяется запятыми следующим образом:

Пример кода 
      
        selector1, selector2 {
          property: value;
        }

Удалите три существующих селектора типа и замените их одним списком селекторов, который центрирует текст для элементов h1, h2 и p.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8" />
            <title>Меню кафе</title>
            <style>
              h1 {
                text-align: center;
              }
              h2 {
                text-align: center;
              }
              p {
                text-align: center;
              }
            </style>
          </head>
          <body>
            <main>
              <h1>КАФЕ НА КОЛЕСАХ</h1>
              <p>Основано в 2020 г.</p>
              <section>
                <h2>Кофе</h2>
              </section>
            </main>
          </body>
        </html>
