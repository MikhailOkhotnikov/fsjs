Шаг 8

На предыдущем шаге вы использовали селектор типа(type selector) для оформления элемента h1. Выровняйте содержимое элементов h2 и p по центру, добавив для каждого из них новый селектор типа в существующем элементе style.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8" />
            <title>Меню кафе</title>
            <style>
              h1 {
                text-align: center;
              }

            </style>
          </head>
          <body>
            <main>
              <h1>КАФЕ НА КОЛЕСАХ</h1>
              <p>Основано в 2020 г.</p>
              <section>
                <h2>Кофе</h2>
              </section>
            </main>
          </body>
        </html>
