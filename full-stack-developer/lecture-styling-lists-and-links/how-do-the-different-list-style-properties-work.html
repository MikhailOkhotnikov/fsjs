Как работают различные свойства списков?

В CSS свойство list-style используется для управления внешним видом списков на веб-странице.

Независимо от того, работаете ли вы с упорядоченными списками (ordered lists, ol) или неупорядоченными списками (unordered lists, ul), свойство list-style позволяет настроить способ отображения элементов списка.

Свойство list-style на самом деле является сокращенной записью для трех других свойств:

  • list-style-type
  • list-style-position
  • list-style-image

Каждый из них играет свою роль в определении внешнего вида списков.

Свойство list-style-type позволяет определить тип маркера или номера, используемого в списке.

Для неупорядоченных списков можно выбрать один из нескольких стилей маркировки, например, в виде дисков (discs), кругов (circles) или квадратов (squares).

Для упорядоченных списков можно использовать различные системы нумерации, такие как десятичные (decimal), римские цифры (Roman numerals) или даже буквенные обозначения (alphabetical characters).

Вот пример использования list-style-type:

    
      <ul style="list-style-type: square;">
        <li>Пункт 1</li>
        <li>Пункт 2</li>
        <li>Пункт 3</li>
      </ul>

В этом примере маркеры неупорядоченного списка заменены на квадраты.

Свойство list-style-type используется чаще всего из трёх, поскольку оно напрямую влияет на внешний вид маркированного или нумерованного списка.

Свойство list-style-position управляет положением маркера или номера относительно содержимого элемента списка. Можно использовать два значения: inside и outside.

При использовании значения outside маркер или номер отображаются за пределами текста, что является поведением по умолчанию.

А если вы используете значение inside, то маркер или номер отображаются внутри содержимого, что может привести к переносу текста и выравниванию его по маркеру или номеру.

Вот пример использования свойства list-style-position: 

    
      <ul style="list-style-position: inside;">
        <li>Пункт 1</li>
        <li>Пункт 2</li>
        <li>Пункт 3</li>
      </ul>
      <ul style="list-style-position: outside;">
        <li>Пункт 4</li>
        <li>Пункт 5</li>
        <li>Пункт 6</li>
      </ul>

В этом примере для двух разных тегов неупорядоченного списка указаны значения как внутри (inside), так и снаружи (outside).

Свойство list-style-position может быть полезно, когда вам нужно контролировать выравнивание содержимого списка, особенно в ситуациях, когда в одном элементе списка содержится несколько строк текста.

Свойство list-style-image позволяет использовать изображение в качестве маркера для элементов списка. Это может быть полезно для придания спискам уникального визуального стиля.

Вот пример использования list-style-image: 

    
      <head>
        <style>
          ul {
            list-style-image: url('https://cdn.freecodecamp.org/platform/universal/freecodecamp-org-gravatar.jpeg');
            list-style-position: inside;
          }
        </style>
      </head>
      <body>
        <ul>
          <li>Пункт 1</li>
          <li>Пункт 2</li>
          <li>Пункт 3</li>
        </ul>
      </body>

В этом примере маркированные списки заменены пользовательским логотипом freeCodeCamp, что добавляет списку индивидуальности.

При использовании свойства list-style-image убедитесь, что выбранное изображение небольшое и соответствует дизайну вашей веб-страницы. Если изображение слишком большое или сложное, это может затруднить чтение списка.

Вы можете объединить три свойства — list-style-type, list-style-position и list-style-image — в одно сокращенное свойство list-style.

Порядок значений в сокращенной записи не имеет значения, но все три можно указать одновременно.

Вот пример использования сокращенного свойства: 

    
      <ul style="list-style: square inside url('https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg');">
        <li>Пункт 1</li>
        <li>Пункт 2</li>
        <li>Пункт 3</li>
      </ul>

В этом примере элементы списка используют квадратные маркеры, расположенные внутри содержимого, с пользовательским изображением в качестве маркера.

Однако, если изображение недоступно или не отображается, в качестве запасного варианта будут использоваться квадратные маркеры.
