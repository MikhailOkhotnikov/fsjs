Как расположить элементы списка с помощью внешнего отступа или межстрочного интервала?

Внешний отступ и межстрочный интервал необходимы для размещения элементов списка, чтобы улучшить читаемость и визуальную привлекательность.

Для начала давайте начнём расставлять элементы списка с помощью внешнего отступа!

Внешние отступы (margin) можно использовать для создания пространства между элементами списка, применяя свойства отступов к элементам li. Этот метод позволяет контролировать расстояние за пределами каждого элемента списка, эффективно увеличивая или уменьшая промежуток между ними.

Рассмотрим пример неупорядоченного списка (unordered list) с тремя пунктами. 

    
      <ul>
        <li>Пункт 1</li>
        <li>Пункт 2</li>
        <li>Пункт 3</li>
      </ul>

По умолчанию в HTML не используется большой интервал между элементами списка.

Чтобы добавить внешние отступы к нижней части каждого элемента списка, можно использовать свойство margin-bottom следующим образом:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <ul>
          <li>Пункт 1</li>
          <li>Пункт 2</li>
          <li>Пункт 3</li>
        </ul>
Файл styles.css 
      
        li {
          margin-bottom: 40px;
        }

В этом примере к нижней части каждого элемента списка внутри неупорядоченного списка будет добавлен отступ в 40px.

Ещё один способ расположить элементы списка на некотором расстоянии друг от друга — использовать свойство line-high.

Свойство line-height регулирует вертикальное расстояние между строками текста в пределах одного элемента списка.

Хотя это в первую очередь влияет на расстояние между строками текста внутри каждого элемента, это также может косвенно влиять на общее расстояние между элементами списка, если элементы содержат только одну строку текста.

Если элементы списка содержат несколько строк текста, параметр line-heading повлияет на расстояние между этими строками, но он не изменяет напрямую расстояние между отдельными элементами списка.

Для управления расстоянием между отдельными элементами списка используются свойства margin или padding.

Используя тот же неупорядоченный список, что и ранее, вот пример применения межстрочного интервала (line-height) к элементам списка:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <ul>
          <li>Пункт 1</li>
          <li>Пункт 2</li>
          <li>Пункт 3</li>
        </ul>
Файл styles.css 
      
        li {
          line-height: 2; 
        }

В этом примере параметр line-height: 2; устанавливает высоту строки равной удвоенному размеру шрифта, создавая больше вертикального пространства внутри каждого элемента списка.
