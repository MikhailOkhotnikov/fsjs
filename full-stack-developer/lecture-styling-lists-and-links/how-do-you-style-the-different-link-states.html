Как оформить различные состояния ссылок?

Ссылка может находиться в разных состояниях, включая link, visited, hover, focus и active. Эти состояния важны для того, чтобы помочь пользователям распознавать ссылки и предоставлять четкую обратную связь после взаимодействия, что улучшает как удобство использования, так и доступность.

Оформление различных состояний ссылок имеет решающее значение для удобства использования и доступности, поскольку оно предоставляет визуальные подсказки о текущем состоянии ссылки. Это помогает пользователям понять, какие ссылки они посетили, с какой ссылкой взаимодействуют и что произойдет при нажатии.

Для пользователей с нарушениями зрения или когнитивными расстройствами эти различные стили могут значительно упростить и сделать более интуитивно понятной навигацию.

Кроме того, четкое отображение состояний ссылок улучшает общее взаимодействие с пользователем, обеспечивая мгновенную обратную связь о действиях пользователя, уменьшая путаницу и улучшая навигацию по сайту.

Эти состояния можно стилизовать с помощью так называемых псевдоклассов (pseudo-classes) в CSS.

Псевдокласс — это ключевое слово, добавляемое к селектору, которое определяет особое состояние выбранного элемента.

Например, свойство :hover может изменить цвет кнопки при наведении на неё указателя мыши, а свойство :visited — цвет ссылки, которая уже была посещена.

Псевдоклассы позволяют стилизовать элементы в зависимости от их состояния или взаимодействия пользователя с ними, без необходимости дополнительной разметки в HTML-коде.

Синтаксис псевдокласса выглядит примерно так, где A — селектор, а :B — псевдокласс: 

    
      A:B {
        property: value;
      }

Чтобы лучше понять, как оформлять различные состояния ссылок, давайте рассмотрим несколько примеров.

Псевдокласс :link стилизует непосещенные ссылки, указывая на то, что они кликабельны.

Вот пример использования псевдокласса :link для обращения к элементу привязки:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <a href="/">Пример ссылки</a>
Файл styles.css 
      
        /* Нормальное состояние (непросмотренная ссылка) */
        a:link { 
          color: red;
        }

В приведенном выше примере стандартный синий цвет ссылки изменится на красный (red), пока она не будет посещена.

:visited стилизует ссылки, которые уже были посещены или которые были нажаты, помогая пользователям отслеживать, по каким ссылкам они переходили ранее. Вот пример использования псевдокласса :visited:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <a href="https://www.freecodecamp.org/learn/" target="_blank>freeCodeCamp</a>
Файл styles.css 
      
        /* Посещенная ссылка */
        a:visited {
          color: green;
        }

Этот код окрасит ссылку в зеленый (green) цвет при нажатии.

:hover изменяет стиль ссылки при наведении на неё курсора, визуально указывая на интерактивность ссылки. Вот пример использования псевдокласса :hover:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <a href="/">Пример ссылки</a>
Файл styles.css 
      
        /* Состояние при наведении курсора */
        a:hover {
          color: green;
        }

Этот код окрасит ссылку в зеленый цвет при наведении на неё курсора мыши.

:focus добавляет стили вокруг ссылки, когда она находится в фокусе, например, при навигации с помощью клавиатуры или для повышения доступности.

Вот пример использования свойства outline для применения сплошной оранжевой обводки при фокусировке ссылки.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <a href="/">Пример ссылки</a>
Файл styles.css 
      
        /* Состояние фокуса */
        a:focus {
          outline: 2px solid orange;
        }

:active изменяет стили ссылки во время клика, обеспечивая мгновенную обратную связь пользователю о том, что его действие регистрируется. Вот пример использования псевдокласса :active:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <a href="/">Пример ссылки</a>
Файл styles.css 
      
        /* Активное состояние */
        a:active {
          color: pink;
        }

В этом примере кода ссылка мгновенно окрасится в розовый (pink) цвет при нажатии на неё.
