В чём заключается специфичность селекторов типов?

Селекторы типов, также известные как селекторы элементов, выбирают элементы на основе их имени тега.

Эти селекторы являются основополагающими в CSS и позволяют применять стили ко всем экземплярам определенного HTML-элемента.

Селекторы типов просты в использовании и записываются просто как имя тега элемента, который вы хотите стилизовать.

Вот пример селектора типа, который применяется ко всем элементам абзаца на странице:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <p>Первый абзац</p>
        <p>Второй абзац</p>
        <p>Третий абзац</p>
Файл styles.css 
      
        p {
          color: blue;
        }

В этом примере цвет текста всех элементов p будет установлен на синий (blue).

Селекторы типов имеют относительно низкую специфичность по сравнению с другими селекторами. Значение специфичности для селектора типа равно (0, 0, 0, 1).

Это означает, что селекторы типов будут переопределены селекторами классов, селекторами идентификаторов и встроенными стилями, но стили по-прежнему могут применяться, если отсутствуют правила более высокой специфичности.

Рассмотрим пример, в котором селекторы классов будут переопределять стили селектора типа.

Вот пример с двумя элементами абзаца: 

    
      <p class="para">Я — абзац</p>
      <p class="para">Вот ещё один абзац.</p>

Оба элемента абзаца имеют класс с именем para.

Внутри CSS-файла селектор типа нацелен на абзацы(p), а селектор класса — на элементы с классом para.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <p class="para">Я — абзац</p>
        <p class="para">Вот ещё один абзац.</p>
Файл styles.css 
      
        p {
          color: blue;
        }

        .para {
          color: red;
        }

Все абзацы на странице с классом para будут иметь красный (red) цвет текста вместо синего(blue), поскольку селекторы классов обладают большей специфичностью, чем селекторы типов.

Вопросы

Каково значение специфичности селектора типа (например, div)?




Какой из следующих вариантов обладает меньшей специфичностью, чем селектор типа?




Учитывая следующий CSS-код, какой цвет будет у текста?

        
          <head>
            <style>
              * {
                color: blue;
              }
              p {
                color: red;
              }
            </style>
          </head>
          <body>
            <p>Этот текст</p>
          </body>
