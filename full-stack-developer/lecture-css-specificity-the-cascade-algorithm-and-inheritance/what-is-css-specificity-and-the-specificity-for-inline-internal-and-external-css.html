Что такое специфичность CSS, и какова специфичность для встроенных, внутренних и внешних CSS-стилей?

Специфичность CSS — это фундаментальное понятие, определяющее, какие стили применяются к элементу, когда могут применяться несколько правил.

Понимание специфики помогает разработчикам разрешать конфликты между различными правилами CSS и гарантирует последовательное применение желаемых стилей.

Специфичность CSS рассчитывается на основе типа используемых селекторов.

Наибольшую специфичность обеспечивают встроенные стили, которые применяются непосредственно к элементу через атрибут style.

В этом примере первый абзац будет красным, а остальные элементы p — синими. Это связано с тем, что встроенные стили обладают большей специфичностью, чем селекторы типов, такие как селектор p, показанный в файле styles.css.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <p style="color: red;">Красный абзац</p>
        <p>Другой абзац</p>
        <p>Еще один абзац</p>
        <p>И еще один абзац</p>
Файл styles.css 
      
        p {
          color: blue;
        }

Таким образом, селекторы id обладают высокой степенью специфичности.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <p id="para"">Красный абзац</p>
        <p>Другой абзац</p>
        <p>Еще один абзац</p>
        <p>И еще один абзац</p>
Файл styles.css 
      
        #para {
          color: red; 
        }

        p {
          color: blue;
        }

Далее в дело вступают селекторы классов, селекторы атрибутов и псевдоклассы.

Примерами могут служить селекторы классов, такие как .container и .button, селекторы атрибутов, например [type="text"], и псевдоклассы, такие как :hover. Они обладают умеренной степенью специфичности.

Примечание: Подробнее о псевдоклассах вы узнаете на следующих уроках.

В этом примере первый абзац будет красным, поскольку селектор id имеет более высокую специфичность, чем селекторы классов и типов. Элементы .example-para будут иметь зеленый цвет, а остальные элементы абзаца — синий.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <p id="para"">Красный абзац</p>
        <p class="example-para>Другой абзац</p>
        <p class="example-para>Еще один абзац</p>
        <p>И еще один абзац</p>
Файл styles.css 
      
        #para {
          color: red; 
        }

        .example-para {
          color: green;
        }

        p {
          color: blue;
        }

Селекторы типов, такие как div и h1, и псевдоэлементы, такие как ::before и ::after, обладают меньшей специфичностью по сравнению с предыдущими группами.

        <link rel="stylesheet" href="styles.css">

        <p id="para"">Красный абзац</p>
        <p>Другой абзац</p>
        <p>И еще один абзац</p>
Файл styles.css 
      
        #para {
          color: red; 
        }

        p {
          color: blue;

Наконец, универсальный селектор, обозначенный звездочкой *, обладает самой низкой специфичностью из всех.

Вот пример установки красного цвета для всех элементов с помощью селектора *. Однако вы не увидите ни одного красного элемента, поскольку существуют селекторы ID и типа, которые переопределяют эти стили, что подчеркивает низкую специфичность селектора *.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <p id="para"">Красный абзац</p>
        <p>Другой абзац</p>
        <p>И еще один абзац</p>
Файл styles.css 
      
        * {
          color: red;
        }

        #para {
          color: green; 
        }

        p {
          color: blue;
        }

Значения специфичности рассчитываются как четырехкомпонентная величина (a, b, c, d):

  • а: Встроенные стили (1 или 0).
  • b: Количество селекторов идентификаторов.
  • c: Количество селекторов классов, селекторов атрибутов и псевдоклассов.
  • d: Количество селекторов типов, псевдоэлементов и универсальных селекторов.

Каждая составляющая значения специфичности имеет разный вес:

  • Встроенные стили, или Inline styles (a) имеют наибольший вес, внося значение 1 в первую часть значения специфичности.
  • Селекторы идентификаторов, или ID selectors (b) имеют следующий по величине вес, при этом каждый идентификатор вносит 1 во вторую часть значения специфичности.
  • Селекторы классов, селекторы атрибутов и псевдоклассы, или Class selectors, attribute selectors, and pseudo-classes (c) имеют умеренный вес, при этом каждый из них вносит 1 в третью часть значения специфичности.
  • Селекторы типов и псевдоэлементы, или Type selectors and pseudo-elements (d) имеют наименьший вес, при этом каждый из них вносит 1 в четвертую часть значения специфичности.
  • Универсальный селектор, или Universal selector (*): Универсальный селектор не влияет на расчет специфичности и не изменяет ее 0. Его включение в селектор не меняет значение специфичности.

Встроенные CSS-стили(Inline CSS) обладают наибольшей специфичностью, поскольку применяются непосредственно к элементу. Они переопределяют любые внутренние или внешние CSS-стили. Значение специфичности для встроенных стилей равно (1, 0, 0, 0).

Вот ещё один пример встроенного стиля: 

    
      <p style="color: red;">Этот текст красный.</p>

Внутренние CSS-стили(Internal CSS) определяются внутри элемента style в разделе head HTML-документа. Они имеют меньшую специфичность, чем встроенные стили. И внутренние, и внешние CSS-стили имеют одинаковый уровень специфичности — специфичность определяется используемыми селекторами, а не местом определения CSS. Если два правила имеют одинаковую специфичность, то побеждает то, которое находится позже в документе. Таким образом, если после внутреннего блока стилей подключен внешний файл стилей, приоритет имеют внешние правила. Если же он подключен раньше, то побеждают внутренние правила.

Значение специфичности для внутренних стилей определяется используемыми селекторами. Например, селектор ID во внутренних CSS имеет значение специфичности (0, 1, 0, 0). 

    
      <head>
        <style>
          #text {
            color: blue;
          }
        </style>
      </head>
      <body>
        <div id="text">Этот текст синего цвета.</div>
      </body>

В этом примере текст будет синим, если не будет применен встроенный стиль или более специфический селектор.

Внешние CSS-стили(External CSS) подключаются через элемент link в разделе head и пишутся в отдельных файлах .css. Как и внутренние CSS-стили, специфичность внешних стилей определяется используемыми селекторами. Внешние CSS-стили обеспечивают наилучшую поддержку для крупных проектов.

Значение специфичности для внешних стилей также определяется используемыми селекторами. Например, селектор класса во внешнем CSS имеет значение специфичности (0, 0, 1, 0).

Файл index.html 
      
        <head>
          <link rel="stylesheet" href="styles.css">
        </head>
        <body>
          <div class="text">Цвет этого текста задаётся во внешнем CSS-файле.</div>
        </body>
Файл styles.css 
      
        .text {
          color: purple;
        }

В этом примере цвет текста определяется в файле styles.css и будет применяться, если не будет переопределен более конкретным селектором или встроенным стилем.
