В чём заключается специфичность селекторов идентификаторов?

Селекторы идентификаторов являются одними из самых мощных селекторов в CSS, позволяя разработчикам применять стили к конкретным элементам с уникальными идентификаторами.

Это делает их очень эффективными для целенаправленного воздействия на отдельные элементы, нуждающиеся в уникальном оформлении.

Селекторы идентификаторов определяются символом решетки (#), за которым следует имя идентификатора. Они должны быть уникальными в пределах HTML-документа, то есть два элемента не должны иметь одинаковый идентификатор.

Вот пример использования селектора по идентификатору:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <p id="unique">Пример абзаца</p>
        <p>Ещё один абзац</p>
        <p>И ещё один абзац</p>
Файл styles.css 
      
        #unique {
          color: purple;
        }

В этом примере элементу с идентификатором unique будет присвоен фиолетовый (purple) цвет текста.

Селекторы идентификаторов обладают очень высокой специфичностью, выше, чем селекторы типов и классов, но ниже, чем встроенные стили. Значение специфичности для селектора по идентификатору равно (0, 1, 0, 0).

Это означает, что селекторы идентификаторов могут переопределять селекторы по классу и селекторы по типу, но могут быть переопределены встроенными стилями.

            <style>
              #unique {
                color: purple;
              }
              .highlight {
                color: green;
              }
              p {
                color: blue;
              }
            </style>
          </head>
          <body>
            <p id="unique" class="highlight">Этот текст</p>
          </body>
