Что такое специфичность селекторов классов?

Селекторы классов являются ключевой частью CSS, позволяя разработчикам выбирать несколько элементов с одним и тем же атрибутом класса и применять единообразные стили.

Это делает их очень универсальными и эффективными для применения стилей на всем веб-сайте.

Селекторы классов определяются точкой (.) с последующим именем класса. Они могут применяться к любому элементу в HTML-документе.

Вот пример использования селектора класса:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">
        <p class="highlight">Пример абзаца</p>
Файл styles.css 
      
        .highlight {
          color: green;
        }

В этом примере любому элементу с классом highlight будет присвоен зеленый цвет текста.

Селекторы классов обладают более высокой специфичностью, чем селекторы типов, но более низкой, чем селекторы идентификаторов и встроенные стили.

Значение специфичности для селектора класса равно (0, 0, 1, 0). Это означает, что селекторы классов могут переопределять селекторы типов, но могут быть переопределены селекторами идентификаторов и встроенными стилями.

Селекторы классов можно комбинировать с другими селекторами для создания более конкретных правил.

Вот пример объединения селектора типа абзаца с селектором класса:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <p class="bold-text">Пример абзаца</p>
        <p class="bold-text">Пример абзаца</p>
        <p>Ещё один абзац</p>
        <p>И ещё один абзац</p>
Файл styles.css 
      
        p.bold-text {
          font-weight: bold;
        }

Это правило применяется только к элементам p, которые также имеют класс bold-text, делая их текст жирным.
