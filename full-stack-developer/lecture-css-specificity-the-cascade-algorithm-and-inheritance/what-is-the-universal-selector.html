Что такое универсальный селектор и какова его специфичность?

Универсальный селектор, или universal selector (*) — это особый тип селектора CSS, который соответствует любому элементу в документе.

Его часто используют для применения стиля ко всем элементам на странице, что может быть полезно для сброса или нормализации стилей в разных браузерах.

Универсальный селектор можно использовать для выбора всех элементов в определенном контексте или глобально по всему документу.

Вот пример использования универсального селектора для установки margin и padding для всего HTML-документа:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <h1>Элемент заголовка</h1>
        <p>Пример элемента абзаца</p>
Файл styles.css 
      
        * {
          margin: 0;
          padding: 0;
        }

В этом примере кода селектор * сбрасывает внешние и внутренние отступы всех элементов до нуля, что является распространенным приемом, используемым в CSS-скриптах сброса.

Универсальный селектор имеет наименьшее значение специфичности среди всех селекторов. Он вносит нулевой вклад во все части значения специфичности (0, 0, 0, 0).

Это означает, что любой другой селектор, включая селекторы типов, селекторы классов, селекторы идентификаторов и встроенные стили, переопределит стили, установленные универсальным селектором.

Рассмотрим следующий пример HTML и CSS: 

    
      <head>
        <style>
          * {
            color: blue;
          }
          p {
            color: red;
          }
          .highlight {
            color: green;
          }
          #unique {
            color: purple;
          }
        </style>
      </head>
      <body>
        <p id="unique" class="highlight">К этому тексту применено несколько стилей.</p>
      </body>

Универсальный селектор * устанавливает синий(blue) цвет текста для всех элементов.

Однако селектор типа p переопределяет это, устанавливая цвет текста красным(red) специально для элементов p.

Если элемент имеет класс highlight, то селектор класса имеет приоритет, изменяя цвет текста на зеленый(green).

Наконец, селектор идентификатора #unique, обладающий наивысшей специфичностью, переопределит все остальные, установив цвет текста на фиолетовый(purple).
