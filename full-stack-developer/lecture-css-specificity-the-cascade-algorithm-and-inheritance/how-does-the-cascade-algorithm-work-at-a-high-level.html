Как работает каскадный алгоритм на высоком уровне?

Каскадный алгоритм — это процесс, который браузер использует для определения того, какие правила CSS применять, когда к одному и тому же элементу применяются несколько стилей. Он гарантирует использование наиболее подходящих стилей на основе набора четко определенных правил.

Процесс начинается с проверки релевантности(relevance). Браузер сначала фильтрует все правила CSS, чтобы найти те, которые действительно применимы к рассматриваемому элементу. Это включает в себя сопоставление селекторов и учет медиа-запросов, которые могут действовать.

Медиа-запрос — это метод CSS, используемый для применения стилей в зависимости от характеристик устройства или области просмотра, таких как ширина, высота или ориентация.

Далее алгоритм учитывает происхождение(origin) и важность(importance). CSS может поступать из разных источников: стили по умолчанию браузера (user-agent), стили, установленные пользователем, и стили, написанные автором (вами).

После определения источника алгоритм оценивает важность каждого правила, отдавая приоритет правилам, помеченным как !important, которые имеют приоритет над другими правилами независимо от их источника.

После фильтрации по происхождению и важности алгоритм проверяет специфичность(specificity). Если применяются два правила из одного источника и с одинаковым уровнем важности, будет применено правило с более высокой специфичностью.

Специфичность — это показатель того, насколько целенаправленным является селектор, при этом более специфические селекторы имеют приоритет над более общими.

Наконец, если все остальные условия равны, вступает в силу порядок появления(order of appearance). Если два правила имеют одинаковую специфичность, будет применено то, которое указано последним в CSS.

Именно поэтому порядок написания стилей иногда может влиять на результат.

Давайте рассмотрим пример.

В HTML-файле находится один элемент абзаца. Затем в CSS у нас есть два правила, каждое из которых нацелено на этот элемент абзаца.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <p>пример абзаца</p>
Файл styles.css 
      
        p {
          color: blue;
        }

        p {
          color: green; 
        }

Первое правило устанавливает для всех элементов абзаца синий (blue) цвет текста, а второе правило — зеленый (green) цвет текста.

Какой цвет будет использован? Зеленый цвет будет использован для элементов абзаца.

Учитывая релевантность, происхождение и важность, специфичность, область применения и порядок появления, каскадный алгоритм гарантирует предсказуемое поведение ваших CSS-стилей, позволяя создавать более сложные и многогранные веб-страницы.
