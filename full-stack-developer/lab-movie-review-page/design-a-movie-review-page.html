Создайте страницу для рецензирования фильма.

Создайте приложение, функционально похожее на этот пример проекта. Старайтесь не копировать пример проекта, придайте ему свой собственный стиль.

Цель: Выполнить приведенные ниже пользовательские истории и добиться успешного прохождения всех тестов для завершения лабораторной работы.

Истории пользователей:

  1. У вас должен быть элемент main.
  2. Внутри элемента main должен быть элемент h1 для названия фильма.
  3. Под элементом h1 должен располагаться элемент img, отображающий обложку фильма. Элемент img должен содержать описательный alt-текст, описывающий изображение. Вы можете использовать следующее изображение, если хотите: https://cdn.freecodecamp.org/curriculum/labs/rise-beyond-2.png.
  4. Вам потребуется элемент p, содержащий краткое описание фильма.
  5. Вам потребуется ещё один элемент p для отображения рейтинга фильма. Внутри него должны располагаться следующие элементы в указанном порядке:
    • Элемент strong с текстом Рейтинг фильма:.
    • Элемент span с атрибутом aria-hidden, установленным в значение true, содержащий визуальное представление рейтинга в виде звездочек ⭐⭐⭐⭐⭐⭐⭐⭐⭐☆.
    • Числовое значение, представляющее рейтинг, в скобках (например, 9,2/10) после span.
  6. Вам потребуется элемент h2 с текстом Актёры.
  7. У вас должен быть элемент ul.
  8. Внутри элемента ul должно быть несколько элементов li, каждый из которых содержит элемент strong с именем актера, за которым следует имя соответствующего персонажа, а перед ним — текст в роли (например, Джеймс Холлоуэй в роли Итана Картера).

Тесты

  1. У вас должен быть элемент main.
  2. Внутри элемента main должен находиться элемент h1, содержащий название фильма.
  3. Под элементом h1 должен располагаться элемент img с описательным атрибутом alt.
  4. После изображения должен быть элемент p, содержащий описание фильма.
  5. Вам потребуется еще один элемент p, содержащий рейтинг.
  6. Внутри второго элемента p должен находиться элемент strong с текстом Рейтинг фильма: .
  7. Внутри абзаца с рейтингом должен быть элемент span.
  8. Элемент span внутри p с рейтингом должен содержать десять звезд, либо заполненных (⭐), либо пустых (☆), после которых в скобках следует числовое значение.
  9. Элемент span внутри абзаца с рейтингом должен иметь атрибут aria-hidden со значением true.
  10. У вас должен быть элемент h2.
  11. Элемент h2 должен располагаться после второго элемента p.
  12. В заголовке h2 должен быть текст Актёры.
  13. После заголовка h2 должен быть элемент ul.
  14. Элемент ul должен содержать несколько элементов li.
  15. Каждый элемент li должен содержать заголовок с именем актера, за которым следует имя соответствующего персонажа, а перед ним текст в роли (например, Джеймс Холлоуэй в роли Итана Картера).
