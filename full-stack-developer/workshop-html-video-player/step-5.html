Шаг 5

Атрибут controls предоставляет элементы управления воспроизведением, включая воспроизведение, перемотку назад и регулировку громкости для видеоэлемента.

Атрибут controls является логическим и не требует указания значения.

Добавьте атрибут controls к элементу video.

Теперь на странице должен отобразиться элемент video.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Работаем с элементом HTML-видео</title>
        </head>
        <body>
          <h1>Работаем с элементом HTML-видео</h1>
          
          <video width="640" loop></video>

        </body>
        </html>
