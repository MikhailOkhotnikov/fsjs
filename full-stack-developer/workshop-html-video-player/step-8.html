Шаг 8

Возможно, вы заметили, что не добавили ссылку на само видео. Вы сделаете это на следующем этапе. Что касается типов видеофайлов, браузеры поддерживают разные форматы. Для этого вы можете использовать элементы source внутри элемента video, и браузер выберет первый совместимый source.

Вот пример исходного элемента: 

    
      <video controls width="250">
        <source src="url-адрес src находится здесь" type="тип видео находится здесь" />
      </video>

Элемент source является пустым элементом, поэтому у него нет закрывающего тега.

Добавьте элемент source внутрь элемента video.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Работаем с элементом HTML-видео</title>
        </head>
        <body>
          <h1>Работаем с элементом HTML-видео</h1>          
          <video 
            width="640" 
            loop 
            controls
            muted
            poster="https://cdn.freecodecamp.org/curriculum/labs/past-event2.jpg"
          >

          </video>
        </body>
        </html>
