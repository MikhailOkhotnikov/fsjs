Шаг 13

Последним элементом source, который вы добавите, будет MIME-тип video/quicktime.

QuickTime — это расширяемая мультимедийная архитектура, созданная компанией Apple, которая поддерживает воспроизведение, потоковую передачу, кодирование и перекодирование различных цифровых медиаформатов. Хотя она не так популярна, как формат MP4, она может понадобиться для поддержки устаревших приложений.

Ниже третьего элемента source добавьте четвертый элемент source и присвойте ему атрибут src со значением https://cdn.freecodecamp.org/curriculum/labs/mapmethod.mov, а атрибут type — со значением video/quicktime.

Поздравляем! Вы успешно завершили мастер-класс по HTML-видеоплееру.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Работаем с элементом HTML-видео</title>
        </head>
        <body>
          <h1>Работаем с элементом HTML-видео</h1>          
          <video 
            width="640" 
            loop 
            controls
            muted
            poster="https://cdn.freecodecamp.org/curriculum/labs/past-event2.jpg"
          >
          <source
            src="https://cdn.freecodecamp.org/curriculum/labs/what-is-the-map-method-and-how-does-it-work.mp4"
            type="video/mp4"
          >
          <source
            src="https://cdn.freecodecamp.org/curriculum/labs/mapmethod.webm"
            type="video/webm"
          >
          <source
            src="https://cdn.freecodecamp.org/curriculum/labs/mapmethod.ogg"
            type="video/ogg"
          >
                  
          </video>
        </body>
        </html>
