Шаг 3

В предыдущих уроках вы узнали о различных атрибутах, доступных элементу video. Атрибут width определяет ширину видео в пикселях.

Добавьте к элементу video атрибут width со значением 640.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Работаем с элементом HTML-видео</title>
        </head>
        <body>
          <h1>Работаем с элементом HTML-видео</h1>
          
          <video></video>

        </body>
        </html>
