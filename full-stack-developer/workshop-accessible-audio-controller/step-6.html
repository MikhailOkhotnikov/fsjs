Шаг 6

Как вы помните из предыдущих уроков, атрибут aria-labelledby используется, когда на странице уже есть текст, который можно использовать в качестве метки. В данном случае вы добавили атрибут id к элементам span, который теперь будет использоваться здесь.

Добавьте атрибут aria-labelledby к элементу input и установите значение volume-label volume-description.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <title>Доступные элементы управления</title>
          </head>
          <body>
            <h1>Управление звуком</h1>
            <button type="button">Воспроизвести</button>
             <div>
              <span id="volume-label">Громкость</span>
              <span id="volume-description">Отрегулируйте уровень звука</span>

              <input type="range" min="0" max="100" value="50" />
              
             </div>  
          </body>
        </html>
