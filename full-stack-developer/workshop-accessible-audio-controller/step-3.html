Шаг 3

Далее добавьте элемент div, который будет содержать все элементы для регулятора громкости.

Внутри элемента div разместите элемент span с атрибутом id, установленным на volume-label. Текст элемента span должен быть Громкость.

id будет впоследствии использоваться атрибутами ARIA, чтобы вспомогательные технологии могли идентифицировать этот текст как метку для регулятора громкости.
