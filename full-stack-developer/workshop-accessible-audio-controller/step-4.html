Шаг 4

Теперь пришло время добавить еще один элемент span.

Этот элемент span должен иметь атрибут id со значением volume-description. Кроме того, текст для этого элемента span должен быть следующим: Отрегулируйте уровень звука.

Аналогично предыдущему элементу span, идентификатор этого элемента span будет впоследствии использоваться атрибутами ARIA, чтобы вспомогательные технологии могли идентифицировать этот текст как описание регулятора громкости.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <title>Доступные элементы управления</title>
          </head>
          <body>
            <h1>Управление звуком</h1>
            <button type="button">Воспроизвести</button>
             <div>
               <span id="volume-label">Громкость</span>

             </div>  
          </body>
        </html>
