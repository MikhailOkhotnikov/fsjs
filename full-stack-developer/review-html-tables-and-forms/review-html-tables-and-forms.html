Обзор HTML-таблиц и форм

Элементы и атрибуты HTML-формы

  • Элемент form: используется для создания HTML-формы для ввода данных пользователем.
  • Атрибут action: используется для указания URL-адреса, куда должны быть отправлены данные формы.
  • Атрибут method: используется для указания метода HTTP, который будет применяться при отправке данных формы. Наиболее распространенные методы — GET и POST. 
    •         
          <form method="здесь-размещается-значение" action="здесь-размещается-URL">
            <!-- Здесь размещаются входные данные. -->
          </form>
  • Элемент input: используется для создания поля ввода для ввода данных пользователем.
  • Атрибут type: используется для указания типа поля ввода. Например: text(текст), email(электронная почта), number(число), radio(радио), checkbox(флажок) и т. д..
  • Атрибут placeholder: используется для отображения подсказки пользователю, показывающей, что нужно ввести в поле ввода.
  • Атрибут value: используется для указания значения поля ввода. Если поле ввода имеет тип button, атрибут value можно использовать для установки текста кнопки.
  • Атрибут name: используется для присвоения имени полю ввода, которое служит ключом при отправке данных формы. Для переключателей (радиокнопок) присвоение им одинакового name группирует их вместе, так что одновременно можно выбрать только один вариант из группы.
  • Атрибут size: используется для определения количества символов, которые должны быть видны, когда пользователь вводит текст в поле ввода.
  • Атрибут min: может использоваться с типами ввода, например, number, для указания минимального допустимого значения в поле ввода.
  • Атрибут max: может использоваться с типами ввода, такими как number, для указания максимально допустимого значения в поле ввода.
  • Атрибут minlength: используется для указания минимального количества символов, необходимых для заполнения поля ввода.
  • Атрибут maxlength: используется для указания максимального количества символов, разрешенных в поле ввода.
  • Атрибут required: используется для указания того, что поле ввода должно быть обязательно заполнено перед отправкой формы.
  • Атрибут disabled: используется для указания того, что поле ввода должно быть отключено.
  • Атрибут readonly: используется для указания того, что поле ввода доступно только для чтения.
    •       
        <!-- Текстовое поле ввода -->
        <input 
          type="text"
          id="name"
          name="name"
          placeholder="например, Иванов Иван" 
          size="20"
          minlength="5"
          maxlength="30"
          required
        />

        <!-- Числовое поле ввода -->
        <input 
          type="number"
          id="quantity"
          name="quantity"
          min="2"
          max="10"
          disabled
        />

        <!-- Кнопка-->
        <input type="button" value="Показать оповещение" />
  • Элемент label: используется для создания метки для поля ввода.
  • Атрибут for: используется для указания того, к какому полю ввода относится метка.
  • Неявная привязка к форме: поля ввода можно связать с метками, заключив поле ввода в элемент label.
    •       
        <form action="">
          <label>
            ФИО:
            <input type="text" />
          </label>
        </form>
  • Явная привязка к форме: поля ввода можно связать с метками, используя атрибут for элемента label.
    •       
        <form action="">
          <label for="email">Адрес электронной почты: </label>
          <input type="email" id="email" />
        </form>
  • Элемент button: используется для создания кликабельной кнопки. Кнопка также может иметь атрибут type, который используется для управления поведением кнопки при её активации. Например: submit, reset, button. 
    •       
        <button type="button">Показать форму</button>
        <button type="submit">Отправить форму</button>
        <button type="reset">Сбросить форму</button>
  • Элемент fieldset: используется для группировки связанных полей ввода.
  • Элемент legend: используется для добавления подписи, описывающей группу входных данных.
    •       
        <!-- Группа радиокнопок -->
        <fieldset>
          <legend>Вы впервые остановились в нашем отеле?</legend>

          <label for="yes-option">Да</label>
          <input id="yes-option" type="radio" name="hotel-stay" value="yes" />

          <label for="no-option">Нет</label>
          <input id="no-option" type="radio" name="hotel-stay" value="no" />
        </fieldset>

        <!-- Группа флажков -->
        <fieldset>
          <legend>
            Почему вы выбрали именно наш отель для проживания? (Отметьте все подходящие варианты)
          </legend>

          <label for="location">Местоположение</label>
          <input type="checkbox" id="location" name="location" value="location" />

          <label for="price">Цена</label>
          <input type="checkbox" id="price" name="price" value="price" />
        </fieldset>
  • Состояние фокусировки(focused state): это состояние поля ввода, когда оно выбрано пользователем.

Работа с элементами и атрибутами HTML-таблиц

  • Элемент table: используется для создания HTML таблицы.
  • Элемент Table Head (thead): используется для группировки содержимого заголовков в HTML-таблице.
  • Элемент Table Row (tr): используется для создания строки в HTML-таблице.
  • Элемент Table Header (th): используется для создания ячейки заголовка в HTML-таблице.
  • Элемент Table body (tbody): используется для группировки основного содержимого в HTML-таблице.
  • Элемент Table Data Cell (td): используется для создания ячейки данных в HTML-таблице.
  • Элемент Table Foot (tfoot): используется для группировки содержимого нижнего колонтитула в HTML-таблице.
  • Элемент caption: используется для добавления заголовка HTML-таблицы.
  • Атрибут colspan: используется для указания количества столбцов, которые должна занимать ячейка таблицы.
    •       
        <table>
          <caption>Оценки за итоговый экзамен по математическому анализу</caption>
          <thead>
            <tr>
              <th>Фамилия</th>
              <th>Имя</th>
              <th>Оценка</th>
            </tr>
          </thead>
          <tbody>
              <tr>
                <td>Иванов</td>
                <td>Иван</td>
                <td>5</td>
              </tr>
              <tr>
                <td>Петров</td>
                <td>Пётр</td>
                <td>4</td>
              </tr>
              <tr>
                <td>Сидоров</td>
                <td>Сидор</td>
                <td>3</td>
              </tr>
              <tr>
                <td>Михайлов</td>
                <td>Михаил</td>
                <td>4</td>
              </tr>
              <tr>
                <td>Дмитриев</td>
                <td>Дмитрий</td>
                <td>5</td>
              </tr>                 
          </tbody>
          <tfoot>
            <tr>
              <td colspan="2">Средняя оценка</td>                  
              <td>4,2</td>
            </tr>
          </tfoot>
        </table>

Работа с инструментами HTML

  • Валидатор HTML: инструмент, проверяющий синтаксис HTML-кода на предмет его корректности.
  • Инспектор DOM: инструмент, позволяющий проверять и изменять HTML-структуру веб-страницы.
  • Devtools: набор инструментов для веб-разработчиков, встроенных непосредственно в браузер, которые помогают отлаживать, профилировать и анализировать веб-страницы.
