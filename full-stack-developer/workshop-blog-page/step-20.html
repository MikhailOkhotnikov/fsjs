В разделе Контакты необходимо указать контактную информацию автора блога. Для этого используется элемент address.

Элемент address используется для представления контактной информации человека или организации.

Вот пример использования элемента address для физического адреса. Элемент br здесь используется для создания разрыва(break) строки между текстом.

Пример кода: 
      
        <address>
          ул. Советская, д. 123 <br />
          г. Пенза, Россия
        </address>

Под элементом h2 добавьте элемент address.
