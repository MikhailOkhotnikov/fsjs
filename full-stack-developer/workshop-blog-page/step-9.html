Шаг 9

Внутри каждого элемента li вам понадобится элемент-привязка(anchor).

Для первого элемента привязки текст должен быть Обо мне, а значение атрибута href должно быть #about. Символ решетка(hash) перед словом about представляет собой идентификатор(id), который будет добавлен позже в проект.

Для второго элемента привязки текст должен быть Посты, а значение атрибута href должно быть #posts

Для третьего элемента привязки текст должен быть Контакты, а значение атрибута href должно быть #contact
