Шаг 5

Первый раздел, который вам предстоит создать, — это заголовок(header) страницы.

Элемент header используется для представления вводного содержимого, такого как навигация по страницам и другая вводная информация.

Вот пример использования элемента header:

Пример кода: 
      
        <header>
          <h1>Заголовок главной Страницы Находится Здесь</h1>
          <img src="example-logo.png" alt="Example logo" />
        </header>

Внутри элемента body добавьте элемент header.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Блог мистера Уискерса</title>
          </head>
          <body>

          </body>
        </html>
