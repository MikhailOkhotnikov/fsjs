Шаг 3

Следующее, что нужно сделать, — установить атрибут viewBox элемента svg. Он будет контролировать видимую часть изображения. Первые два числа задают центр изображения.

Следующие два числа задают размер изображения, которое мы видим: ширина и высота.

Поскольку в данном случае значок должен быть виден полностью, необходимо задать атрибуту viewBox значение 0 0 24 24.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Значок сердца</title>
          </head>
          <body>
            <svg width="24" height="24">
            </svg>
          </body>
        </html>
