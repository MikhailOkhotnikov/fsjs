Какова роль тегов Open Graph и как они влияют на SEO?

Протокол Open Graph позволяет управлять отображением контента вашего сайта в различных социальных сетях, таких как Facebook, LinkedIn и многих других. Настраивая эти свойства Open Graph, вы можете побудить пользователей кликнуть по ссылке и взаимодействовать с вашим контентом. Эти свойства можно задать с помощью набора meta-элементов в HTML-разделе head.

Первое важное свойство OG, которое следует включить, — это title. Вот пример настройки title OG для домашней страницы freeCodeCamp: 

    
      <meta content="freeCodeCamp.org" property="og:title" />

Для атрибута property необходимо указать значение og:title. В атрибуте content нужно указать заголовок, который будет отображаться в социальных сетях.

Следующим важным свойством OG является тип(type). Вот пример использования type OG для домашней страницы freeCodeCamp: 

    
      <meta property="og:type" content="website" />

Свойство type используется для представления типа контента, публикуемого в социальных сетях. Примерами такого контента являются статьи, веб-сайты, видео или музыка.

Третьим важным свойством OG является изображение(image). Вот пример установки image OG для главной страницы freeCodeCamp: 

    
      <meta
        content="https://cdn.freecodecamp.org/platform/universal/fcc_meta_1920X1080-indigo.png"
        property="og:image"
      />

В этом примере изображение Open Graph указывает на логотип freeCodeCamp. Все эти изображения должны быть высокого качества, с правильными размерами и соотношением сторон. Большинство социальных сетей устанавливают критерии требований к изображениям, которые помогут вам обеспечить корректное отображение вашего контента на их сайте. Например, на странице документации developer.facebook.com указано:

Для оптимального отображения на устройствах с высоким разрешением используйте изображения размером не менее 1200 на 630 пикселей. Для отображения постов со ссылками на страницы с более крупными изображениями следует использовать изображения размером не менее 600 на 315 пикселей.

Четвёртым важным свойством OG будет url. Вот пример настройки url OG для домашней страницы freeCodeCamp: 

    
      <meta property="og:url" content="https://www.freecodecamp.org" />

Вы можете настроить множество других свойств Open Graph, таких как description, audio, video и locale. Однако наиболее важными Open Graph являются url, image, type и title.

Как же эти свойства Open Graph влияют на поисковую оптимизацию? Когда ваш контент публикуется в социальных сетях, грамотно продуманные свойства Open Graph могут улучшить его внешний вид в лентах пользователей. Это может привести к повышению CTR(Click-Through Rate — метрика в интернет-маркетинге, показатель кликабельности), что может быть сигналом для поисковых систем о релевантности и интересности вашего контента.
