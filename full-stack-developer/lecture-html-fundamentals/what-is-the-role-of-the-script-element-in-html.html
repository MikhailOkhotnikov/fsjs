Какова роль элемента Script в HTML и как его можно использовать для ссылки на внешние файлы JavaScript?

Элемент script используется для внедрения исполняемого кода. Большинство разработчиков используют его для выполнения кода JavaScript. JavaScript используется для добавления интерактивности на веб-страницы. Распространенные примеры использования JavaScript включают интерактивные игры, слайдеры изображений и динамические формы, проверяющие вводимые пользователем данные в режиме реального времени. Вот пример использования элемента script в HTML-документе: 

    
      <body>
        <script>
          alert("Добро пожаловать на JSST");
        </script>
      </body>

В этом примере мы используем оповещение(alert) с сообщением Добро пожаловать на JSST. При первой загрузке страницы появляется оповещение. После этого пользователь может нажать кнопку «ОК», чтобы закрыть сообщение.

Хотя технически вы можете разместить весь код JavaScript внутри тегов script, рекомендуется вместо этого ссылаться на внешний файл JavaScript. Вот пример использования элемента script для ссылки на внешний файл JavaScript: 

    
      <script src="path-to-javascript-file.js"></script>

Атрибут src используется здесь для указания расположения внешнего файла JavaScript. src означает «source» (источник). Не рекомендуется размещать весь код JavaScript в HTML-документе из-за принципа разделения ответственности. Разделение ответственности — это принцип проектирования, согласно которому программы разделяются на отдельные разделы, каждый из которых отвечает за отдельную задачу. В данном случае мы хотим отделить код JavaScript от кода HTML.
