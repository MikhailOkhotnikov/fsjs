Обзор Основы работы с компьютером

Понимание основ работы с компьютерами, интернетом и инструментами для разработчиков.

  • Материнская плата(Motherboard): содержит всю необходимую для работы компьютера память, разъемы и жесткие диски. Она служит основной печатной платой компьютера.
  • Центральный процессор(ЦП) (Central Processing Unit, CPU): процессор, отвечающий за выполнение инструкций и вычислений.
  • Оперативная память (Random Access Memory, RAM): временное место хранения данных для центрального процессора компьютера.
  • Жесткий диск (Hard Disk Drive, HDD): постоянное хранилище данных, используемое для хранения информации даже при выключенном компьютере.
  • Твердотельный накопитель (Solid State Drive, SSD): энергонезависимая флэш-память, которую можно использовать вместо жесткого диска.
  • Блок питания (БП) (Power Supply Unit, PSU): отвечает за преобразование электроэнергии из розетки в форму, пригодную для использования компьютером.
  • Графический процессор (Graphics Processing Unit, GPU): отвечает за отрисовку изображений на экране компьютера.
  • Различные типы интернет-провайдеров: Интернет-провайдер (Internet Service Provider, ISP) — это компания, предоставляющая доступ к интернету. Существуют различные типы интернет-провайдеров, включая коммутируемый доступ(dial-up), DSL, кабельный, оптоволоконный и спутниковый.
  • Безопасные способы входа в систему: Примерами безопасных способов входа в систему являются использование надежного пароля, включение двухфакторной аутентификации и использование менеджера паролей.
  • Интегрированная среда разработки (Integrated Development Environment, IDE): инструмент, помогающий разработчикам эффективно писать, тестировать и отлаживать код.
  • Редактор кода (Code Editor): инструмент, который разработчики используют для написания и отладки кода.
  • Git: популярная система контроля версий, которая позволяет разработчикам отслеживать изменения в своем коде и сотрудничать с другими разработчиками.
  • Облачные сервисы хостинга для репозиториев: Репозиторий — это место хранения файлов проекта и истории версий. Популярные облачные сервисы хостинга для репозиториев включают GitHub, GitLab и Bitbucket.
  • Менеджеры пакетов (Package Managers): инструменты, помогающие разработчикам упростить процесс добавления, обновления и удаления библиотек и зависимостей проекта. Примеры включают npm, pip и Maven.
  • Библиотеки и фреймворки для тестирования: Тестирование программного обеспечения проводится для обеспечения корректной работы кода. Примерами библиотек и фреймворков для тестирования являются Jest, PHPUnit и JUnit.

Работа с файлами, файловыми системами и форматами носителей.

  • Лучшие практики по именованию файлов: Называйте файлы так, чтобы их было легко понять и поддерживать. Например, about-us.html — более описательное имя, чем page1.html.
  • Корневой каталог (root directory): каталог верхнего уровня в файловой системе. Каталог — это другое название папки.
  • Markdown: язык разметки, широко используемый для документации и файлов README. Файл README — это файл, содержащий информацию о проекте, например, как его установить и использовать.
  • index.html: представляет собой страницу по умолчанию, которая отображается при посещении пользователем веб-сайта.
  • Создание, перемещение и удаление файлов и папок с помощью Проводника/Finder: Проводник — это файловый менеджер в Windows, а Finder — в macOS. С помощью этих инструментов можно создавать, перемещать и удалять файлы и папки.
  • Поиск файлов и папок: Вы можете использовать функцию поиска в Проводнике или Finder, чтобы найти файлы и папки на вашем компьютере.
  • Типы файлов HTML, CSS и JS: расширение .html используется для файлов HTML, .css — для файлов CSS, а .js — для файлов JavaScript.
  • Распространенные форматы изображений и графики: JPEG и PNG — распространенные форматы файлов изображений. GIF — еще один распространенный формат файлов изображений, поддерживающий анимацию. SVG — формат файлов для векторной графики.
  • Распространенные аудио и видео форматы: Формат MP3 обычно используется для аудиофайлов. Формат MP4 обычно используется для видеофайлов. Формат MOV был разработан компанией Apple и обычно используется для видеофайлов.
  • Распространенные форматы шрифтов: Формат TTF обычно используется для шрифтов TrueType. Формат WOFF обычно используется для веб-шрифтов. Преемником WOFF является WOFF2, который обеспечивает лучшее сжатие.
  • ZIP: формат файлов, используемый для сжатия файлов и папок.

Эффективный просмотр веб-страниц

  • Что такое веб-браузер?: это программное приложение, позволяющее пользователям получать доступ к веб-сайтам в интернете и просматривать их.
  • Что такое поисковая система: инструмент, позволяющий пользователям искать информацию в интернете. Примеры: Yandex, Google, Bing и Yahoo.
  • Популярные веб-браузеры: Yandex, Google Chrome, Mozilla Firefox и Microsoft Edge.
  • Распространенные стратегии поиска: Для поиска контента на конкретном сайте можно использовать site: с последующим URL-адресом веб-сайта. Для поиска файлов конкретного типа можно использовать filetype: с последующим расширением файла. Для исключения результатов, содержащих конкретный термин, можно добавить перед поисковым запросом знак минус. Для включения результатов, содержащих конкретный термин, можно добавить перед поисковым запросом знак плюс.
