Как отображать адреса в HTML?

Элемент «Контактный адрес» используется для представления контактной информации в разделе веб-страницы. Элемент address универсален и может использоваться для бизнес-страниц, страниц авторов, персональных сайтов и т. д.

При создании раздела контактов на вашем сайте следует использовать семантический элемент address вместо общего элемента, например div.

Вот пример использования элемента address на странице контактов компании: 

    
      <address>
        <h2>ООО "Рога и копыта"</h2>
        <p>
          634050<br />
          Россия,<br />
          Томская область,<br />
          г. Томск,<br />
          ул. Пушкина,<br />
          д. 1000,<br />
          кв. 100<br />
        </p>
        <p>Телефон: <a href="tel:+15555555555">+1 (555) 555-5555</a></p>
        <p>Email: <a href="mailto:contact@company.ru">contact@company.ru</a></p>
      </address>

В этом примере указаны название компании, физический адрес, номер телефона и адрес электронной почты. Для физического адреса используется элемент переноса строки br, который обозначает разделение между названием улицы, городом, страной и т.п.

Для номера телефона у нас есть элемент привязки со значением href, заданным для телефонных номеров. Значение tel:+ внутри атрибута href создаёт кликабельную ссылку для совершения телефонного звонка на некоторых устройствах, поддерживающих эту функцию.

Для адреса электронной почты используется другой элемент привязки со значением href, равным ссылке mailto. Ссылки mailto используются в HTML-документах, чтобы пользователи могли открывать новые письма в предпочитаемом ими почтовом клиенте.

Один из недостатков использования ссылки mailto заключается в том, что пользователи часто воспринимают её как спам. К сожалению, многие спамеры используют этот вариант для отправки писем пользователям. Поэтому помните об этом, когда пользуетесь этой функцией.
