Какие способы добавления рамок вокруг изображений существуют?

В CSS существует несколько способов добавления рамок вокруг изображений, каждый из которых предлагает различные варианты оформления и уровни контроля.

Давайте рассмотрим некоторые из наиболее распространенных и универсальных методов.

Самый простой способ добавить рамку к изображению — использовать свойство border. Это свойство представляет собой сокращенную запись, позволяющую одновременно задать ширину, стиль и цвет рамки.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">
        <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Милый кот лежит на спине.">
Файл styles.css 
      
        img {
          border: 2px solid red;
        }

Это правило CSS добавляет сплошную (solid) красную (red) рамку шириной 2 пикселя (2px) вокруг всех элементов img. Вы можете настроить ширину, стиль (например, пунктирную (dashed), точечную (dotted) или двойную(double)) и цвет в соответствии с вашими дизайнерскими потребностями.

Если вам требуется более точный контроль над отдельными сторонами границы, вы можете использовать специальные свойства границы для каждой стороны:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">
        <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Милый кот лежит на спине.">
Файл styles.css 
      
        img {
          border-top: 10px solid red;
          border-right: 10px dashed green;
          border-bottom: 10px dotted blue;
          border-left: 10px double purple;
        }

Это позволяет создавать уникальные стили рамок для каждой стороны изображения.

Ещё один способ создать эффект границы — использовать свойство outline. Хотя оно похоже на border, свойство outline не влияет на размеры или расположение элемента:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">
        <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Милый кот лежит на спине.">
Файл styles.css 
      
        img {
          outline: 3px solid gold;
        }

При этом вокруг изображения создается золотой (gold) контур без изменения его размера или положения.

Если вы хотите создать закругленные углы на границе, вы можете использовать свойство border-radius в сочетании со свойством border:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">
        <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Милый кот лежит на спине.">
Файл styles.css 
      
        img {
          border: 2px solid black;
          border-radius: 10px;
        }

Помните, что эти методы можно комбинировать и настраивать для создания уникальных эффектов оформления границ. Выбор метода зависит от ваших конкретных требований к дизайну и необходимого уровня сложности.
