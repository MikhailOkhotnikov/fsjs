Как работают размер, повтор, положение и прикрепление фонового изображения?

При работе с фоновыми изображениями в CSS у вас есть несколько свойств, позволяющих управлять отображением этих изображений.

Основные свойства, на которых мы сосредоточимся, — это background-size, background-repeat, background-position и background-attachment.

Давайте сначала рассмотрим свойство background-image: 

    
      <style>
        body {
          background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
        }
      </style>

Приведённый выше CSS-код устанавливает фоновое изображение кошки для элемента body.

Если вы хотите задать размер фонового изображения, вы можете использовать свойство background-size.

С помощью функции contain можно максимально увеличить масштаб изображения без обрезки или растягивания.

Вот пример с параметром background-size: contain: 

    
      <style>
        body {
          background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
          background-size: contain;
          min-height: 100px;
        }
      </style>

Мы устанавливаем минимальную высоту (min-height) в 100px, чтобы фоновое изображение было видно, а макет сохранял базовую высоту, гарантируя, что даже при минимальном количестве контента дизайн будет выглядеть сбалансированным и визуально привлекательным.

Если мы изменим свойство background-size, используя значение cover, то фоновое изображение будет масштабироваться, чтобы покрыть весь элемент body, сохраняя при этом его соотношение сторон. 

    
      <style>
        body {
          background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
          background-size: cover;
          min-height: 100px;
        }
      </style>

В предыдущих примерах вы, вероятно, заметили, что фоновое изображение непрерывно повторялось.

По умолчанию фоновые изображения повторяются как по горизонтали, так и по вертикали, заполняя весь элемент. Однако вы можете настроить это поведение.

Вы можете использовать свойство background-repeat со значением no-repeat. 

    
      <style>
        body {
          background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
          background-size: contain;
          background-repeat: no-repeat;
          min-height: 100px;
        }
      </style>

Если установить параметр background-size в значение contain и параметр background-repeat в значение no-repeat, изображение больше не будет повторяться на экране.

Если вы хотите, чтобы фоновое изображение повторялось по горизонтали, вы можете использовать значение repeat-x для свойства background-repeat. 

    
      <style>
        body {
          background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
          background-size: contain;
          background-repeat: repeat-x;
          min-height: 100px;
        }
      </style>

Если вы хотите, чтобы фоновое изображение повторялось по вертикали, вы можете использовать значение repeat-y для свойства background-repeat. 

    
      <style>
        body {
          background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
          background-size: contain;
          background-repeat: repeat-y;
          min-height: 100px;
        }
      </style>

Для размещения фонового изображения на экране можно использовать свойство background-position.

Свойство background-position позволяет задать местоположение фонового изображения в элементе. Можно использовать ключевые слова, такие как top, bottom, left, right и center, или конкретные значения в пикселях или процентах.

Вот пример использования параметров center и top для свойства background-position: 

    
      <style>
        body {
          background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
          background-size: contain;
          background-repeat: no-repeat;
          background-position: center top;
          min-height: 100px;
        }
      </style>

Этот CSS-код размещает фоновое изображение по горизонтали в центре элемента и по вертикали в верхней части.

Наконец, параметр background-attachment определяет, будет ли фоновое изображение прокручиваться вместе с содержимым или оставаться неподвижным при прокрутке страницы.

Основные значения параметра — scroll (по умолчанию), при котором фоновое изображение прокручивается вместе с содержимым, и fixed, при котором фоновое изображение остается в одном и том же положении на экране.

Вот пример использования значения fixed свойства background-attachment: 

    
      <style>
        body {
          background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
          background-position: center top;
          background-attachment: fixed;
        }
      </style>

Этот CSS-код устанавливает фоновое изображение таким образом, чтобы оно оставалось неподвижным даже при прокрутке страницы.

Если вам нужно объединить несколько свойств в одну строку, вы можете сделать это, используя сокращенное свойство background.

Вот пример: 

    
      <style>
        body {
          background: center top fixed
            url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
        }
      </style>

В приведенном выше примере параметры background-image, background-position и background-attachment объединены в одну строку.

Овладев этими свойствами, вы получите полный контроль над отображением фоновых изображений в ваших веб-дизайнах, что позволит создавать более привлекательные и функциональные макеты.
