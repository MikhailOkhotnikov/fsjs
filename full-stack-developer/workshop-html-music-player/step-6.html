Шаг 6

Атрибут controls предоставляет элементы управления воспроизведением, включая паузу/возобновление воспроизведения, перемотку и регулировку громкости для элемента audio.

Атрибут controls является логическим и не требует указания значения.

Добавьте атрибут controls к элементу audio.

Теперь на странице должен отображаться элемент audio.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
        <head>
          <meta charset="UTF-8">
          <meta name="viewport" content="width=device-width, initial-scale=1.0">
          <title>Работа с элементом HTML Audio</title>
        </head>
        <body>
          <h1>Мелодии freeCodeCamp</h1>
          <h2>Can't Stay Down</h2>
          <p>Исполнитель: Куинси Ларсон</p>

          <audio src="https://cdn.freecodecamp.org/curriculum/js-music-player/can't-stay-down.mp3" loop ></audio>
          
        </body>
        </html>
