Создайте страницу рецептов

Создайте приложение, функционально похожее на этот пример проекта. Постарайтесь не копировать пример проекта, а придайте ему свой собственный стиль.

Цель: реализовать приведенные ниже пользовательские истории и пройти все тесты, необходимые для завершения лабораторной работы.

Истории пользователей:

  1. У вас должно быть объявление !DOCTYPE html
  2. У вас должен быть элемент html с атрибутом lang, со значением равным en.
  3. У вас должен быть элемент head, содержащий элемент title с названием вашего рецепта, и элемент meta с атрибутом charset, установленным на UTF-8.
  4. У вас должен быть элемент body.
  5. У вас должен быть элемент h1 с названием вашего рецепта.
  6. У вас должен быть элемент p, который представляет рецепт под h1.
  7. У вас должен быть первый элемент h2 с текстом Ингредиенты для раздела ингредиентов.
  8. У вас должен быть неупорядоченный список (элемент ul) с как минимум четырьмя пунктами списка (элементы li), в котором перечислены ваши ингредиенты под первым элементом h2.
  9. У вас должен быть второй элемент h2 с текстом Инструкции для раздела инструкций.
  10. У вас должен быть упорядоченный список (элемент ol) с как минимум четырьмя пунктами, в котором перечислены шаги рецепта по порядку, под вторым тегом h2.
  11. У вас должен быть один элемент img с атрибутом src, установленным на допустимое изображение, вы можете использовать https://cdn.freecodecamp.org/curriculum/labs/recipe.jpg если хотите, и атрибутом alt, описывающим изображение.

Тесты

  1. У вас должно быть объявление !DOCTYPE html
  2. У вас должен быть элемент html с атрибутом lang, со значением равным en.
  3. Внутри элемента html должен быть элемент head.
  4. Элемент title должен находиться внутри элемента head.
  5. Элемент title должен содержать название вашего рецепта.
  6. Внутри элемента head должен быть элемент meta.
  7. Для атрибута charset вашего элемента meta должно быть установлено значение UTF-8.
  8. Внутри элемента html должен быть элемент body.
  9. У вас должен быть элемент h1 с названием вашего рецепта.
  10. У вас должен быть только один элемент h1.
  11. Под элементом h1 должен быть элемент p.
  12. Ваш первый элемент p должен описывать ваш рецепт.
  13. Ваш первый элемент h2 должен содержать текст Ингредиенты.
  14. Под первым элементом h2 должен располагаться неупорядоченный список.
  15. В неупорядоченном списке ингредиентов должно быть не менее четырех элементов.
  16. Второй элемент h2 должен содержать текст Инструкции.
  17. Под вторым элементом h2 должен располагаться элемент упорядоченного списка.
  18. В упорядоченном списке с инструкциями должно быть не менее четырех элементов списка.
  19. У вас должен быть хотя бы один элемент img.
  20. Все элементы img должны иметь действительный атрибут src и значение.
  21. Все элементы img должны иметь атрибут alt для описания изображения.
