Шаг 14

Последний шаг — добавить в таблицу еще одну строку.

Начните с добавления еще одного элемента tr. Внутри этого элемента tr добавьте элемент th с атрибутом scope, установленным на row, и текстовым содержимым 2:00 PM.

Затем добавьте три элемента td со следующим текстовым содержимым:

  • Мастер-класс по голосовому пользовательскому интерфейсу
  • Основы Git и GitHub
  • Цвет и контраст в пользовательском интерфейсе

С учетом этих последних изменений ваша таблица теперь полностью готова!
