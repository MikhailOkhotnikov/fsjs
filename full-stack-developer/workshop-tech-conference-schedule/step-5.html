Шаг 5

Атрибут scope используется для указания того, является ли ячейка заголовка заголовком для строки, столбца или группы строк или столбцов. Вот пример:

Пример кода 
      
        <th scope="col">Пример заголовка</th>

Это помогает программам чтения с экрана понять взаимосвязь между ячейками заголовка и данными.

Для всех элементов th добавьте атрибут scope со значением col.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Расписание технологической конференции 2026</title>
          </head>
          <body>
            <h1>Расписание технологической конференции 2026</h1>
            <table>
              <caption>Расписание по треку и времени</caption>
              <thead>
                <tr>

                  <th>Время</th>
                  <th>Трек А</th>
                  <th>Трек Б</th>
                  <th>Трек В</th>
                  
                </tr>
              </thead>
            </table>
          </body>
        </html>
