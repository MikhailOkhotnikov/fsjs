Шаг 6

На следующих этапах вы будете создавать основную часть таблицы.

Для начала добавьте элемент tbody под элементом thead. Затем внутри элемента tbody добавьте элемент tr.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Расписание технологической конференции 2026</title>
          </head>
          <body>
            <h1>Расписание технологической конференции 2026</h1>
            <table>
              <caption>Расписание по треку и времени</caption>
              <thead>
                <tr>
                  <th scope="col">Время</th>
                  <th scope="col">Трек А</th>
                  <th scope="col">Трек Б</th>
                  <th scope="col">Трек В</th>                  
                </tr>
              </thead>
              
            </table>
          </body>
        </html>
