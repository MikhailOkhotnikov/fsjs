Шаг 8

Ещё одно значение атрибута scope — row, которое указывает, что ячейка заголовка является заголовком для всей строки.

Внутри элемента th добавьте атрибут scope со значением row.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Расписание технологической конференции 2026</title>
          </head>
          <body>
            <h1>Расписание технологической конференции 2026</h1>
            <table>
              <caption>Расписание по треку и времени</caption>
              <thead>
                <tr>
                  <th scope="col">Время</th>
                  <th scope="col">Трек А</th>
                  <th scope="col">Трек Б</th>
                  <th scope="col">Трек В</th>                  
                </tr>
              </thead>
              <tbody>
                <tr>

                  <th>9:00 AM</th>
                  
                  <td>Основной доклад: Будущее технологий</td>
                  <td>Введение в веб-разработку</td>
                  <td>UX для всех</td>
                </tr>
              </tbody>
            </table>
          </body>
        </html>
