Шаг 2

Как вы помните из предыдущего урока, списки описаний(description lists) используются для представления терминов и определений в организованном и удобном для чтения формате.

Вот пример:

Пример кода 
      
        <dl>
          <dt>HTML</dt>
          <dd>Язык гипертекстовой разметки</dd>
          <dt>CSS</dt>
          <dd>Каскадные таблицы стилей</dd>
        </dl>

Под элементом h1 создайте элемент dl. Он будет содержать список браузеров.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Список браузеров и их описания</title>
            </head>
            <body>
              <h1>Список основных веб-браузеров</h1>
                    
            </body>
        </html>
