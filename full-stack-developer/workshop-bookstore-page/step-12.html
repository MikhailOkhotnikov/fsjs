Шаг 12

Элемент button используется для создания на веб-странице интерактивных кнопок. Кнопки — это интерактивные элементы, на которые пользователи могут нажимать для выполнения действий.

Вы можете добавить несколько элементов внутри элемента div, чтобы сгруппировать связанный контент. Добавьте элемент button внутри элемента, имеющего класс card, присвойте кнопке атрибут class, установленный на btn, и текст Купить сейчас.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Книжный магазин XYZ</title>
          </head>
          <body>
            <h1>Книжный магазин XYZ</h1>
            <p>Просмотрите нашу коллекцию удивительных книг!</p>
            <div class="card-container">
              <div class="card" id="sally-adventure-book">
                <h2>Научное приключение Салли</h2>
                <p>Это эпическая история о Салли и ее собаке Рексе, путешествующих по иным мирам.</p>
                
              </div>
            </div>
          </body>
        </html>
