Шаг 6

Атрибут id добавляет уникальный идентификатор HTML-элементу. Каждый идентификатор должен быть уникальным на странице и должен использоваться только один раз.

Значения id не могут содержать пробелов и должны содержать только буквы, цифры, символы подчеркивания и тире.

Вот пример:

Пример кода: 
      
        <p id="para">пример абзаца</p>

Добавьте атрибут id к вашему элементу, имеющему класс card, и задайте его значение sally-adventure-book.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Книжный магазин XYZ</title>
          </head>
          <body>
            <h1>Книжный магазин XYZ</h1>
            <p>Просмотрите нашу коллекцию удивительных книг!</p>
            <div class="card-container">
              <div class="card">
              
              </div>
            </div>
          </body>
        </html>
