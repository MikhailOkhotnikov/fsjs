Шаг 3

Теперь улучшите структуру вашего HTML-документа, чтобы убедиться, что страница правильно закодирована и хорошо отображается на экранах всех размеров.

Добавьте атрибут lang к элементу html и установите для него значение ru. Затем внутри элемента head добавьте:

  • тег meta charset="UTF-8"
  • тег meta name="viewport" content="width=device-width, initial-scale=1.0"

Теперь добавьте элемент body под разделом head. Сюда будет помещен весь видимый контент вашей страницы.

Файл index.html 
      
        <!DOCTYPE html>
        <html>
          <head>
            <title>Книжный магазин XYZ</title>

          </head>
        </html>
