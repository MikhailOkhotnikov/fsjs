Шаг 8

Внутри элемента div можно разместить несколько элементов для группировки связанного контента. Внутри элемента с классом card-container создайте ещё один элемент div. Этот div будет представлять первую карточку книги.

Добавьте атрибут class к этому новому элементу div и задайте значение атрибута class равным card.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Книжный магазин XYZ</title>
          </head>
          <body>
            <h1>Книжный магазин XYZ</h1>
            <p>Просмотрите нашу коллекцию удивительных книг!</p>
            <div class="card-container">

            </div>
          </body>
        </html>
