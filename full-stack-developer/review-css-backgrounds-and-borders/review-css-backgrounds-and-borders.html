Обзор Списки, ссылки, фон и границы в CSS

Стилизация списков

  • Свойство line-height: Это свойство используется для создания пространства между строками текста. Допустимые значения line-height включают ключевое слово normal, числа, проценты и единицы измерения длины, такие как em.
  • Свойство list-style-type: это свойство используется для указания маркера элемента списка. Допустимые значения: кружок, диск или квадрат.

    • Вот пример использования свойства list-style-type для изменения стиля маркера списка:

    Файл index.html 
          
        <link rel="stylesheet" href="styles.css">
        <ul>
          <li>Элемент 1</li>
          <li>Элемент 2</li>
          <li>Элемент 3</li>
        </ul>
    Файл styles.css 
          
        ul {
          list-style-type: square;
        }
  • Свойство list-style-position: это свойство используется для установки положения маркера списка. Допустимы только два значения: inside и outside.

    • Вот пример, демонстрирующий разницу между inside и outside:

    Файл index.html 
          
        <link rel="stylesheet" href="styles.css">
        <ul class="inside-list">
          <li>Элемент с внутренней позицией</li>
          <li>Элемент с внутренней позицией</li>
        </ul>
        <ul class="outside-list">
          <li>Элемент с внешней позицией</li>
          <li>Элемент с внешней позицией</li>
        </ul>
    Файл styles.css 
          
        .inside-list {
          list-style-position: inside;
        }

        .outside-list {
          list-style-position: outside;
        }
  • Свойство list-style-image: это свойство используется для размещения изображения в качестве маркера элемента списка. Часто используется функция url со значением, указывающим на допустимое местоположение изображения.

Разделение элементов списка с помощью внешних отступов (margin)

  • Помимо line-height, в CSS можно использовать внешние отступы для увеличения расстояния между элементами списка и улучшения их читабельности.
  • Внешние отступы создают пространство за пределами каждого элемента списка (li), позволяя контролировать расстояние между элементами списка.
  • margin-bottom используется для создания пространства под каждым элементом списка. Например, margin-bottom: 10px; создаст отступ в 10 пикселей под каждым элементом списка.

Вот пример использования свойства margin-bottom для разделения элементов списка:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">
        <ul class="list">
          <li>Элемент 1</li>
          <li>Элемент 2</li>
          <li>Элемент 3</li>
        </ul>
Файл styles.css 
      
        .list li {
          margin-bottom: 20px;
        }

Оформление ссылок

  • Псевдокласс (pseudo-class): Это ключевое слово, добавляемое к селектору, которое позволяет выбирать элементы на основе определенного состояния. К распространенным состояниям относятся :hover, :visited и :focus.
  • Псевдокласс :link: этот псевдокласс используется для оформления ссылок, которые пользователь не открывал.

    • Вот пример использования псевдокласса :link:

    Файл index.html 
            
          <link rel="stylesheet" href="styles.css">
          <a href="/">Пример ссылки</a>
    Файл styles.css 
            
          a:link {
            color: red;
          }
  • Псевдокласс :visited: этот псевдокласс используется для оформления ссылок, которые пользователь уже посещал.
  • Псевдокласс :hover: этот псевдокласс используется для оформления элементов, когда пользователь наводит на них курсор.

    • Вот пример использования псевдокласса :hover:

    Файл index.html 
            
          <link rel="stylesheet" href="styles.css">
          <a href="/">Наведите курсор на меня!</a>
    Файл styles.css 
            
          a:hover {
            color: green;
            text-decoration: underline;
          }
  • Псевдокласс :focus: этот псевдокласс используется для оформления элемента, когда он находится в фокусе. Например, с его помощью можно стилизовать элементы input или select, когда пользователь нажимает на них или переключает на них фокус.

    • Вот пример использования псевдокласса :focus:

    Файл index.html 
            
          <link rel="stylesheet" href="styles.css">
          <a href="/">Пример ссылки</a>
    Файл styles.css 
            
          a:focus {
            outline: 2px solid orange;
          }
  • Псевдокласс :active: этот псевдокласс используется для оформления элемента, который был активирован пользователем. Например, когда пользователь нажимает на кнопку.

    • Вот пример использования псевдокласса :active:

    Файл index.html 
            
          <link rel="stylesheet" href="styles.css">
          <a href="/">Нажмите сюда!</a>
    Файл styles.css 
            
          a:active {
            color: pink;
          }

Работа с фоном

  • Свойство background-size: Это свойство используется для установки размера фона элемента. Распространенные значения включают cover, чтобы фоновое изображение покрывало весь элемент, и contain, чтобы фоновое изображение помещалось внутри элемента.

    • Вот пример использования background-size: cover: 

          
        <style>
          body {
            background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
            background-size: cover;
            background-repeat: no-repeat;
            min-height: 100px;
          }
        </style>

    Вот пример использования свойства background-size: contain: 

          
        <style>
          body {
            background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
            background-size: contain;
            background-repeat: no-repeat;
            min-height: 100px;
          }
        </style>
  • Свойство background-repeat: Это свойство используется для определения того, как фоновые изображения должны повторяться по горизонтальной и вертикальной осям. Значение по умолчанию для background-repeat — repeat, что означает, что изображение будет повторяться как по горизонтали, так и по вертикали. Вы также можете указать, что повторения быть не должно, используя значение no-repeat.

    • Вот пример использования background-repeat: no-repeat: 

          
        <style>
          body {
            background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
            background-size: contain;
            background-repeat: no-repeat;
            min-height: 100px;
          }
        </style>
  • Свойство background-position: Это свойство используется для указания положения фонового изображения. Его можно задать на определенную длину, в процентах или в виде ключевых слов, таких как top, bottom, left, right и center.

    • Вот пример использования background-position: 

          
        <style>
          body {
            background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
            background-size: contain;
            background-repeat: no-repeat;
            background-position: center top;
            min-height: 100px;
          }
        </style>
  • Свойство background-attachment: Это свойство используется для указания того, должно ли фоновое изображение прокручиваться вместе с содержимым или оставаться неподвижным. Основные значения: scroll (по умолчанию), при котором фоновое изображение прокручивается вместе с содержимым, и fixed, при котором фоновое изображение остается в том же положении на экране.
  • Свойство background-image: Это свойство используется для установки фонового изображения элемента. Вы можете установить несколько фоновых изображений одновременно и использовать в качестве значений функции url, radial-gradient или linear-gradient.

    • Вот пример использования background-image: 

          
        <style>
          body {
            background-image: url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
            min-height: 100px;
          }
        </style>
  • Свойство background: Это сокращенный способ установки всех свойств фона в одном объявлении. Вот пример установки фонового изображения и установки параметра «не повторяться»: background: no-repeat url("example-url-goes-here");

    • Вот пример использования сокращенного свойства background: 

          
        <style>
          body {
            background: center top no-repeat url("https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg");
            min-height: 100px;
          }
        </style>
  • Хорошая контрастность цветов фона и переднего плана: Чтобы текст был читабельным, важно обеспечить хорошую контрастность цветов фона и переднего плана. Согласно Руководству по обеспечению доступности веб-контента (Web Content Accessibility Guidelines, WCAG), минимальный коэффициент контрастности для обычного текста должен составлять 4,5:1, а для крупного — 3:1.

Работа с границами

  • Свойство border-top: Это свойство используется для задания стилей верхней границы элемента. border-top: 3px solid blue; устанавливает сплошную синюю границу шириной 3 пикселя по верхней стороне элемента.
  • Свойство border-right: Это свойство используется для настройки стилей правой границы элемента. border-right: 2px solid red; устанавливает сплошную красную границу шириной 2 пикселя с правой стороны элемента.
  • Свойство border-bottom: это свойство используется для настройки стилей нижней границы элемента. border-bottom: 1px dashed green; устанавливает зеленую пунктирную границу шириной 1 пиксель по нижней стороне элемента.
  • Свойство border-left: это свойство используется для настройки стилей левой границы элемента. border-left: 4px dotted orange; устанавливает пунктирную оранжевую границу шириной 4 пикселя с левой стороны элемента.

    • Вот пример использования отдельных свойств границ:

    Файл index.html 
            
          <link rel="stylesheet" href="styles.css">
          <div class="bordered-box">Коробка с разными границами</div>
    Файл styles.css 
            
          .bordered-box {
            border-top: 3px solid blue;
            border-right: 2px solid red;
            border-bottom: 1px dashed green;
            border-left: 4px dotted orange;
            padding: 20px;
          }
  • Свойство border: сокращенное свойство для установки ширины, стиля и цвета границы элемента. border: 1px solid black; устанавливает сплошную черную границу шириной 1 пиксель.

    • Вот пример использования сокращенного свойства border:

    Файл index.html 
            
          <link rel="stylesheet" href="styles.css">
          <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Милый кот лежит на спине.">
    Файл styles.css 
            
          img {
            border: 2px solid red;
          }
  • Свойство border-radius: это свойство используется для создания скругленных углов у границы элемента.

    • Вот пример использования border-radius:

    Файл index.html 
            
          <link rel="stylesheet" href="styles.css">
          <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Милый кот лежит на спине.">
    Файл styles.css 
            
          img {
            border: 2px solid black;
            border-radius: 10px;
          }
  • Свойство border-style: это свойство используется для установки стиля границы элемента. Допустимые значения: solid, dashed, dotted и double.

    • Вот пример с использованием разных значений border-style:

    Файл index.html 
            
          <link rel="stylesheet" href="styles.css">
          <div class="solid-border">Сплошная граница</div>
          <div class="dashed-border">Пунктирная граница</div>
          <div class="dotted-border">Точечная граница</div>
          <div class="double-border">Двойная граница</div>
    Файл styles.css 
            
          .solid-border {
            border: 3px solid blue;
            margin-bottom: 10px;
            padding: 10px;
          }

          .dashed-border {
            border: 3px dashed red;
            margin-bottom: 10px;
            padding: 10px;
          }

          .dotted-border {
            border: 3px dotted green;
            padding: 10px;
          }

          .double-border {
            border: 3px double blueviolet;
            padding: 10px;
          }

Градиенты

  • Функция linear-gradient(): эта функция CSS используется для создания перехода между несколькими цветами по прямой линии.

    • Вот пример использования функции linear-gradient():

    Файл index.html 
            
          <link rel="stylesheet" href="styles.css">
          <div class="linear-gradient"></div>
    Файл styles.css 
            
          .linear-gradient {
            background: linear-gradient(to right, red, blue);
            height: 40vh;
          }
  • Функция radial-gradient(): эта функция CSS создает изображение, которое расходится от определенной точки, например круга или эллипса, и постепенно переходит от одного цвета к другому.

    • Вот пример использования функции radial-gradient():

    Файл index.html 
            
          <link rel="stylesheet" href="styles.css">
          <div class="radial-gradient"></div>
    Файл styles.css 
            
          .radial-gradient {
            background: radial-gradient(circle, red, blue);
            height: 40vh;
          }
