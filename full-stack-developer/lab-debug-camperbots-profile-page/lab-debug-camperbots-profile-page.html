Отладка страницы профиля Camperbot's

Camperbot пытается создать страницу профиля. Они попросили друга просмотреть их код, и тот сказал, что в нем есть некоторые ошибки.

Ваша задача — исправить все ошибки Camperbot, чтобы они могли продолжить создание своей страницы профиля. Заполните пункты в пользовательских историях ниже и нажмите «Запустить тесты», чтобы увидеть, исправили ли вы все ошибки.

Истории пользователей:

  1. Camperbot пытается использовать элемент heading2, но этот элемент не существует. Исправьте эти теги, чтобы он использовал правильный элемент заголовка второго уровня.
  2. Camperbot пытается добавить два абзаца с pp, но их тоже нет. Исправьте их, чтобы они использовали правильные теги абзацев.
  3. Camperbot использует элемент h3 для подзаголовка Background and Interests, но в нем есть синтаксическая ошибка. Найдите проблему и устраните ее.
Файл index.html 
      
        <h1>Привет от Camperbot!</h1>

        <heading2>Обо мне</heading2>

        <pp>Меня зовут Camperbot, и я люблю узнавать новое.</pp>

        <h3>Биография и интересы<h3/>
        <pp>Мне нравится решать головоломки.</pp>

Тесты

  1. На странице должен быть ровно один элемент h1.
  2. Не следует удалять элемент <h1>Привет от Camperbot!</h1> со страницы.
  3. На странице должен быть ровно один элемент h2.
  4. У вас должен быть элемент h2 с текстом Обо мне. Вот пример: <h2>Текст здесь</h2>
  5. У вас должен быть элемент абзаца с текстом Меня зовут Camperbot, и я люблю узнавать новое. Вот пример: <p>Текст здесь</p>
  6. У вас должен быть элемент абзаца с текстом Мне нравится решать головоломки. Вот пример: <p>Текст здесь</p>
  7. На странице должно быть ровно два элемента абзаца.
  8. На странице должен быть ровно один элемент h3.
  9. У вас должен быть элемент h3 с текстом Биография и интересы. Вот пример: <h3>Текст здесь</h3>
  10. Всего на странице должно быть ровно пять элементов.
