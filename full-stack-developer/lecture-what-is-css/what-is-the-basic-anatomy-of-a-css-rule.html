Какова основная структура правила CSS?

CSS отвечает за стили веб-страницы. Все эти стили состоят из различных правил CSS.

Правило CSS состоит из двух основных частей: селектора(selector) и блока объявлений(declaration block).

Давайте рассмотрим базовый синтаксис: 

    
      selector {
        property: value;
      }

Селектор — это шаблон, используемый в CSS для идентификации и выбора конкретных HTML-элементов для стилизации.

Примерами селекторов являются селекторы типов(type selectors), селекторы классов(class selectors) и селекторы идентификаторов(ID selectors).

Фигурные скобки, используемые в базовом синтаксисе, называются блоком объявления. Блок объявления применяет набор стилей к заданному селектору или селекторам.

Внутри блока объявлений находится ряд объявлений. Каждое объявление состоит из свойства и значения.

Свойство(property) — это идентификатор CSS, указывающий, какой элемент стиля применяется. Примером свойства может служить свойство background-color.

Значением(value) будет конкретная настройка, примененная к этому свойству. Например, если свойство называется background-color, значением может быть purple, что устанавливает фиолетовый цвет фона.

После каждого названия свойства необходимо поставить двоеточие, а после каждого значения — точку с запятой.

Теперь, когда вы знаете синтаксис правил CSS, давайте рассмотрим пример. Включите интерактивный редактор и перейдите на вкладку styles.css, чтобы увидеть код CSS.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">
        <h1>Изучение CSS</h1>

        <p>Пример элемента абзаца</p>
        <p>Ещё один пример элемента абзаца</p>
Файл styles.css 
      
        p {
          color: blue;
        }

В этом правиле CSS селектор типа применяется ко всем элементам абзаца на странице.

Внутри блока объявлений находится одно объявление.

В объявлении используется свойство color со значением blue. Это изменит цвет текста для всех элементов абзаца на синий.

Если вы хотите применить один и тот же набор стилей к нескольким селекторам, вы можете создать список селекторов, разделив каждый селектор запятой.

Вот пример оформления нескольких селекторов (для взаимодействия с примером включите интерактивный редактор):

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <h1 id="title">Пример заголовка</h1>
        <h2 class="subheading">Пример подзаголовка</h2>
        <p>Данный абзац не затрагивается селектором.</p>
Файл styles.css 
      
        #title,
        .subheading {
          color: navy;
        }

В этом списке селекторов есть селектор id, указывающий на HTML-элемент со значением id, равным title. Все селекторы id должны начинаться с символа решетки #.

Затем следует запятая, за которой следует селектор class, который выбирает все HTML-элементы со значением класса subheading. Все селекторы классов должны начинаться с точки .

Весь список селекторов получит одинаковое оформление, при этом цвет текста будет установлен на темно-синий(navy).
