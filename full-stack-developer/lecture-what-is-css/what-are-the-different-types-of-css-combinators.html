Какие существуют типы комбинаторов CSS?

CSS-комбинаторы используются для определения взаимосвязи между селекторами в CSS. Они помогают выбирать элементы на основе их взаимосвязи с другими элементами, что позволяет более точно и эффективно стилизовать их.

Мы обсудим некоторые основные комбинаторы CSS и варианты их использования, начиная с комбинатора потомков(descendant combinator).

Комбинатор потомков используется для выбора элементов, соответствующих второму селектору, если они вложены в родительский элемент, соответствующий первому селектору. Родительским элементом может быть родительский элемент или родительский элемент родительского элемента.

Чтобы лучше понять, как это работает, давайте рассмотрим пример.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <figure>
          <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Спящий кот">
          <figcaption>Спящий кот с границей</figcaption>
        </figure>
Файл styles.css 
      
        figure img {
          border: 4px solid red;
        }

В приведенном выше примере мы используем комбинатор потомков, чтобы выбрать все элементы img внутри элементов figure и применить сплошную(solid) красную(red) рамку(border) со всех четырех сторон.

Обратите внимание, что между селекторами родительского и дочернего элементов стоит пробел.

В этом случае родительским элементом будет figure, а дочерним — img.

Если внутри элемента figure находится несколько изображений, то использование комбинатора потомков позволит создать сплошную черную рамку вокруг каждого из этих изображений.

Другой тип комбинатора — это дочерний комбинатор (child combinator).

В CSS комбинатор дочерних элементов (>) используется для выбора элементов, являющихся непосредственными потомками указанного родительского элемента.

Этот комбинатор применяется только к элементам с определенным родительским элементом, что делает ваши правила CSS более точными и предотвращает непреднамеренное оформление элементов, находящихся глубже вложенных друг в друга.

Давайте рассмотрим следующий пример HTML: 

    
      <div class="container">
        <p>Первый</p>
        <div>
          <p>Второй</p>
        </div>
        <div>
          <p>Третий</p>
        </div>
      </div>

В приведенной выше HTML-структуре класс container применяется к элементу div.

Внутри этого контейнера находится дочерний элемент p (Первый), за которым следуют два дополнительных элемента div, каждый из которых содержит элемент p (Второй и Третий).

Первый элемент p является прямым потомком элемента .container, в то время как два других элемента p вложены в другие элементы div, являясь их более глубокими потомками.

Чтобы применить стили только к непосредственному дочернему элементу класса container, можно использовать комбинатор дочерних элементов следующим образом:

Файл index.html 
      
        <div class="container">
          <p>Первый</p>
          <div>
            <p>Второй</p>
          </div>
          <div>
            <p>Третий</p>
          </div>
        </div>
Файл styles.css 
      
        .container > p {
          color: blue;
        }

В приведенном выше примере мы воздействуем только на непосредственный дочерний элемент класса container. Это придаст дочернему элементу синий(blue) цвет текста.

Поскольку два других элемента абзаца(p) вложены в элементы div, они не считаются прямыми дочерними элементами класса container и не получат синий цвет текста.

Еще одним распространенным комбинатором является комбинатор ближайших соседей(next-sibling combinator).

Комбинатор ближайших соседей (+) в CSS выбирает элемент, непосредственно следующий за указанным соседним элементом. Этот комбинатор полезен, когда вы хотите применить стили к элементу, который непосредственно следует за другим элементом, что позволяет целенаправленно стилизовать элемент в зависимости от его положения относительно соседних элементов.

Давайте рассмотрим следующий пример HTML: 

    
      <figure>
        <img
          src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
          alt="Милый рыжий кот лежит на спине."
        />
        <figcaption>Милый рыжий кот лежит на спине.</figcaption>
      </figure>

Здесь у нас есть элемент figure, содержащий элемент img, за которым следует элемент figcaption. Элемент figcaption является непосредственным соседом элемента img.

Если вы хотите добавить черную рамку вокруг элемента figcaption, вы можете использовать комбинатор ближайших соседей следующим образом:

Файл index.html 
      
        <figure>
          <img
            src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg"
            alt="Милый рыжий кот лежит на спине."
          />
          <figcaption>Милый рыжий кот лежит на спине.</figcaption>
        </figure>
Файл styles.css 
      
        img + figcaption {
          border: 4px solid black;
        }

В этом примере комбинатор ближайших соседей (+) выбирает элемент figcaption, который непосредственно следует за элементом img. Применённое правило CSS добавляет сплошную чёрную рамку толщиной 4 пикселя(4px solid black border) вокруг элемента figcaption.

Еще одним распространенным комбинатором является комбинатор последующих одноуровневых элементов(subsequent-sibling combinator).

В CSS комбинатор последующих одноуровневых элементов (~) выбирает все соседние элементы указанного элемента, которые следуют за ним.

В отличие от комбинатора ближайших соседей, который нацелен только на непосредственно следующий за ним элемент, комбинатор последующих одноуровневых элементов (~) нацелен на любые элементы, следующие за указанным элементом, что обеспечивает большую гибкость в стилизации.

Давайте рассмотрим следующий пример HTML: 

    
      <div class="container">
        <h2>Подзаголовок</h2>
        <p>Первый абзац.</p>
        <p>Второй абзац.</p>
        <p>Третий абзац.</p>
        <p>Еще один абзац.</p>
      </div>

В этой HTML-структуре у нас есть элемент h2, за которым следуют четыре элемента абзаца. Элементы абзаца являются соседними элементами по отношению к элементу h2.

Если вы хотите стилизовать все элементы абзаца, следующие за элементом h2, то можете использовать комбинатор последующих одноуровневых элементов следующим образом:

Файл index.html 
      
        <div class="container">
          <h2>Подзаголовок</h2>
          <p>Первый абзац.</p>
          <p>Второй абзац.</p>
          <p>Третий абзац.</p>
          <p>Еще один абзац.</p>
        </div>
Файл styles.css 
      
        h2 ~ p {
          color: green;
        }

В этом примере все элементы абзаца, следующие за элементом h2, будут иметь зеленый цвет текста.

Комбинатор последующих одноуровневых элементов (~) нацелен на все соседние абзацы, которые появляются после элемента h2, независимо от того, являются ли они непосредственными соседями.

В заключение отметим, что понимание и использование различных комбинаторов CSS позволяет применять точные стили к элементам HTML, повышая контроль и гибкость вашего дизайна.

Овладев этими селекторами, вы сможете создавать более сложные и целенаправленные правила стилизации, которые улучшат как внешний вид, так и функциональность ваших веб-страниц.
