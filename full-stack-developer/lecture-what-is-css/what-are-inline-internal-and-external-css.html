Что такое встроенные, внутренние и внешние CSS-стили, и когда следует использовать каждый из них?

CSS можно применять к веб-странице тремя основными способами: встроенным, внутренним или внешним.

Каждый метод имеет свои области применения, преимущества и ограничения, и знание того, когда использовать каждый из них, имеет важное значение для написания чистого, эффективного и поддерживаемого кода.

Давайте разберем три типа CSS и когда их следует использовать.

Встроенные CSS-стили пишутся непосредственно внутри HTML-элемента с помощью атрибута style. Они применяют стили к конкретному элементу.

Вот пример использования встроенных CSS-стилей: 

    
      <p style="color: green;">Это абзац с встроенным стилем.</p>

В этом примере мы используем атрибут style, чтобы сделать текст абзаца зеленым(green).

Встроенные CSS-стили обычно используются для быстрого применения разовых стилей или для переопределения других стилей для конкретного элемента.

Однако в большинстве случаев этого следует избегать, поскольку это может загромоздить HTML-код и затруднить его сопровождение.

В большинстве случаев лучше использовать внутренние или внешние CSS-стили, чтобы ваши стили были организованы и их было легко поддерживать.

Внутренние CSS-стили записываются внутри тегов style в разделе head HTML-документа. Они применяют стили ко всей странице и полезны, когда необходимо стилизовать только один документ.

Вот пример внутреннего CSS: 

    
      <head>
        <style>
          p {
            color: blue;
          }
        </style>
      </head>
      <body>
        <p>Этот абзац оформлен с использованием внутренних CSS-стилей.</p>
      </body>

В приведенном выше примере внутренний CSS применяет синий текст ко всем элементам p на странице.

Встроенные CSS-стили лучше всего использовать, когда необходимо применить стили к конкретной странице, а не ко многим другим. Это полезно для одностраничных сайтов или когда стили не нужно повторно использовать на других страницах.

Однако есть и некоторые недостатки, например, отсутствие возможности повторного использования кода на разных страницах. Кроме того, как и встроенные CSS-стили, он смешивает HTML и CSS, что затрудняет поддержку кода в крупных проектах.

Внешние CSS-стили пишутся в отдельном файле .css и связываются с HTML-документом с помощью элемента link в разделе head.

Это позволяет стилизовать несколько страниц единообразным образом и является предпочтительным методом в профессиональной веб-разработке.

Вот пример оформления всех элементов абзаца:

Файл index.html 
      
        <head>
          <link rel="stylesheet" href="styles.css">
        </head>
        <body>
          <p>Этот абзац оформлен с использованием внешнего CSS-стиля.</p>
        </body>
Файл styles.css 
      
        p {
          color: red;
        }

В предыдущем уроке вы узнали, что элемент link имеет атрибут rel, который определяет связь между текущим документом и связанным ресурсом, например, ссылку на таблицу стилей или внешний ресурс.

С другой стороны, атрибут href указывает URL-адрес связанного ресурса, показывая, откуда этот ресурс должен быть получен.

Внешние CSS-стили идеально подходят для крупных проектов, где необходимо поддерживать единообразный стиль на нескольких страницах.

Это способствует разделению задач, поскольку HTML отвечает за структуру, а CSS — за стилизацию, что делает код более удобным для сопровождения и масштабируемым.

Понимание того, когда следует использовать каждый тип CSS, имеет решающее значение для эффективной и результативной веб-разработки.

В большинстве случаев использование внешних CSS-стилей является предпочтительным подходом, особенно для крупных и сложных проектов.
