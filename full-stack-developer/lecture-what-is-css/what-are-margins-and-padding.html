Что такое margin и padding, и как они работают?

Внешний отступ (margin) и внутренний отступ (padding) — это важные свойства CSS для создания хорошо структурированных, читаемых и визуально привлекательных веб-страниц.

Внешний отступ (margin) контролирует пространство за пределами элемента, помогая отделить его от других элементов и определить структуру макета, а внутренний отступ (padding) контролирует пространство внутри элемента, улучшая читаемость контента и эстетическую привлекательность.

Чтобы лучше понять разницу между margin и padding, давайте рассмотрим несколько примеров.

Вот пример HTML-кода, отображающего три элемента абзаца на странице: 

    
      <p>Первый абзац</p>
      <p>Второй абзац</p>
      <p>Третий абзац</p>

Чтобы задать отступ сверху для каждого элемента абзаца, можно использовать свойство margin-top следующим образом:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <p>Первый абзац</p>
        <p>Второй абзац</p>
        <p>Третий абзац</p>
Файл styles.css 
      
        p {
          margin-top: 20px;
          border: 2px solid black;
        }

В этом примере мы применяем отступ в 20 пикселей только к верхней части каждого элемента абзаца.

Кроме того, со всех четырех сторон имеется сплошная черная рамка, чтобы поля были лучше видны.

Существует четыре различных свойства для внешних отступов: margin-top(сверху), margin-right(справа), margin-bottom(снизу) и margin-left(слева).

Вот пример использования всех четырех свойств:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <span>Первый абзац</span>
        <p>Второй абзац</p>
        <span>Третий абзац</span>
Файл styles.css 
      
        p {
          margin-top: 10px;
          margin-right: 20px;
          margin-bottom: 30px;
          margin-left: 40px;
          border: 2px solid black;
        }

В этом примере мы задаем значения внешних отступов со всех четырех сторон элемента абзаца. Также добавлена ​​сплошная черная рамка, чтобы вы могли лучше видеть поля.

Ещё один способ использования свойства margin — сокращённая запись. Вы можете указать одно, два, три или четыре значения одновременно. Давайте рассмотрим эти варианты вместе.

При использовании единственного значения в сокращенной записи margin, это же значение будет применено ко всем четырем сторонам целевого элемента.

Вот пример использования одного значения в сокращенной записи полей:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <span>Первый абзац</span>
        <p>Второй абзац</p>
        <span>Третий абзац</span>
Файл styles.css 
      
        p {
          margin: 10px;
        }

В этом примере кода будет применен margin в 10px ко всем четырем сторонам элемента абзаца.

При использовании двух значений первое значение применяется к верхней(top) и нижней(bottom) сторонам элемента, а второе — к левой(left) и правой(right) сторонам.

Вот пример использования двух значений для сокращенной записи margin:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <span>Первый абзац</span>
        <p>Второй абзац</p>
        <span>Третий абзац</span>
Файл styles.css 
      
        p {
          margin: 10px 20px;
        }

Таким образом, верхний и нижний внешний отступ элемента абзаца устанавливаются на 10px, а левый и правый внешний отступ — на 20px.

Если указаны три значения, первое значение применяется к верхнему(top) внешнему отступу, второе — к левому(left) и правому(right) внешнему отступу, а третье — к нижнему(bottom) внешнему отступу.

Вот пример для лучшего понимания:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <span>Первый абзац</span>
        <p>Второй абзац</p>
        <span>Третий абзац</span>
Файл styles.css 
      
        p {
          margin: 10px 20px 30px;
        }

Это устанавливает внешний отступ в 10px сверху, 20px слева и справа, и 30px снизу.

Использование четырех значений дает вам больше контроля, поскольку вы можете независимо задать значение внешнего отступа для каждой стороны целевого элемента.

Первое значение нацелено на верхнюю точку(top), второе — на правую(right), третье — на нижнюю(bottom), а четвёртое — на левую(left).

Вот пример использования сокращенной записи margin с четырьмя значениями:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <span>Первый абзац</span>
        <p>Второй абзац</p>
        <span>Третий абзац</span>
Файл styles.css 
      
        p {
          margin: 10px 20px 30px 40px;
        }

Это устанавливает внешние отступы в 10 пикселей сверху, 20 пикселей справа, 30 пикселей снизу и 40 пикселей слева.

Свойство внутренний отступ(padding) используется для добавления пространства внутри элемента, между содержимым и его границей.

Подобно свойству margin, четыре свойства padding — это padding-top, padding-right, padding-bottom и padding-left.

Вот пример того, как задать внутренние отступы(padding) для элемента абзаца:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <span>Первый абзац</span>
        <p>Второй абзац</p>
        <span>Третий абзац</span>
Файл styles.css 
      
        p {
          padding-top: 10px;
          padding-right: 20px;
          padding-bottom: 30px;
          padding-left: 40px;
          border: 2px solid black;
        }

Это устанавливает внутренние отступы: 10 пикселей сверху, 20 пикселей справа, 30 пикселей снизу и 40 пикселей слева.

Как видите, padding применяются к содержимому, находящемуся внутри границы, в отличие от margin, которые применяются к содержимому снаружи границы.

Как и в случае со свойством margin, вы также можете использовать сокращенную запись для свойства padding.

В качестве сокращенного свойства padding можно указать одно, два, три или четыре значения.

Вот пример использования сокращенной записи padding для элемента абзаца, о котором говорилось ранее:

Файл index.html 
      
        <link rel="stylesheet" href="styles.css">

        <span>Первый абзац</span>
        <p>Второй абзац</p>
        <span>Третий абзац</span>
Файл styles.css 
      
        p {
          padding: 10px 20px 30px 40px;
          border: 2px solid black;
        }

В приведенном примере, используя сокращенную запись, код установит внутренние отступы (padding) равными 10 пикселям сверху, 20 пикселям справа, 30 пикселям снизу и 40 пикселям слева от элемента абзаца.
