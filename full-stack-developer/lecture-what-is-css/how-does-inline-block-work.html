Как работает элемент Inline-Block и чем он отличается от элементов Inline и Block?

При работе с CSS часто встречаются три различных типа поведения элементов при отображении: inline, block и inline-block.

Каждое из этих свойств влияет на позиционирование элементов и на то, как они взаимодействуют с другими элементами на странице.

В этом уроке мы сосредоточимся на том, как работает свойство inline-block(строчно-блочный) и чем оно отличается от свойств строчных и блочных элементов.

Блочные элементы ведут себя как большие контейнеры или «блоки», занимающие всю ширину родительского контейнера. Они всегда начинаются с новой строки, и их высота и ширина могут быть отрегулированы.

Встроенные(строчные) элементы занимают ровно столько места, сколько необходимо. Они органично вписываются в окружающий контент и не переходят на новую строку.

Свойство inline-block представляет собой гибрид этих двух типов поведения. Подобно строчным элементам, элементы inline-block остаются в текстовом потоке, не начинаясь с новой строки.

Однако, в отличие от строчных элементов, ширину и высоту строчного блочного элемента можно регулировать так же, как и у блочных элементов.

Вкратце, ключевое различие между inline и inline-block элементами заключается в том, что inline элементы не позволяют контролировать их размер, тогда как inline-block элементы позволяют полностью контролировать размеры, оставаясь при этом на одной линии с остальным содержимым.

Давайте рассмотрим пример.

Файл index.html 
      
        <link href="styles.css" rel="stylesheet">

        <div class="container">
          <span class="inline-block-element element1">Inline-Block</span>
          <span class="inline-block-element element2">Inline-Block</span>
        </div>
Файл styles.css 
      
        .inline-block-element {
          display: inline-block;
          width: 150px;
          height: 100px;
        }

        .element1 {
          background-color: lightblue;
        }

        .element2 {
          background-color: lightgreen;
        }

В приведенном выше примере у нас есть div с классом container. Внутри этого div находятся два элемента span.

Каждому элементу span задано свойство display:inline-block, а также ширина и высота.

Элементы inline-block будут отображаться рядом друг с другом, поскольку они ведут себя как строчные элементы, но при этом имеют заданную ширину и высоту, что придает им блочные свойства.

Но если удалить свойство display: inline-block, ни height, ни width не будут применены, даже если вы четко их задали.

Вот исправленный CSS-код:

Файл index.html 
      
        <link href="styles.css" rel="stylesheet">

        <div class="container">
          <span class="inline-block-element element1">Span element</span>
          <span class="inline-block-element element2">Span element</span>
        </div>
Файл styles.css 
      
        .inline-block-element {
          width: 150px;
          height: 100px;
        }

        .element1 {
          background-color: lightblue;
        }

        .element2 {
          background-color: lightgreen;
        }

В этом коде мы удалили свойство display: inline-block;, но оставили все остальное без изменений. Здесь элементы span возвращаются к своему поведению по умолчанию, становясь строчными элементами.

В результате указанные ширина и высота игнорируются, и элементы занимают только то пространство, которое необходимо для их содержимого.

Понимание принципа работы inline-block полезно, поскольку их можно использовать для создания макетов, требующих как выравнивания, так и контроля размеров в рамках непрерывного текстового потока.
