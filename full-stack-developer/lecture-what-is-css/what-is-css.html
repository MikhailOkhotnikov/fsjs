Что такое CSS и какова его роль в веб-разработке?

CSS(Cascading Style Sheets), что расшифровывается как «каскадные таблицы стилей», является важнейшим компонентом современной веб-разработки. Это язык разметки, используемый для применения стилей к HTML. Проще говоря, если HTML — это структура веб-страницы, то CSS — это то, что делает её красивой.

Вот пример оформления панели навигации.

ПРИМЕЧАНИЕ: Пока не стоит пытаться понять код CSS. Вы узнаете, как всё это работает, на будущих уроках и мастер-классах. Это лишь для того, чтобы дать вам представление о том, что можно сделать с помощью небольшого количества CSS.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css" />

        <nav class="navbar">
            <ul class="nav-links">
                <li><a href="#">Главная</a></li>
                <li><a href="#">О нас</a></li>
                <li><a href="#">Услуги</a></li>
                <li><a href="#">Контакты</a></li>
            </ul>
        </nav>
Файл styles.css 
      
        * {
          margin: 0;
          padding: 0;
          box-sizing: border-box;
        }

        body {
          font-family: 'Segoe UI', sans-serif;
          background-color: #f4f4f4;
        }

        .navbar {
          display: flex;
          justify-content: space-between;
          align-items: center;
          background-color: #333;
          padding: 1rem 2rem;
          color: white;
        }

        .navbar .logo {
          font-size: 1.5rem;
          font-weight: bold;
        }

        .nav-links {
          list-style: none;
          display: flex;
          gap: 1.5rem;
        }

        .nav-links a {
          color: white;
          text-decoration: none;
          font-weight: 500;
          transition: color 0.3s ease;
        }

        .nav-links a:hover {
          color: #ff9800;
        }

Основная роль CSS заключается в разделении содержимого веб-страницы от её визуального представления. Такое разделение позволяет веб-разработчикам создавать более гибкие и поддерживаемые веб-сайты.

С помощью CSS вы можете управлять расположением элементов, цветами, шрифтами и общим визуальным оформлением веб-страниц, не изменяя структуру HTML.

Рассмотрим простую аналогию. Если представить веб-сайт как дом, то HTML — это фундамент и каркас, а CSS — краска, обои и украшения, которые делают дом визуально привлекательным и уникальным.

CSS работает путем выбора HTML-элементов и применения к ним стилей.

Здесь представлен пример, демонстрирующий нестилизованный и стилизованный элементы button. Для взаимодействия с этим примером необходимо включить интерактивный редактор.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css" />

        <button class="regular-btn">Кнопка без стиля</button>
        <button class="round-btn">Закруглённая кнопка</button>
Файл styles.css 
      
        .round-btn {
          font-size: 1rem;
          font-family: 'Segoe UI', sans-serif;
          border: none;
          cursor: pointer;
          transition: background-color 0.3s ease, transform 0.2s ease;
          background-color: #2ecc71;
          color: white;
          border-radius: 50px;
          padding: 0.6rem 1.6rem;
        }

        .round-btn:hover {
          background-color: #27ae60;
          transform: translateY(-2px);
        }

Эти стили могут включать такие свойства, как цвет, размер шрифта и многое другое. Изменяя эти свойства, вы можете кардинально изменить внешний вид веб-страницы, не меняя её содержимого.

Одним из самых мощных преимуществ CSS является его способность создавать адаптивный дизайн.

Это означает, что с помощью CSS вы можете сделать так, чтобы ваш веб-сайт отлично выглядел на любом устройстве, будь то настольный компьютер, планшет или смартфон.

CSS позволяет настраивать макеты, размеры шрифтов и другие визуальные элементы в зависимости от размера экрана устройства, на котором просматривается веб-сайт.

Попробуйте включить интерактивный редактор и отрегулировать размер окна предварительного просмотра, чтобы увидеть, как макет адаптируется к размеру экрана.

Файл index.html 
      
        <link rel="stylesheet" href="styles.css" />

        <header class="header">
            <h1>Добро пожаловать на мой адаптивный сайт!</h1>
            <p>Этот макет адаптируется к размеру вашего экрана!</p>
        </header>

        <main class="content">
            <section class="box">Блок 1</section>
            <section class="box">Блок 2</section>
            <section class="box">Блок 3</section>
        </main>
Файл styles.css 
      
        body {
          margin: 0;
          font-family: 'Segoe UI', sans-serif;
          background-color: #f9f9f9;
          color: #333;
        }

        .header {
          background-color: #4CAF50;
          color: white;
          padding: 2rem;
          text-align: center;
        }

        .content {
          display: flex;
          gap: 1rem;
          padding: 1rem;
          justify-content: center;
        }

        .box {
          flex: 1;
          min-width: 200px;
          background-color: #ddd;
          padding: 2rem;
          text-align: center;
          font-size: 1.2rem;
          border-radius: 8px;
          transition: background-color 0.3s;
        }

        .box:hover {
          background-color: #ccc;
        }

        /* Адаптивный дизайн: для планшетов и более старых моделей (≤ 768px) */
        @media (max-width: 768px) {
          .content {
            flex-direction: column;
            align-items: center;
          }

          .box {
            width: 90%;
            font-size: 1.1rem;
          }
        }

        /* Адаптивный дизайн: для смартфонов (≤ 480px) */
        @media (max-width: 480px) {
          .header {
            padding: 1.5rem 1rem;
          }

          .header h1 {
            font-size: 1.5rem;
          }

          .header p {
            font-size: 1rem;
          }

          .box {
            font-size: 1rem;
            padding: 1.5rem;
          }
        }

Еще одна важная особенность CSS — его каскадный характер, отсюда и слово «каскадный» в его названии.

Это означает, что стили могут наследоваться и переопределяться, что позволяет создавать иерархическую структуру оформления. Подробнее о том, как это работает, вы узнаете в будущих уроках.

CSS также поддерживает использование внешних таблиц стилей.

Это означает, что вы можете хранить все свои правила оформления в отдельном файле, который затем можно связать с несколькими HTML-страницами.

Эта функция значительно повышает удобство сопровождения веб-сайтов, особенно крупных. Вместо того чтобы менять стили на каждой отдельной странице, вы можете внести изменения в один CSS-файл, которые повлияют на все связанные страницы.

В мире веб-разработки CSS играет жизненно важную роль в создании визуально привлекательных, адаптивных и удобных для пользователя веб-сайтов.

Это позволяет разработчикам преобразовывать простые HTML-документы в привлекательные веб-интерфейсы, которые привлекают внимание пользователей и улучшают их взаимодействие с веб-контентом.
