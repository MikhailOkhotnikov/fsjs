В чём разница между строчными и блочными элементами в CSS?

В HTML и CSS элементы классифицируются как строчные(inline elements) или блочные(block-level elements), и эта классификация определяет их поведение в макете документа.

Понимание этой разницы имеет решающее значение для управления тем, как ваш контент отображается на веб-странице.

Блочные элементы — это элементы, которые по умолчанию занимают всю доступную им ширину, растягиваясь на всю ширину своего контейнера.

Эти элементы всегда начинаются с новой строки и сдвигают остальное содержимое на следующую строку, создавая «блок» контента.

Для блочных элементов по умолчанию применяется свойство CSS display: block;. Это свойство гарантирует, что элемент растянется на всю ширину контейнера и отобразится на новой строке.

К числу распространенных блочных элементов относятся элементы div, абзацы(paragraphs), заголовки(headings), упорядоченные списки(ordered lists), неупорядоченные списки(unordered lists) и элементы разделов(section).

Вот пример кода блочного элемента: 

    
      <p style="border: 2px solid red;">
        Первый абзац
      </p>
      <p>Второй абзац</p>

В этом примере у нас есть два элемента абзаца, причем первый из них имеет красную рамку.

Два элемента абзаца не находятся на одной строке, поскольку по умолчанию являются блочными элементами.

Таким образом, оба элемента абзаца займут всю ширину своего контейнера, которым в данном случае является элемент body.

Блочные элементы идеально подходят, когда вы хотите, чтобы контент располагался вертикально, например, абзацы, разделы или большие блоки контента. Они обычно используются для определения макета и структуры веб-страницы.

В отличие от блочных элементов, строчные элементы занимают ровно столько ширины, сколько им необходимо, и не начинаются с новой строки. Эти элементы располагаются внутри контента, позволяя тексту и другим строчным элементам появляться рядом с ними.

Для строчных элементов по умолчанию применяется свойство CSS display: inline;. Это свойство гарантирует, что элемент останется в пределах потока содержимого и не будет переноситься на новую строку.

К распространенным строчным элементам относятся span, anchor и img.

Вот пример для лучшего понимания строчных элементов: 

    
      <p>Это слово внутри абзаца
        <span style="color: red; border: 2px solid green;">красное</span>.
      </p>

В этом примере у нас есть элемент span, вложенный в элемент p. Элемент span имеет красный(red) цвет текста с зеленой(green) рамкой вокруг него, чтобы вы могли лучше видеть выделенное слово.

Как видите, элемент span занимает ровно столько места, сколько занимает слово красное, и не переносит следующее содержимое на новую строку.

Встроенные элементы лучше всего использовать для оформления небольших фрагментов текста или контента внутри строки, например, для выделения слова, создания гиперссылок или применения определенных стилей к частям абзаца.
