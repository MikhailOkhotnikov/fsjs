Шаг 4

Если источником цитаты является веб-сайт, вы можете указать его с помощью атрибута cite. Значение этого атрибута должно быть корректным URL-адресом. Хотя этот атрибут не меняет внешний вид цитаты, он очень полезен для предоставления программам чтения с экрана и поисковым системам дополнительной информации о цитате.

Вот пример элемента встроенной цитаты с атрибутом cite:

Пример кода 
      
        <p>
          Нэнси сказала: <q cite="https://example.com">Учиться - это весело!</q>
        </p>

Добавьте атрибут cite к элементу встроенной цитаты с помощью этого URL: https://www.freecodecamp.org/news/learn-to-code-book/

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>Советы Куинси по поиску работы разработчика</title>
            </head>
            <body>
              <h1>Советы Куинси по получению работы разработчика</h1>
              <p>
                Научиться программировать сложно, но, как говорит Куинси Ларсон, 
                <q>Вы можете стать разработчиком.</q>
                
              </p>
            </body>
        </html>
