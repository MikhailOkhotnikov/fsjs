Создайте стилизованный список дел

Создайте приложение, функционально похожее на этот пример проекта. Старайтесь не копировать пример проекта, придайте ему свой собственный стиль.

В этой лабораторной работе вы попрактикуетесь в применении различных стилей к ссылкам при наведении курсора, фокусировке, клике и посещении.

Цель: Выполнить приведенные ниже пользовательские истории и добиться успешного прохождения всех тестов для завершения лабораторной работы.

Истории пользователей:

  1. У вас должен быть один неупорядоченный список с классом todo-list.
  2. Внутри неупорядоченного списка должно быть четыре элемента.
  3. В каждом элементе списка должно быть:
    • Элемент input типа checkbox с уникальным id.
    • Элемент label, у которого атрибут for установлен на id соответствующего элемента input.
    • Неупорядоченный список, с классом sub-item.
    • Элемент списка, содержащий элемент привязки. Элемент привязки должен иметь класс sub-item-link, допустимое значение href и значение target, которое открывает ссылку в новой вкладке.
  4. В элементах a не должно быть никаких текстовых украшений.
  5. Вы можете установить цвет текста для непосещенных ссылок на свой выбор.
  6. При переходе по ссылкам цвет должен меняться на другой цвет по вашему выбору.
  7. При наведении курсора на ссылки их цвет должен измениться на другой цвет по вашему выбору.
  8. Когда ссылки находятся в фокусе, вокруг них должна быть цветная рамка.
  9. При нажатии на ссылки цвет должен измениться на другой цвет по вашему выбору.

Примечание: Обязательно подключите таблицу стилей в свой HTML-код и примените CSS-стили.

Тесты

  1. У вас должен быть один неупорядоченный список с классом todo-list.
  2. Внутри неупорядоченного списка должно быть четыре элемента.
  3. Элемент li внутри ul с классом todo-list должен содержать элемент input типа checkbox.
  4. Элемент li внутри ul с классом todo-list должен содержать элемент label.
  5. Все элементы input должны иметь id.
  6. Все элементы label должны иметь атрибут for.
  7. Все элементы label должны содержать какой-либо текст.
  8. Атрибуты id и for в парах элементов input и label должны иметь соответствующие значения.
  9. После элементов label должен следовать неупорядоченный список с классом sub-item.
  10. Элемент li внутри ul с классом sub-item должен содержать элемент a с классом sub-item-link.
  11. Все элементы привязки должны иметь допустимый атрибут href.
  12. Все элементы привязки должны содержать текст.
  13. Каждый элемент a должен иметь атрибут target со значением _blank.
  14. В элементах a не должно быть никаких текстовых украшений.
  15. Непосещаемые ссылки следует задавать цветом текста по вашему выбору.
  16. При наведении курсора на ссылки они должны менять цвет.
  17. Цвет ссылок должен меняться при нажатии.
  18. При фокусировке ссылки должны иметь контур.
  19. При переходе по ссылкам их цвет должен измениться.
