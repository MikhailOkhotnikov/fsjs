Создайте открытку для поста в блоге

Создайте приложение, функционально похожее на этот пример проекта. Старайтесь не копировать пример проекта, придайте ему свой собственный стиль.

В этой лабораторной работе вы попрактикуетесь в оформлении фона и рамок, создав открытку для блога.

Цель: Выполнить приведенные ниже пользовательские истории и добиться успешного прохождения всех тестов для завершения лабораторной работы.

Пользовательские истории:

  1. Для размещения всех элементов карточек вам понадобится элемент div с классом blog-post-card.
  2. Внутри элемента .blog-post-card должен находиться элемент изображения с допустимым атрибутом alt и текстом, а также классом post-img. В качестве атрибута src для изображения можно использовать https://cdn.freecodecamp.org/curriculum/labs/cover-photo.jpg.
  3. Внутри элемента .blog-post-card должен находиться div с классом post-content, содержащий следующий код:
    • Элемент h2 с классом post-title и текстом для заголовка вашей записи в блоге.
    • Элемент p с классом post-excerpt и текстом для краткого изложения вашего поста в блоге.
    • Элемент a с классом read-more и текстом Читать далее.
  4. Цвет фона элемента body следует установить на значение, отличное от белого.
  5. К элементу .blog-post-card следует применить следующие стили:
    • Белый фон.
    • Закругленные углы.
    • Ширина на ваш выбор.
    • Выравнивание текста по вашему выбору.
  6. Элемент .post-img следует оформить таким образом, чтобы изображение заполняло всю ширину карточки и имело нижнюю границу.
  7. Элемент .post-content следует стилизовать таким образом, чтобы внутри карточки был внутренний отступ.
  8. Элементы .post-title и .post-excerpt должны иметь цвет текста, отличный от цвета по умолчанию, и внешние отступы со всех сторон.
  9. Элемент .read-more` должен быть оформлен как кнопка и содержать:
    • Цвет текста, отличный от цвета по умолчанию.
    • Цвет фона.
    • Внешние отступы со всех сторон.
    • Внутренние отступы со всех сторон.
    • Закругленные углы.
    • Свойство display, установленное в значение inline-block.
    • Эффект наведения курсора, изменяющий цвет фона.

Примечание: Обязательно подключите таблицу стилей в свой HTML-код и примените CSS-стили.

Тесты

  1. У вас должен быть div с классом blog-post-card.
  2. У вас должен быть элемент img с классом post-img. Убедитесь, что у вашего изображения есть атрибут alt и атрибут src с непустыми значениями.
  3. У вас должен быть div с классом post-content.
  4. У вас должен быть элемент h2 с классом post-title. Убедитесь, что в нем есть текст, соответствующий заголовку вашего поста в блоге.
  5. У вас должен быть элемент p с классом post-excerpt. Убедитесь, что он содержит текстовое содержимое, кратко описывающее содержание вашей записи в блоге.
  6. У вас должен быть элемент a с классом read-more.
  7. В элементе .read-more должен отображаться текст Читать далее.
  8. Элемент .blog-post-card должен иметь свойство border-radius со значением (не должно быть 0 или отрицательное значение).
  9. Элемент .blog-post-card должен иметь белый фон.
  10. У элемента body должен быть цвет фона, отличный от белого.
  11. Вам нужно выбрать элемент .blog-post-card и задать для него свойство width.
  12. Вам нужно выбрать элемент .blog-post-card и задать для него свойство text-align.
  13. Вам нужно выбрать элемент .post-content и задать для него свойство padding.
  14. Элемент .read-more должен иметь эффект наведения курсора.
  15. Вам нужно выбрать элемент .read-more и задать для него свойство color.
  16. Вам нужно выбрать элемент .read-more и задать для него свойство background-color.
  17. Вам нужно выбрать элемент .read-more и задать для него свойство margin.
  18. Вам нужно выбрать элемент .read-more и задать для него свойство display.
  19. Вам нужно выбрать элемент .read-more и задать для него свойство border-radius.
  20. Вам нужно выбрать элемент .read-more и задать для него свойство padding.
  21. Элемент .post-img должен занимать всю ширину карточки.
  22. У элемента .post-img должно быть значение border-bottom.
  23. Элементы .post-title и .post-excerpt должны иметь внешние отступы и нестандартные цвета текста.
