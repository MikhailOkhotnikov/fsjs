Шаг 9

Последним элементом, который нужно добавить в таблицу, будет элемент tfoot. Этот элемент будет использоваться для отображения средней оценки всех студентов в таблице.

Добавьте элемент tfoot к таблице.

Внутри элемента tfoot добавьте элемент tr.

Внутри элемента tr добавьте два элемента td.

Первый элемент td должен содержать текст Средняя оценка. Второй элемент td должен содержать текст 4,2.

Файл index.html

          </head>

              <caption>Оценки за итоговый экзамен по математическому анализу</caption>
              <thead>
                <tr>
                  <th>Фамилия</th>
                  <th>Имя</th>
                  <th>Оценка</th>
                </tr>
              </thead>
              <tbody>

        </html>
