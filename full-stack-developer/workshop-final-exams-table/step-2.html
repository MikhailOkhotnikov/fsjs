Шаг 2

Чтобы добавить подпись к table, можно использовать элемент caption.

Вот пример использования элемента caption:

Пример кода 
      
        <table>
          <caption>Футбольные результаты</caption>
        </table>

Внутри элемента table разместите элемент caption с текстом Оценки за итоговый экзамен по математическому анализу.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Таблица итоговых экзаменов по математике</title>
          </head>
          <body>
            <table>

            </table>
          </body>
        </html>
