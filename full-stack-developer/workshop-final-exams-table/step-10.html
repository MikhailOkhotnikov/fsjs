Шаг 10

Ваша таблица почти готова, но осталось добавить еще кое-что.

Было бы неплохо, если бы элемент td, используемый для отображения средней оценки, занимал две колонки, а не только одну.

Для этого можно использовать атрибут colspan для элемента td.

Атрибут colspan определяет количество столбцов, которые должна занимать ячейка.

Вот пример использования атрибута colspan для спортивной таблицы:

Пример кода 
      
        <tr>
          <td colspan="3">Общее количество баллов</td>
        </tr>

Внутри открывающего тега td добавьте атрибут colspan и установите для него значение 2.

И с этим изменением ваша таблица будет готова!

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">

            <meta charset="UTF-8" />

                <tr>
                  <th>Фамилия</th>
                  <th>Имя</th>
                  <th>Оценка</th>
                </tr>
              </thead>
              <tbody>
                 <tr>
                   <td>Иванов</td>
                   <td>Иван</td>
                   <td>5</td>
                 </tr>
                 <tr>
                   <td>Петров</td>

                   <td>4</td>
                 </tr>
                 <tr>
                   <td>Сидоров</td>
                   <td>Сидор</td>
                   <td>3</td>
                 </tr>
                 <tr>
                   <td>Михайлов</td>
                   <td>Михаил</td>
                   <td>4</td>
                 </tr>
                 <tr>
                   <td>Дмитриев</td>
                   <td>Дмитрий</td>
                   <td>5</td>
                 </tr>                 
              </tbody>
              <tfoot>
                <tr>

                  <td>Средняя оценка</td>
                  
                  <td>4,2</td>
                </tr>
              </tfoot>
            </table>
          </body>
        </html>
