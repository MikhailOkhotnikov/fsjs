Шаг 6

Для добавления данных о студентах в таблицу вам потребуется использовать элементы «строка таблицы» и «данные таблицы».

Элемент данных таблицы, td, используется для создания ячейки в таблице.

Вот пример использования элемента td для таблицы с именами спортсменов:

Пример кода 
      
        <tr>
          <td>1</td>
          <td>Александр Поветкин</td>
          <td>Россия</td>
        </tr>

Внутри элемента tbody вашей таблицы добавьте элемент строки таблицы (tr) с тремя элементами данных таблицы (td).

Первый элемент данных в таблице должен содержать фамилию Иванов.

Второй элемент данных в таблице должен содержать имя Иван.

Третий элемент данных в таблице должен содержать оценку 5.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Таблица итоговых экзаменов по математике</title>
          </head>
          <body>
            <table>
              <caption>Оценки за итоговый экзамен по математическому анализу</caption>
              <thead>
                <tr>
                  <th>Фамилия</th>
                  <th>Имя</th>
                  <th>Оценка</th>
                </tr>
              </thead>
              <tbody>

              </tbody>
            </table>
          </body>
        </html>
