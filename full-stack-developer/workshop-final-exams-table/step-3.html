Шаг 3

В первой части таблицы вам нужно будет сгруппировать содержимое заголовка, которое представляет собой названия столбцов с именем студента, фамилией и оценкой за итоговый экзамен.

Элемент заголовка таблицы, thead(table head), используется для группировки содержимого заголовка в таблице.

Вот пример использования элемента thead:

Пример кода 
      
        <table>
          <thead>
            <!-- Здесь размещается содержимое заголовка. -->
          </thead>
        </table>

Ниже элемента caption добавьте элемент thead.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Таблица итоговых экзаменов по математике</title>
          </head>
          <body>
            <table>
              <caption>Оценки за итоговый экзамен по математическому анализу</caption>
              
            </table>
          </body>
        </html>
