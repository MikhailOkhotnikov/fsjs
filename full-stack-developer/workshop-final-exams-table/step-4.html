Шаг 4

Элемент заголовка таблицы состоит из элемента строки таблицы, tr(table row), который содержит элементы ячеек заголовка таблицы, th(table header).

Вот пример использования элементов tr и th для спортивной таблицы:

Пример кода 
      
        <table>
          <caption>Результаты футбольных матчей</caption>
          <thead>
            <tr>
              <th>Команда</th>
              <th>Побед</th>
              <th>Поражений</th>
            </tr>
          </thead>
        </table>

Внутри элемента thead добавьте элемент tr.

Внутри элемента tr добавьте три элемента th.

Первый элемент th должен содержать текст Фамилия. Второй элемент th должен содержать текст Имя. Третий элемент th должен содержать текст Оценка.
