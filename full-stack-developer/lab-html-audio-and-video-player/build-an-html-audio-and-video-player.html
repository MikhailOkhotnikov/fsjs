Создайте HTML аудио и видео плеер

Создайте приложение, функционально похожее на этот пример проекта. Постарайтесь не копировать пример проекта, а придайте ему свой собственный стиль.

Цель: реализовать приведенные ниже пользовательские истории и пройти все тесты, необходимые для завершения лабораторной работы.

Истории пользователей:

  1. Для основного заголовка страницы необходим элемент h1.
  2. У вас должно быть два элемента section.
  3. Внутри первого элемента section должен быть элемент h2 для заголовка воспроизводимого видео.
  4. Под элементом h2 должен располагаться элемент video с атрибутами controls и width. Атрибут width должен иметь значение 640.
  5. Внутри элемента video должен быть элемент source с атрибутом src, указывающим на видеофайл, и атрибутом type.
    • Вы можете использовать: https://cdn.freecodecamp.org/curriculum/labs/what-is-the-map-method-and-how-does-it-work.mp4.
  6. Внутри второго элемента section должен быть элемент h2 для названия воспроизводимой песни.
  7. Под элементом h2 должен располагаться элемент audio с атрибутами controls и loop, а также атрибут src, указывающий на аудиофайл.
    • Вы можете использовать https://cdn.freecodecamp.org/curriculum/js-music-player/sailing-away.mp3.
    • Или https://cdn.freecodecamp.org/curriculum/js-music-player/we-are-going-to-make-it.mp3.

Тесты

  1. Для основного заголовка страницы необходим элемент h1.
  2. Ваш h1 не должен быть пустым.
  3. У вас должен быть только один элемент h1.
  4. У вас должно быть два элемента section.
  5. Внутри первого элемента section должен быть элемент h2 для заголовка воспроизводимого видео.
  6. Под элементом h2 должен располагаться элемент video.
  7. Элемент video должен иметь атрибут controls.
  8. Элементу video следует присвоить атрибут width, равный 640.
  9. Внутри элемента video должен быть элемент source.
  10. Элемент source должен иметь атрибут src, указывающий на видеофайл.
  11. Элемент source должен иметь атрибут type
  12. Атрибут type должен соответствовать типу расширения файла, указанному в атрибуте src.
  13. Внутри второго элемента section должен быть элемент h2 для названия воспроизводимой песни.
  14. Все ваши элементы h2 должны содержать текст.
  15. Под вторым элементом h2 должен располагаться элемент audio.
  16. Элемент audio должен иметь атрибут controls.
  17. Элемент audio должен иметь атрибут loop.
  18. Элемент audio должен иметь атрибут src, указывающий на аудиофайл.
