Отладка формы пожертвования

Местная благотворительная организация создала веб-сайт с формой для пожертвований, но в нем есть несколько проблем, которые необходимо исправить. Форма недоступна для людей с ограниченными возможностями и содержит ошибки синтаксиса HTML.

Ваша задача — исправить все ошибки, чтобы форма работала корректно и была доступна всем пользователям. Выполните пункты в приведенных ниже пользовательских историях и нажмите «Запустить тесты», чтобы проверить, исправили ли вы все ошибки.

Истории пользователей:

  1. Элементы ввода являются пустыми элементами и не должны иметь закрывающих тегов. Удалите все закрывающие теги /input из формы.
  2. Добавьте элементы label к каждому полю ввода формы, чтобы пользователи знали, для чего предназначено каждое поле. Текст метки должен соответствовать тексту, расположенному рядом с каждым полем ввода.
  3. В поле Адрес электронной почты следует указывать тип email, а не text.
  4. Каждый элемент label следует связать с соответствующим элементом input, используя атрибут for для label и соответствующий атрибут id для input.
  5. Добавьте атрибут required к текстовым полям ввода, полям ввода электронной почты и числовым полям (но не к флажку или кнопке отправки), чтобы гарантировать, что пользователи заполнят необходимую информацию.
Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
        <head>
          <meta charset="UTF-8">
          <title>Форма пожертвования</title>
        </head>
        <body>
          <h1>Форма пожертвования</h1>
          <form>
            
            ФИО:
            <input type="text" name="name"></input>

            Адрес электронной почты:
            <input type="text" name="email">

            Сумма пожертвования (₽):
            <input type="number" name="amount"></input>

            <input type="checkbox" name="newsletter"></input>
            Подписаться

            <input type="submit" value="Отправить"></input>
          </form>
        </body>
        </html>

Тесты

  1. В вашем коде не должно быть закрывающих тегов /input.
  2. В вашей форме должно быть ровно пять input.
  3. В вашей форме должно быть ровно четыре элемента label.
  4. Первая метка должна содержать текст Полное имя:.
  5. Первый input должен иметь атрибут required.
  6. На второй метке должен быть текст Адрес электронной почты:.
  7. Второй input должен иметь атрибут required.
  8. На третьей метке должен быть текст Сумма пожертвования (₽):.
  9. Третий элемент input должен иметь атрибут required.
  10. Четвертая метка должна содержать текст Подписаться.
  11. Каждый элемент input следует связать с элементом label, используя атрибуты for и id.
  12. В поле Адрес электронной почты следует ввести тип email.
  13. В input с типом checkbox не должно быть атрибута required.
  14. Кнопка отправки не должна иметь атрибута required.
