Создайте центр событий

Создайте приложение, функционально похожее на этот проект-пример. Постарайтесь не копировать проект-пример, а придайте ему свой собственный стиль.

В этой лабораторной работе вы будете использовать семантические элементы HTML для создания структуры веб-страницы. Вы добавите контент и изображения, чтобы она выглядела как настоящий центр событий.

Цель: реализовать приведенные ниже пользовательские истории и пройти все тесты, необходимые для завершения лабораторной работы.

Истории пользователей:

  1. У вас должен быть элемент header.
  2. Внутри элемента header должен быть элемент h1, содержащий текст Центр событий, и элемент nav.
  3. Внутри элемента nav должен быть неупорядоченный список из двух элементов, содержащих ссылки на разные разделы страницы. Первый элемент должен содержать текст Предстоящие события, а второй — текст Прошедшие события.
  4. Каждая ссылка должна быть представлена ​​элементом a с атрибутом href, который ссылается на соответствующий раздел страницы, #upcoming-events и #past-events соответственно.
  5. У вас должен быть элемент main, содержащий различные разделы страницы.
  6. Внутри элемента main должно быть два элемента section.
  7. Первый элемент section должен иметь атрибут id со значением upcoming-events.
  8. Внутри раздела #upcoming-events у вас должно быть:
    • Элемент h2 с текстом Предстоящие события.
    • Два элемента article. Каждая статья должна представлять событие и должна содержать:
      • Элемент h3 для названия мероприятия.
      • Элемент p для описания события. При желании вы можете добавить дату внизу.
  9. Второй элемент section должен иметь атрибут id со значением past-events.
  10. Внутри раздела #past-events у вас должно быть:
    • Элемент h2 с текстом Прошедшие события.
    • Два элемента article. Каждый элемент article должен представлять прошедшее событие и должен иметь:
      • Элемент h3 для названия мероприятия,
      • Элемент p для описания события. При желании вы можете добавить дату внизу.
      • Элемент изображения с атрибутом src, указывающим на файл изображения, и атрибутом alt с описанием изображения.

Примечание: Вы можете использовать любой текст для описания и дат мероприятий. При желании вы можете использовать следующие URL-адреса изображений:

  • https://cdn.freecodecamp.org/curriculum/labs/past-event1.jpg
  • https://cdn.freecodecamp.org/curriculum/labs/past-event2.jpg

Тесты

  1. У вас должен быть элемент header.
  2. Элемент header должен располагаться после открывающего тега body.
  3. Внутри элемента header должен быть элемент h1, содержащий текст Центр событий.
  4. Внутри элемента header, после элемента h1, должен располагаться элемент nav.
  5. Ваш элемент nav должен содержать неупорядоченный список из двух элементов.
  6. Первым элементом неупорядоченного списка должна быть ссылка.
  7. Вторым элементом неупорядоченного списка должна быть ссылка.
  8. Текст первого элемента в неупорядоченном списке должен быть Предстоящие события.
  9. Первый элемент в неупорядоченном списке должен иметь атрибут href, равный #upcoming-events
  10. Текст второго элемента в неупорядоченном списке должен быть Прошедшие события.
  11. Второй элемент в неупорядоченном списке должен иметь атрибут href, равный #past-events
  12. После закрывающего тега /header должен располагаться элемент main.
  13. Внутри элемента main должно быть два элемента section.
  14. Ваш первый элемент section должен иметь атрибут id со значением upcoming-events.
  15. Ваш второй элемент section должен иметь атрибут id со значением past-events.
  16. Внутри #upcoming-events раздела, у вас должен быть элемент h2 с текстом Предстоящие события
  17. Внутри #upcoming-events раздела, у вас должно быть два элемента article под элементом h2.
  18. Оба элемента статьи внутри раздела #upcoming-events должны иметь элемент h3 для заголовка мероприятия.
  19. Оба элемента статьи внутри раздела #upcoming-events должны иметь элемент абзаца для описания события.
  20. Внутри #past-events раздела, у вас должен быть элемент h2 с текстом Прошедшие события
  21. Внутри #past-events раздела, у вас должно быть два элемента article под элементом h2.
  22. Оба элемента статьи внутри раздела #past-events должны иметь элемент h3 для заголовка мероприятия.
  23. Оба элемента статьи внутри раздела #past-events должны иметь элемент абзаца для описания события.
  24. Оба элемента статьи внутри раздела #past-events должны иметь элемент изображения.
  25. Оба элемента изображения внутри раздела #past-events должны иметь атрибут src, указывающий на файл изображения.
  26. Оба элемента изображения внутри раздела #past-events должны иметь атрибут alt с описанием изображения.
  27. Каждый элемент h3 должен иметь заголовок события.
  28. Каждый элемент p должен иметь описание события.
