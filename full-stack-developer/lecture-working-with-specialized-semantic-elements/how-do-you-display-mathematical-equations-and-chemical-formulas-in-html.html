Как отображать математические уравнения и химические формулы в HTML?

Элемент "надстрочный индекс"(sup, superscript) используется для отображения текста в виде надстрочного индекса. Надстрочный индекс — это символ или буква, печатаемая над обычной строкой текста.

Вот пример использования надстрочного элемента для иллюстрации показателей степеней: 

    
      <p>2<sup>2</sup> (2 в квадрате) равно 4.</p>

В этом примере число 2 заключено в теги sup для обозначения надстрочного индекса внутри абзаца. В окне предварительного просмотра вы увидите, что второе число 2 меньше и немного выше первого числа 2.

Элемент «надстрочный индекс» обычно используется для обозначения показателей степеней, надстрочных букв и порядковых числительных. Вот пример использования элемента «надстрочный индекс» для обозначения надстрочных букв: 

    
      <p>
        Монсеньор часто пишется как <strong>М<sup>gr</sup></strong>.
      </p>

Надстрочные буквы обозначают буквы, написанные надстрочным шрифтом, обычно для обозначения сокращений. В данном примере буквы g и r заключены в надстрочные теги для наглядности сокращения.

Важно отметить, что элемент надстрочного индекса следует использовать только в типографских целях. Если вы хотите оформить текст с приподнятой базовой линией, вместо элемента надстрочного индекса следует использовать CSS.

Для представления химических уравнений в HTML используется элемент "нижний индекс"subscript. Этот элемент использует нижний индекс, который отображает заниженную базовую линию с использованием более мелкого текста.

Вот пример использования нижнего индексного элемента для отображения химического представления углекислого газа. 

    
      <p>CO<sub>2</sub></p>

Число два заключено в тег sub, чтобы проиллюстрировать, что символ должен быть нижним индексом.

К распространенным вариантам использования элемента нижнего индекса относятся химические формулы, сноски и переменные нижние индексы.
