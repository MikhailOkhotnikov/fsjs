Для чего используются элементы U, S и Ruby и как они работают?

Элемент неартикулированной(unarticulated) аннотации, или сокращенно элемент u, используется для представления встроенного текста, к которому применена нетекстовая аннотация.

Вот пример использования элемента u для выделения различных орфографических ошибок: 

    
      <p>
        Вы можете использовать элемент неартикулированной аннотации, чтобы выделить
        <u>несооответствие</u> <u>нописание</u> <u>проблеммы</u>.
      </p>

В этом примере слова несоответствие, написание и проблемы написаны с ошибками. Стиль по умолчанию для элемента u — чёрное подчёркивание под текстом.

В HTML4 элемент u использовался для стилизации. Но в HTML5 элемент u следует использовать только для обозначения того, что к тексту применена нетекстовая аннотация.

Если вы хотите подчеркнуть текст, вам следует использовать CSS вместо HTML.

Зачёркнутый(strikethrough) элемент, или сокращённо элемент s, следует использовать для обозначения того, что текст больше не является точным или релевантным. Вот пример использования элемента s для обозначения отмены действия: 

    
      <p><s>Завтрашний поход состоится в полдень.</s></p>

      <p>Из-за непредвиденных погодных условий поход был отменен.</p>

В этом примере первое предложение зачеркнуто, поскольку поход был отменен из-за погодных условий.

Элемент s никогда не следует использовать только для отображения изменений в документе. В этом случае более подходящими элементами будут удалённый текстовый элемент и вставленный текстовый элемент.

Элемент ruby ​​представляет собой небольшой текст, отображаемый над или под основным текстом. Обычно он используется для отображения произношения восточноазиатских иероглифов. Вот пример элемента ruby ​​со страницы веб-документации MDN: 

    
      <ruby> 明日 <rp>(</rp><rt>Ashita</rt><rp>)</rp> </ruby>

Элемент rp(ruby parenthesis), или элемент скобок, используется в качестве запасного варианта для браузеров, не поддерживающих отображение аннотаций ruby.

Элемент rt(ruby text), или элемент текста, используется для указания текста аннотации. Этот текст обычно используется для обозначения произношения или деталей перевода в восточноазиатской типографике.

Хотя элемент ruby ​​можно использовать и для других типов аннотаций, наиболее распространенным вариантом его использования является восточноазиатская типографика.
