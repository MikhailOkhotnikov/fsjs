Как представить компьютерный код в HTML?

Элемент "встроенный код"(inline code) используется для представления коротких фрагментов кода внутри текста. Обычно он используется в технических статьях и страницах документации.

Вот пример использования элемента code для отображения фрагмента кода CSS: 

    
      <p>
        Чтобы установить синий цвет текста в CSS, используйте следующий код:
        <code>color: blue;</code>
      </p>

В этом примере свойство CSS color используется для установки синего(blue) цвета текста. Этот фрагмент кода, заключенный в теги code, сообщает браузеру, что текст представляет собой встроенный код.

Браузер применит стили по умолчанию к содержимому внутри элемента code. По умолчанию используется моноширинный шрифт.

Элемент code предназначен для представления одной строки кода. Если вы хотите представить несколько строк кода, вам необходимо поместить элемент code внутри предварительно отформатированного(preformatted, pre) текстового элемента.

Элемент pre используется для представления предварительно отформатированного текста. Вот пример использования элемента pre для отображения CSS-декларации: 

    
      <pre>
        <code>
          body {
            color: red;
          }
        </code>
      </pre>

При использовании элемента pre необходимо помнить о пробелах, поскольку он будет отображаться точно так же, как написано внутри HTML-документа.

В браузере вы увидите отступ в несколько пробелов вправо. Если изменить отступ в примере кода, то на экране вы увидите разницу.

Когда дело доходит до включения примеров кода в HTML-документ, вам следует использовать элемент code для коротких встроенных примеров.

Если вам необходимо отобразить более длинные фрагменты кода, то вам нужно будет использовать элементы pre и code.
