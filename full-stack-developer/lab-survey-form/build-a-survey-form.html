Создайте форму опроса

Создайте приложение, функционально похожее на этот пример проекта. Старайтесь не копировать пример проекта, придайте ему свой собственный стиль.

Цель: Выполнить приведенные ниже пользовательские истории и добиться успешного прохождения всех тестов для завершения лабораторной работы.

Пользовательские истории:

  1. Заголовок страницы должен быть размещен в элементе h1 с идентификатором title.
  2. Краткое пояснение должно быть размещено в элементе p с идентификатором description.
  3. Вам потребуется элемент form с идентификатором survey-form.
  4. Внутри элемента формы должно быть обязательное поле input для вашего имени, имеющее идентификатор name и type равный text.
  5. Внутри элемента формы должно быть обязательное поле input для вашего адреса электронной почты с идентификатором email.
  6. Если вы введете адрес электронной почты с неправильным форматированием, вы увидите ошибку проверки HTML5.
  7. Внутри элемента формы должно быть поле input для ввода числа, имеющее идентификатор number.
  8. Ввод чисел не должен принимать нечисловые значения, либо путем запрета их ввода, либо путем отображения ошибки валидации HTML5 (в зависимости от вашего браузера).
  9. Если вы введете числа, выходящие за пределы диапазона, заданного атрибутами min и max, вы увидите ошибку валидации HTML5.
  10. Для полей ввода имени, электронной почты и номера телефона необходимо добавить в форму соответствующие элементы label, описывающие назначение каждого поля и имеющие следующие идентификаторы: name-label, email-label и number-label.
  11. Для полей ввода имени, электронной почты и номера телефона следует добавить placeholder с описанием или инструкциями для каждого поля.
  12. Внутри элемента формы должен находиться элемент select с идентификатором dropdown и как минимум двумя вариантами выбора.
  13. Внутри элемента формы можно выбрать вариант из группы, состоящей как минимум из двух переключателей, сгруппированных по атрибуту name.
  14. Внутри элемента формы можно выбрать несколько полей из ряда флажков, каждый из которых должен иметь атрибут value.
  15. Внутри элемента формы должно быть поле textarea для дополнительных комментариев.
  16. Внутри элемента формы должна быть кнопка с идентификатором submit для отправки всех введенных данных.

Тесты

  1. Вам нужен элемент h1 с id равным title.
  2. Ваш #title не должен быть пустым.
  3. У вас должен быть элемент p с id равным description.
  4. Ваше #description не должно быть пустым.
  5. Вам потребуется элемент form с id равным survey-form.
  6. Вам потребуется элемент input с id равным name.
  7. Ваш #name должен содержать type равный text.
  8. Ваше #name должен потребовать ввода.
  9. Ваше #name должно быть потомком #survey-form.
  10. Вам потребуется элемент input с id равным email.
  11. В вашем #email должен быть указан type равный email.
  12. Ваш #email должен потребовать ввода.
  13. Ваш #email должен быть потомком #survey-form.
  14. Вам потребуется элемент input с id равным number.
  15. Ваш #number должен быть потомком #survey-form.
  16. Ваш #number должен иметь type равный number.
  17. Ваш #number должен иметь атрибут min с числовым значением.
  18. Ваш #number должен иметь атрибут max с числовым значением.
  19. У вас должен быть элемент label с id равным name-label.
  20. У вас должен быть элемент label с id равным email-label.
  21. У вас должен быть элемент label с id равным number-label.
  22. Ваш #name-label должен содержать текст, описывающий вводимые данные.
  23. Ваш #email-label должен содержать текст, описывающий вводимые данные.
  24. Ваш #number-label должен содержать текст, описывающий вводимые данные.
  25. Ваш #name-label должен быть потомком #survey-form.
  26. Ваш #email-label должен быть потомком #survey-form.
  27. Ваш #number-label должен быть потомком #survey-form.
  28. Ваш #name должен содержать атрибут placeholder и значение.
  29. Ваш #email должен содержать атрибут placeholder и значение.
  30. Ваш #number должен содержать атрибут placeholder и значение.
  31. Вам потребуется поле select с id равным dropdown.
  32. В вашем #dropdown должно быть как минимум два выбираемых (не отключенных) элемента option.
  33. Ваш #dropdown должен быть потомком #survey-form.
  34. У вас должно быть как минимум два элемента input с type равным radio(радиокнопки).
  35. У вас должно быть как минимум две радиокнопки, которые являются потомками #survey-form.
  36. Все ваши радиокнопки должны иметь атрибут value и значение.
  37. Все ваши радиокнопки должны иметь атрибут name и значение.
  38. В каждой группе радиокнопок должно быть как минимум две радиокнопки.
  39. У вас должно быть как минимум два элемента input с type равным checkbox(флажки), которые являются потомками #survey-form.
  40. Все ваши флажки внутри #survey-form должны иметь атрибут value и значение.
  41. У вас должен быть хотя бы один элемент textarea, который является потомком #survey-form.
  42. Вам потребуется элемент input или button с id равным submit.
  43. В вашем #submit должен быть указан атрибут type равный submit.
  44. Ваш #submit должен быть потомком #survey-form.
