Создайте дизайн визитной карточки

Создайте приложение, функционально похожее на этот пример проекта. Старайтесь не копировать пример проекта, придайте ему свой собственный стиль.

Цель: Выполнить приведенные ниже пользовательские истории и добиться успешного прохождения всех тестов для завершения лабораторной работы.

Истории пользователей:

  1. Вам потребуется div с атрибутом class, имеющим значение business-card, который будет содержать все остальные элементы.
  2. Внутри элемента .business-card должен находиться элемент img с атрибутом class, значение которого равно profile-image. При желании вы можете установить в качестве источника изображения https://cdn.freecodecamp.org/curriculum/labs/flower.jpg. Также должен быть атрибут alt с содержательным описанием.
  3. Внутри элемента .business-card, после элемента img, должны находиться три элемента p с атрибутом class, имеющим, соответственно, значения full-name, designation и company.
  4. Первый элемент p должен содержать ваше имя.
  5. Второй элемент p должен содержать вашу должность.
  6. Третий элемент p должен содержать название вашей компании.
  7. Под третьим элементом p должен располагаться элемент hr.
  8. После элемента hr должны быть два элемента p. В первом элементе p должен быть текст, содержащий адрес электронной почты. Во втором элементе p должен быть текст, содержащий номер телефона.
  9. После двух элементов p должен быть элемент a с классом portfolio-link, текстом Портфолио, который должен вести на действительный URL-адрес.
  10. После элемента a, содержащего ссылку на портфолио, должен быть элемент hr.
  11. Вам потребуется еще один элемент div с атрибутом class, имеющим значение social-media. Внутри этого элемента должен находиться элемент h2 с текстом Свяжитесь со мной.
  12. Внутри элементов .social-media должны находиться три элемента a. Элементы a должны содержать текст Twitter, LinkedIn и GitHub соответственно, со ссылками на действительные веб-сайты.
  13. Для подключения таблицы стилей styles.css используйте тег link в разделе head HTML-файла.
  14. Фон страницы должен быть розово-коричневого цвета(rosybrown). В качестве основного шрифта следует использовать Arial, в качестве запасного — sans-serif.
  15. Селектор .business-card должен иметь свойства, позволяющие задать ширину элемента 300px и цвет фона по вашему выбору. Также необходимо установить внутренние отступы 20px со всех сторон и верхний внешний отступ 100px. Текст должен быть выровнен по центру, а размер шрифта — 16px. Левый и правый внешние отступы должны быть установлены в значение auto.
  16. Селектор .profile-image должен иметь свойство max-width со значением 100%.
  17. Все элементы p должны иметь верхний и нижний внешние отступы в 5px.
  18. Все ссылки на странице не должны быть подчеркнуты.

Тесты

  1. У вас должен быть один основной div с атрибутом class, значение которого равно business-card.
  2. Внутри элемента .business-card должен находиться элемент img с атрибутом class, значение которого равно profile-image.
  3. В качестве источника изображения следует указать допустимое изображение.
  4. Атрибуту alt изображения следует присвоить содержательное описание.
  5. Внутри элемента .business-card, после элемента img, должен находиться элемент p с атрибутом class, значение которого равно full-name.
  6. После элемента .full-name должен следовать элемент p с атрибутом class, значение которого равно designation.
  7. После элемента .designation должен следовать элемент p с атрибутом class, значение которого равно company.
  8. Первый элемент p, тот, который имеет класс full-name, должен содержать ваше имя.
  9. Второй элемент p, тот, у которого атрибут class имеет значение designation, должен содержать вашу должность.
  10. Третий элемент p, тот, который имеет атрибут class со значением company, должен содержать название вашей компании.
  11. После третьего элемента p, имеющего атрибут class со значением company, должен следовать элемент hr.
  12. После первого элемента hr должен быть элемент p с текстом, содержащим адрес электронной почты.
  13. После элемента p с адресом электронной почты должен быть еще один элемент p с текстом, содержащим номер телефона.
  14. После элемента p, содержащего номер телефона, должен следовать элемент a с классом portfolio-link.
  15. Элемент .portfolio-link должен содержать текст Портфолио.
  16. Элемент .portfolio-link должен иметь действительный атрибут href.
  17. После элемента, содержащего ваше портфолио, должен быть еще один элемент hr в качестве разделителя.
  18. После второго элемента hr должен следовать элемент div с атрибутом class, значение которого равно social-media.
  19. Внутри элемента .social-media должен находиться элемент h2 с текстом Свяжитесь со мной.
  20. Внутри элемента .social-media должно быть три элемента a.
  21. Первый элемент a должен содержать текст Twitter.
  22. Ссылка href первого элемента a должна быть действительной.
  23. Второй элемент a должен содержать текст LinkedIn.
  24. Ссылка href второго элемента a должна быть действительной.
  25. В третьем элементе a должен быть текст GitHub.
  26. Ссылка href третьего элемента a должна быть действительной.
  27. Ваш элемент link должен находиться внутри элемента head.
  28. Элемент link должен иметь атрибут rel со значением stylesheet
  29. Элемент link должен иметь атрибут href со значением styles.css.
  30. В селекторе body следует установить цвет фона(background-color) страницы на rosybrown.
  31. В элементе body следует установить шрифт(font-family) страницы Arial, а в качестве запасного варианта использовать шрифт sans-serif.
  32. В селекторе .business-card должно быть свойство width со значением 300px
  33. В селекторе .business-card должно быть свойство background-color с допустимым значением цвета.
  34. Селектор .business-card должен иметь свойство padding со значением 20px.
  35. У вашего селектора .business-card должен быть margin-top со значением 100px.
  36. Текст внутри элемента .business-card должен быть выровнен по центру(text-align: center).
  37. Размер шрифта(font-size) текста внутри элемента .business-card должен составлять 16px.
  38. Свойства margin-left и margin-right элемента .business-card должны быть установлены в значение auto.
  39. В селекторе .profile-image должно быть свойство max-width со значением 100%.
  40. Элементы p должны иметь margin-top и margin-bottom по 5px.
  41. Ссылки на странице не должны быть подчеркнуты(text-decoration: none).
