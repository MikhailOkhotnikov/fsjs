Наконец, JavaScript делает вашу веб-страницу интерактивной — он позволяет вам сообщать странице, что делать, когда кто-то нажимает кнопку, отправляет форму или выполняет множество других действий.

На последнем этапе семинара добавьте еще один элемент h3 и p на страницу, описывающую JavaScript.

Сначала добавьте элемент h3 с текстом: Введение в JavaScript

Затем под элементом h3 добавьте элемент p со следующим текстом: JavaScript добавляет интерактивности веб-странице.
