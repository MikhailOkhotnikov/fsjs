Каковы преимущества качественного ссылочного текста с точки зрения доступности, и какие примеры качественного ссылочного текста?

Давайте рассмотрим преимущества написания качественных ссылочных текстов в контексте доступности, а также несколько примеров хороших ссылочных текстов.

Первое очевидное преимущество хорошего текста ссылки заключается в том, что он облегчает пользователям быстрый поиск информации. Описательные ссылки помогают пользователям понять, куда они направляются и к чему получат доступ. Это гарантирует, что пользователь не будет чувствовать себя потерянным, и улучшает общее впечатление от использования сайта.

Для тех, кто использует программы чтения с экрана, четкий и информативный текст ссылки просто необходим. Программы чтения с экрана зачитывают текст ссылки вслух, поэтому текст типа «Прочитайте наше руководство по доступности» гораздо лучше, чем «Нажмите здесь».

Четкий и информативный текст ссылок полезен не только для людей с нарушениями зрения. Информативные ссылки также помогают людям с когнитивными расстройствами, предоставляя понятный контекст.

Вот несколько рекомендаций, которые следует учитывать при написании текстов для ссылок:

  • Убедитесь, что ссылки визуально выделяются с помощью подчеркивания и других визуальных подсказок, чтобы пользователи могли легко их идентифицировать и перемещаться по ним.
  • Избегайте общих текстов ссылок, таких как «здесь», «нажмите здесь» и «подробнее», поскольку они не содержат полезной информации.
  • Старайтесь создавать краткие и содержательные тексты для ссылок, в идеале от 2 до 5 слов, которые передают цель ссылки.
  • Избегайте профессионального жаргона и сокращений, которые могут быть непонятны пользователям.
  • Сосредоточьтесь на результате, а не на действии. Например, используйте «результаты поведения пользователя» вместо «нажмите здесь, чтобы узнать больше».
  • Не используйте один и тот же текст для ссылок на разные ресурсы.
  • Размещайте ссылки таким образом, чтобы они органично вписывались в окружающий текст. Например, вместо «Нажмите здесь для получения дополнительной информации» используйте фразу «для получения более подробной информации посетите нашу страницу мероприятий».

Вот несколько примеров удачных ссылочных текстов для конкретных случаев использования, в сравнении с менее полезными. Допустим, вы хотите сослаться на страницу с подробной информацией о мероприятии, например, такую: 

    
      <a href="webinar-details-link">Подробности</a>

Сообщение Подробности расплывчатое и не содержит конкретной информации о том, что пользователь найдет, перейдя по ссылке. Без дополнительного контекста пользователи могут не понимать, ведет ли ссылка к подробностям о вебинаре, продукте, политике или чем-то еще.

А вот пример удачного текста для ссылки: 

    
      <a href="webinar-details-link">
        Узнайте подробности о нашем предстоящем вебинаре.
      </a>

Текст ссылки предоставляет пользователям контекст о содержании, которое они найдут, облегчая принятие решения о том, стоит ли по ней переходить. Он уменьшает двусмысленность, указывая на то, что ссылка относится к вебинару.

Вот ещё один пример ссылки на пост в блоге: 

    
      <a href="/blog-post-link">Читать далее</a>

Текст ссылки Читать далее не идеален с точки зрения доступности, поскольку ему не хватает контекста.

Вот более удачный пример текста ссылки: 

    
      <a href="/blog-post-link">
        Прочитайте нашу последнюю статью в блоге о доступности веб-сайтов.
      </a>

Обновленный текст ссылки дает пользователям более четкое представление о том, чего ожидать и почему они могут захотеть перейти по ссылке, что особенно важно для тех, кто использует программы чтения с экрана.

Рассмотрим еще один пример. Допустим, вы хотите предоставить дополнительную информацию по определенной теме: 

    
      <a href="/link-to-topic">Дополнительная информация</a>

Фраза Дополнительная информация также расплывчата и может создавать проблемы с точки зрения доступности. Она не содержит конкретных сведений о пункте назначения ссылки, что может сбивать с толку пользователей, использующих программы чтения с экрана, или тех, кто перемещается по сайту нелинейным способом.

Вот более удачный пример текста для ссылки: 

    
      <a href="/link-to-topic">
        Узнайте больше о наших усилиях по обеспечению доступности.
      </a>

Обновленный текст ссылки стал лучше, потому что он четко описывает, что пользователь увидит, перейдя по ссылке. Пользователи сразу поймут, что ссылка ведет к информации о ваших мерах по обеспечению доступности.
