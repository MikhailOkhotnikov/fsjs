Когда нужен атрибут alt и какие примеры хорошего замещающего текста можно привести?

Альтернативный текст(alternative text), часто сокращаемый как alt text, — это краткое текстовое описание изображения. Он предоставляет важную информацию об изображении для пользователей, которые не могут его видеть, например, для людей, использующих программы чтения с экрана и другие вспомогательные технологии.

Это крайне важно для обеспечения доступности веб-сайтов для людей с нарушениями зрения.

Альтернативный текст также используется поисковыми системами для распознавания изображений. Некоторые браузеры могут отображать его и в случае некорректной загрузки изображения. Это может произойти, например, если файл изображения отсутствует или у пользователя возникли проблемы с подключением к интернету. Таким образом, он полезен для самых разных целей.

Человек с нарушением зрения не сможет понять, что изображено на картинке, если это не описано в альтернативном тексте (alt text). Примером неудачного альтернативного текста для изображения щенка может быть «Милый щенок».

Этот текст недостаточно конкретен, чтобы передать важные детали изображения, например, как выглядит щенок? Где находится щенок? Есть ли вокруг него какие-либо важные предметы?

Давайте улучшим этот текст. В качестве хорошего альтернативного варианта можно привести следующий текст: «Черно-белый щенок с оранжевым ошейником лежит на животе в песке, глядя в сторону. Рядом с его передними лапами лежит ярко-оранжевый мячик».

Здесь вы можете увидеть это в HTML с элементом изображения, атрибутом alt и более подробным описанием: 

    
      <img 
        src="puppy.png" 
        alt="Черно-белый щенок с оранжевым ошейником лежит на животе в песке, глядя в сторону. Рядом с его передними лапами лежит ярко-оранжевый мячик" 
      />

Важно отметить, что нет единственно правильного способа написать альтернативный текст для изображения. Содержание описания будет зависеть от контекста использования изображения. Например, если фотография щенка размещена на сайте о породах собак, вы можете указать более подробное описание внешности щенка и, возможно, проигнорировать тот факт, что щенок играет на пляже с оранжевым мячом. В конечном итоге, альтернативный текст для изображения должен отражать основную цель его размещения на странице, а предоставленная информация должна дать людям, не видящим изображение, необходимые сведения для понимания этой цели.

Вот ещё один пример прекрасного тропического курорта. Давайте опишем его.

Примером неудачного альтернативного текста для этого изображения может быть слово "Курорт".

Это слишком короткое описание, и оно не содержит достаточно информации об изображении. Чтобы улучшить это описание, вы могли бы расширить его, включив в него наиболее важные элементы изображения:

«Тропический курорт с бассейном, окруженным пальмами и бунгало».

Атрибут alt в HTML можно использовать следующим образом: 

    
      <img src="resort.png" alt="Тропический курорт с бассейном, окруженным пальмами и бунгало" />

Теперь, когда вы знаете, чем отличается хороший альтернативный текст от плохого, давайте рассмотрим некоторые из лучших практик.

  • Альтернативный текст (alt) должен быть коротким. Он должен быть достаточно подробным, чтобы изображение было понятно, но не настолько длинным, чтобы вызывать путаницу.
  • Не стоит пытаться описать каждую мелочь. Сосредоточьтесь на самых важных аспектах изображения.
  • Как правило, начинать со слов «изображение» или «картинка» необязательно. Можно сразу начать описание.
  • Кроме того, если вокруг изображения есть похожий текст, повторять его не нужно.
  • Для единообразия обычно рекомендуется заканчивать альтернативный текст точкой.
  • Если изображение является ссылкой на другую страницу, то вместо описания самого изображения, в alt следует указать, что произойдет, если пользователь нажмет на него.

Например, если на вашем сайте есть иконка стрелки вправо, которая переводит пользователя на следующую страницу, вместо того, чтобы писать alt-текст только "стрелка вправо", как в этом примере, где вы можете увидеть атрибут alt с таким описанием: 

    
      <a href="about.html">
        <img src="arrow-right.png" alt="стрелка вправо." />
      </a>

Вместо этого следует написать что-то вроде этого, где альтернативный текст описывает, что произойдет, если пользователь нажмет на изображение. Он перейдет на следующую страницу. 

    
      <a href="about.html">
        <img src="arrow-right.png" alt="Перейти на следующую страницу." />
      </a>

Альтернативный текст (alt) должен быть только у изображений, передающих важную информацию. Если изображение используется только в декоративных целях, у него должен быть пустой (null) альтернативный текст, чтобы его могли игнорировать программы чтения с экрана и другие вспомогательные технологии.

Вот пример пустого атрибута alt: 

    
       <img src="decorative_image.jpg" alt="" />

Каждое изображение на вашем сайте должно иметь атрибут alt, даже если он пустой. Если вы полностью опустите атрибут alt, некоторые программы чтения с экрана будут читать вместо него имя файла, что может отвлекать людей, использующих вспомогательные технологии, поэтому это не рекомендуется.

Наконец, перед запуском вашего сайта следует тщательно проверить, могут ли программы чтения с экрана правильно считывать альтернативный текст.

Создание эффективного альтернативного текста (alt) имеет важное значение для обеспечения доступности веб-контента. Как веб-разработчик, предоставляя четкие описания изображений, вы можете обеспечить инклюзивный онлайн-опыт для всех пользователей вашего сайта.
