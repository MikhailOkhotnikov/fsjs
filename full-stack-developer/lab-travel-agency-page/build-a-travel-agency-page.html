Создайте страницу туристического агентства

Создайте приложение, функционально похожее на этот пример проекта. Постарайтесь не копировать пример проекта, а придайте ему свой собственный стиль.

Цель: реализовать приведенные ниже пользовательские истории и пройти все тесты, необходимые для завершения лабораторной работы.

Истории пользователей:

  1. У вас должно быть объявление DOCTYPE.
  2. У вас должен быть элемент html со значением lang, равным ru.
  3. У вас должен быть элемент head, содержащий элемент meta с charset равным utf-8 и title с текстом Страница туристического агентства.
  4. В элементе head должен быть meta-тег, содержащий краткое описание вашего веб-сайта для SEO.
  5. Для представления направлений ваших путешествий вам необходим элемент h1.
  6. Под элементом h1 должен быть абзац(p), описывающий возможности путешествий.
  7. У вас должен быть элемент h2 с текстом Пакеты.
  8. У вас должен быть элемент p, кратко описывающий различные пакеты.
  9. Вам нужен неупорядоченный список с двумя пунктами. Текст каждого пункта должен быть Групповые поездки и Частные туры соответственно. Текст каждого пункта должен быть заключен в элемент привязки(anchor).
  10. У вас должен быть элемент h2 с текстом Лучшие маршруты.
  11. У вас должно быть не менее трех элементов figure, каждый из которых содержит элемент привязки(anchor) и элемент figcaption.
  12. Три элемента привязки должны содержать элемент img с соответствующим атрибутом alt и атрибутом src, для которого в качестве содержимого должно быть установлено допустимое изображение. При желании вы можете использовать https://cdn.freecodecamp.org/curriculum/labs/colosseo.jpg, https://cdn.freecodecamp.org/curriculum/labs/alps.jpg и https://cdn.freecodecamp.org/curriculum/labs/sea.jpg.
  13. Все ваши пять элементов привязки должны иметь атрибут href со значением https://www.freecodecamp.org/learn и атрибут target со значением _blank.

Тесты

  1. Страница вашего туристического агентства должна иметь декларацию !DOCTYPE html.
  2. У вас должен быть элемент html со значением lang, равным ru.
  3. Внутри элемента html должен быть элемент head.
  4. Внутри элемента head должно быть два элемента meta.
  5. Один meta-элемент должен иметь атрибут name со значением description и непустой атрибут content.
  6. Для одного meta-элемента атрибут charset должен быть установлен на UTF-8.
  7. Внутри элемента head должен находиться элемент title.
  8. В title должно быть указано название вашего туристического агентства.
  9. Внутри элемента html должен быть элемент body.
  10. Для представления направлений ваших путешествий вам необходим элемент h1.
  11. У вас должен быть только один элемент h1.
  12. Под элементом h1 должен располагаться элемент p.
  13. Ваш первый элемент p должен представить возможности путешествий.
  14. Ваш первый элемент h2 должен иметь текст Пакеты.
  15. Элемент p должен располагаться под первым элементом h2.
  16. Второй элемент p должен кратко представлять различные пакеты.
  17. У вас должен быть элемент неупорядоченного списка под вторым элементом p.
  18. В вашем неупорядоченном списке должно быть два элемента.
  19. Оба элемента списка должны содержать элемент привязки.
  20. Элемент привязки первого пункта списка должен содержать текст Групповые поездки.
  21. Элемент привязки второго пункта списка должен содержать текст Частные туры.
  22. После неупорядоченного списка должен быть элемент h2.
  23. Второй элемент h2 должен содержать текст Лучшие маршруты.
  24. У вас должно быть не менее трех элементов figure.
  25. Каждый элемент figure должен содержать элемент привязки в качестве своего первого дочернего элемента.
  26. Каждый элемент figure должен содержать элемент figcaption в качестве своего второго дочернего элемента.
  27. Каждый figcaption должен содержать некоторый текст.
  28. Каждый из элементов a, являющихся дочерними элементами элементов figure, должен содержать изображение.
  29. Каждый элемент img должен иметь действительный атрибут src.
  30. Каждый элемент img должен иметь атрибут alt с соответствующим значением.
  31. Каждый элемент a должен иметь атрибут href со значением https://www.freecodecamp.org/learn. Не забудьте о ссылках в элементах списка.
  32. Каждый элемент a должен иметь атрибут target со значением _blank. Не забудьте про ссылки в элементах списка.
