Шаг 2

Похоже, на странице всё ещё есть некоторые проблемы с использованием заголовков. Если вы посмотрите на первый элемент div, вы увидите два элемента h4, используемых в качестве подзаголовков. Лучше было бы использовать элементы h3 для этих подзаголовков, поскольку они находятся на один уровень ниже элемента h2.

Замените два элемента h4 на элементы h3.

Файл index.html 
      
        <!DOCTYPE html>

          </ul>
