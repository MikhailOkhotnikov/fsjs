Шаг 1

Camperbot создал блог о процессе обучения программированию, но, похоже, на странице есть проблемы с доступностью и несоответствия стандартам. Ваша задача в этом мастер-классе — исправить эти проблемы для Camperbot.

Первая группа ошибок связана с использованием заголовков. На предыдущем уроке вы узнали, что лучшей практикой является использование только одного заголовка h1 на странице. Но, похоже, Camperbot использует несколько заголовков h1.

Оставьте заголовок <h1>Добро пожаловать в блог Camperbot!</h1> без изменений, а остальные элементы h1 замените на элементы h2.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
        <head>
          <meta charset="UTF-8">
          <title>Блог Camperbot</title>
        </head>
        <body>
          <h1>Добро пожаловать в блог Camperbot!</h1>

          <h2>Навигация</h2>
          <ul>
            <li><a href="#post1">Моё путешествие</a></li>
            <li><a href="#post2">Доступность</a></li>
            <li><a href="#post3">Следующие шаги</a></li>
          </ul>
          <div>
            <h1 id="post1">Мой путь к обучению программированию</h1>
            <p>Я начал изучать программирование несколько месяцев назад, и это было невероятно!</p>
            
            <h4>Первые трудности</h4>
            <p>Поначалу синтаксис был очень запутанным.</p>
            
            <h4>Прорывы</h4>
            <p>В конце концов, все начало вставать на свои места.</p>
          </div>

          <div>
            <h1 id="post2">Вопросы доступности</h1>
            <p>Сегодня я узнал, что не все пользуются интернетом так же, как я.</p>

            <h5>Программы чтения с экрана</h5>
            <p>Эти инструменты помогают пользователям с нарушениями зрения просматривать веб-сайты.</p>
          </div>

          <div>
            <h1 id="post3">Что дальше?</h1>
            <p>Я с нетерпением жду возможности погрузиться в JavaScript и создавать интерактивные функции!</p>

            <h3>Скоро: Мой первый проект на JavaScript!</h3>
            <p>Следите за обновлениями, вас ждут интересные интерактивные функции блога.</p>
          </div>
          <h3>Свяжитесь со мной</h3>
          <p>Напишите мне по электронной почте: camperbot@blog.io</p>
        </body>
        </html>
