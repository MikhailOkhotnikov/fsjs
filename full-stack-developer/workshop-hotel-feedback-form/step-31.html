Шаг 31

Если вы хотите, чтобы у пользователей было больше места для написания комментариев, вы можете использовать элемент textarea.

Элемент textarea — это многострочный текстовый элемент управления, позволяющий пользователям вводить текст длиннее одной строки. Его можно использовать для создания поля для комментариев, поля для сообщений или другого текстового поля, требующего ввода нескольких строк.

Вот пример элемента textarea:

Пример кода 
      
        <textarea id="comments" name="comments" rows="4" cols="50"></textarea>

Атрибут rows используется для указания видимой высоты текстового поля, а атрибут cols — для указания видимой ширины текстового поля.

Ниже элемента label добавьте элемент textarea. На следующем шаге вы добавите необходимые атрибуты.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">

            <meta charset="UTF-8" />

                </fieldset>
                <fieldset>
                  <legend>Вы впервые остановились в нашем отеле?</legend>
                  <input type="radio" name="hotel-stay" id="yes-option">
                  <label for="yes-option">Да</label>
                  <input type="radio" name="hotel-stay" id="no-option">
                  <label for="no-option">Нет</label>
                </fieldset>
                <fieldset>
                  <legend>Почему вы выбрали именно наш отель?(Отметьте все подходящие варианты)</legend>
                  <input type="checkbox" name="ads" id="ads" value="ads">

                  <input type="checkbox" name="recommendation" id="recommendation" value="recommendation">

                  <label for="reputation">Репутация</label>
                  <input type="checkbox" name="price" id="price" value="price">
                  <label for="price">Цена</label>
                </fieldset>      
                <fieldset>
                  <legend>Рейтинги</legend>
                  <label for="service">Как вам обслуживание?</label>
                  <select name="service" id="service">
                    <option value="poor">Плохое</option>
                    <option value="satisfactory">Удовлетворительное</option>
                    <option value="good">Хорошее</option>
                    <option value="very-good">Очень хорошее</option>
                    <option value="excellent" selected>Отличное</option>
                  </select>
                  <label for="food">Как вам еда?</label>
                  <select name="food" id="food">
                    <option value="poor">Плохое</option>
                    <option value="satisfactory">Удовлетворительное</option>
                    <option value="good">Хорошее</option>
                    <option value="very-good">Очень хорошее</option>
                    <option value="excellent" selected>Отличное</option>
                  </select>
                </fieldset>
                <label for="comments">Другие комментарии?</label>

              </form>
            </main>
          </body>
        </html>
