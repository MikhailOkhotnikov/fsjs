Шаг 19

Если вы хотите, чтобы пользователь выбрал несколько вариантов из списка, вы можете использовать флажки(checkboxes).

Вот пример того, как работать с флажками, предназначенными для выбора блюд:

Пример кода 
      
        <fieldset>
          <legend>Варианты блюд</legend>
          <input type="checkbox" id="pizza" name="food" value="pizza">
          <label for="pizza">Пицца</label>
          <input type="checkbox" id="burger" name="food" value="burger">
          <label for="burger">Бургер</label>
        </fieldset>

Атрибут value используется для указания значения, которое будет отправлено на сервер при отправке формы.

Ниже элемента legend добавьте input с типом checkbox, с атрибутами id, name и value, установленными на ads.

Примечание: ADS (Advertisement) — общий термин, обозначающий рекламные объявления в цифровой среде.

Под input с типом checkbox добавьте элемент label с текстом Реклама в социальных сетях. Атрибут for должен быть установлен на значение ads.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Форма обратной связи отеля</title>
          </head>
          <body>
            <header>
              <h1>Форма обратной связи отеля</h1>
              <p>Спасибо за то, что остановились у нас. Пожалуйста, оставьте отзыв о вашем недавнем пребывании.</p>
            </header>
            <main>
              <form method="POST" action="https://hotel-feedback.freecodecamp.org">
                <fieldset>
                  <legend>Персональная информация</legend>
                  <label for="full-name">Имя(обязательно):</label>
                  <input type="text" id="full-name" name="name" placeholder="например, Джон Доу" required size="20">
                  <label for="email">Адрес электронной почты(обязательно):</label>
                  <input type="email" name="email" id="email" placeholder="example@email.com" required size="20">
                  <label for="age">Возраст(необязательно):</label>
                  <input type="number" name="age" id="age" min="3" max="120">
                </fieldset>
                <fieldset>
                  <legend>Вы впервые остановились в нашем отеле?</legend>
                  <input type="radio" name="hotel-stay" id="yes-option">
                  <label for="yes-option">Да</label>
                  <input type="radio" name="hotel-stay" id="no-option">
                  <label for="no-option">Нет</label>
                </fieldset>
                <fieldset>
                  <legend>Почему вы выбрали именно наш отель?(Отметьте все подходящие варианты)</legend>

                </fieldset>                
              </form>
            </main>
          </body>
        </html>
