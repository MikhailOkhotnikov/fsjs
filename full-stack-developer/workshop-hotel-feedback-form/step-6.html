Шаг 6

На предыдущих уроках вы узнали, как связать элемент label с input следующим образом:

Пример кода 
      
        <label for="name">Имя:</label>
        <input type="text" id="name" name="name">

Атрибут for элемента label должен соответствовать атрибуту id элемента input. Это называется явной ассоциацией.

Под элементом legend добавьте элемент label с текстом Имя(обязательно):. Задайте для его атрибута for значение full-name.

Затем под элементом label добавьте элемент input без атрибутов. На следующих шагах вы добавите необходимые атрибуты.
