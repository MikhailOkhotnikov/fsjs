Шаг 8

Атрибут name используется для идентификации данных формы после их отправки на сервер.

Вот пример использования атрибута name:

Пример кода 
      
        <input type="email" name="email">

Добавьте атрибут name к элементу input со значением name.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Форма обратной связи отеля</title>
          </head>
          <body>
            <header>
              <h1>Форма обратной связи отеля</h1>
              <p>Спасибо за то, что остановились у нас. Пожалуйста, оставьте отзыв о вашем недавнем пребывании.</p>
            </header>
            <main>
              <form method="POST" action="https://hotel-feedback.freecodecamp.org">
                <fieldset>
                  <legend>Персональная информация</legend>
                  <label for="full-name">Имя(обязательно):</label>

                  <input type="text" id="full-name">
                  
                </fieldset>
              </form>
            </main>
          </body>
        </html>
