Шаг 11

Затем добавьте input с типом email под label электронной почты. Этот input должен иметь атрибут id со значением установленным на email. Также установите значение атрибута name на email.

Этот input также является обязательным, поэтому обязательно добавьте атрибут required.

Наконец, добавьте атрибут placeholder со значением example@email.com.
