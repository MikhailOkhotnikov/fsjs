При работе с элементами fieldset обычно используется заголовок для описания группы входных данных. Для этого можно использовать элемент legend.

Вот пример использования элемента legend:

Пример кода 
      
        <form action="/example-url">
          <fieldset>
            <legend>Персональная информация</legend>
            <!-- поля ввода находятся здесь -->
          </fieldset>
        </form>

Внутри элемента fieldset добавьте элемент legend с текстом Персональная информация.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Форма обратной связи отеля</title>
          </head>
          <body>
            <header>
              <h1>Форма обратной связи отеля</h1>
              <p>Спасибо за то, что остановились у нас. Пожалуйста, оставьте отзыв о вашем недавнем пребывании.</p>
            </header>
            <main>
              <form method="POST" action="https://hotel-feedback.freecodecamp.org">
                <fieldset>

                </fieldset>
              </form>
            </main>
          </body>
        </html>
