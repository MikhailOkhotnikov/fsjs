Шаг 7

Если пользователь вводит свое полное имя, input будет принимать открытый текст.

На предыдущих уроках вы узнали, как работать с атрибутом type следующим образом:

Пример кода 
      
        <input type="text">

Для существующего элемента input добавьте атрибут type со значением text.

Также присвойте элементу ввода атрибут id со значением full-name.
