Числовой input используется для создания числового поля ввода.

Вот пример поля ввода числа:

Пример кода 
      
        <input type="number" id="age" name="age" min="18" max="120">

Атрибуты min и max используются для установки минимального и максимального значений, которые можно ввести в поле ввода.

Под элементом label добавьте поле ввода с атрибутом type, установленным на number, и id, установленным на age.

Атрибут name должен быть установлен на age, атрибут min должен быть установлен на 3, а атрибут max должен быть установлен на 120.
