Формы состоят из полей ввода(input), куда пользователи могут вводить свои данные. Связанные поля ввода можно группировать с помощью элемента fieldset.

Вот пример использования элемента fieldset:

Пример кода 
      
        <form action="/example-url">
          <fieldset>
          <!-- поля ввода находятся здесь-->
          </fieldset>
        </form>

Внутри элемента формы добавьте элемент fieldset.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <title>Форма обратной связи отеля</title>
          </head>
          <body>
            <header>
              <h1>Форма обратной связи отеля</h1>
              <p>Спасибо за то, что остановились у нас. Пожалуйста, оставьте отзыв о вашем недавнем пребывании.</p>
            </header>
            <main>
              <form method="POST" action="https://hotel-feedback.freecodecamp.org">

              </form>
            </main>
          </body>
        </html>
