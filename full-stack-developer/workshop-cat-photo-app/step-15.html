Чтобы открыть ссылки в новой вкладке, можно использовать атрибут target в элементе привязки (a).

Атрибут target указывает, где открыть связанный документ. target="_blank" открывает связанный документ в новой вкладке или окне.

Вот базовый синтаксис для элемента a с атрибутом target: 

    
      <a href="https://www.freecodecamp.org" target="_blank">freeCodeCamp</a>

Добавьте атрибут target со значением _blank к открывающему тегу элемента привязки (a), чтобы ссылка открывалась в новой вкладке.

Файл index.html 
      
        <html>
          <body>
            <main>
              <h1>CatPhotoApp</h1>
              <h2>Фотографии кошек</h2>
              <!-- Задача: Добавить ссылку на фотографии кошек -->
              
              <p>Все любят <a href="https://cdn.freecodecamp.org/curriculum/cat-photo-app/running-cats.jpg">милых котиков</a> в сети!</p>

              <p>Больше <a href="https://www.freecodecamp.org">фотографий кошек</a> в нашей галерее.</p>
              <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Симпатичный оранжевый кот, лежащий на спине">         
            </main>
          </body>
        </html>
