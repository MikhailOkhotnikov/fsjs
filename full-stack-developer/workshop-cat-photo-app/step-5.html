Шаг 5

В HTML5 есть несколько элементов, которые определяют различные области контента. Эти элементы упрощают чтение HTML и помогают с поисковой оптимизацией (SEO) и доступностью.

Элемент main используется для представления основного содержимого основного блока HTML-документа. Содержимое внутри элемента main должно быть уникальным для документа и не должно повторяться в других частях документа.

Пример кода

    
      <main>
        <h1>Самое важное содержание документа</h1>
        <p>Еще кое-что важное...</p>
      </main>

Выделите основной раздел этой страницы, добавив открывающий тег <main> перед элементом h1 и закрывающий тег </main> после элемента p.

Файл index.html 
      
        <html></html>
          <body>
            
            <h1>CatPhotoApp</h1>
            <h2>Фотографии кошек</h2>
            <!-- Задача: Добавить ссылку на фотографии кошек -->
            <p>Смотрите больше фотографий кошек в нашей галерее.</p>
  
          </body>
        </html>
