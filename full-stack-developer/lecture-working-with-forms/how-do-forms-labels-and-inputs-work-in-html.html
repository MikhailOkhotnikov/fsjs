Как работают формы, метки и поля ввода в HTML?

Элемент form в HTML используется для сбора информации о пользователе, такой как имя и адрес электронной почты. Вот пример элемента form: 

    
      <form action="url-goes-here">
        <!-- элементы ввода идут сюда -->
      </form>

Атрибут action указывает, куда будут отправлены данные формы после отправки. Для сбора конкретной информации, например, имен и адресов электронной почты, используется элемент input. Вот пример использования элемента input:

Включите интерактивный редактор и взаимодействуйте с элементом input в окне предварительного просмотра, введя свое имя в поле. 

    
      <form action="">
        <input type="text" />
      </form>

Элементы input являются пустыми и не имеют закрывающих тегов. Атрибут type определяет тип данных, ожидаемый от пользователя. В данном случае данные будут представлены в виде обычного текста. Чтобы добавить метку для поля ввода, используйте элемент label. Вот пример использования элемента label с текстом Полное имя:.

Нажмите на текст Полное имя в окне предварительного просмотра, и вы увидите, что поле ввода станет активным. Для работы с примером необходимо включить интерактивный редактор. 

    
      <form action="">
        <label>
          Полное имя:
          <input type="text" />
        </label>
      </form>

Вкладывая input в элемент label, вы создаёте неявную связь между меткой и полем ввода. Термин «неявный» относится к чему-то, что понимается или подразумевается без необходимости явного указания или определения с помощью дополнительных атрибутов или элементов. Чтобы явно связать label с input, можно использовать атрибут for. Ниже приведён пример использования атрибута for для метки адреса электронной почты.

Взаимодействуйте с элементом input в окне предварительного просмотра, введя поддельный адрес электронной почты, например, jane@example.com. Для взаимодействия с примером необходимо включить интерактивный редактор. 

    
      <form action="">
        <label for="email"> адрес электронной почты: </label>
        <input type="email" id="email" />
      </form>

При использовании явной связи значения атрибутов for и id должны быть одинаковыми. В данном случае оба значения равны email. Тип email в поле ввода обеспечивает базовую проверку корректности форматирования адресов электронной почты. Если вы хотите показать пользователям дополнительные подсказки относительно ожидаемого ввода, можно использовать атрибут placeholder(заполнитель). Ниже приведен пример использования атрибута placeholder в поле ввода email.

Включите интерактивный редактор, нажмите на поле ввода адреса электронной почты и начните вводить адрес электронной почты, и вы увидите, как заполнитель текста исчезнет. 

    
      <form action="">
        <label for="email"> адрес электронной почты: </label>
        <input type="email" id="email" placeholder="example@email.com" />
      </form>

В качестве текста-заполнителя вы хотите предоставить краткий текст, который будет полезен, чтобы показать формат и тип данных, которые вы ожидаете от своих пользователей. В этом случае текст-заполнитель example@email.com показывает пользователю, что он должен ввести правильно отформатированный адрес электронной почты.
