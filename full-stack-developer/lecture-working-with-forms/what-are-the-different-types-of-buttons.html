Какие существуют типы кнопок и когда следует их использовать?

Элемент button используется для выполнения определённого действия при активации. Вот пример элемента button с текстом Начать игру. 

    
      <button>Начать игру</button>

Другие примеры использования элемента button включают отправку формы, отображение модального окна или открытие/закрытие бокового меню. Элемент button имеет атрибут type, который управляет поведением кнопки при её активации. Первым возможным значением атрибута type будет тип button. Вот пример использования элемента button с типом button и текстом Показать оповещение: 

    
      <button type="button">Показать оповещение</button>

По умолчанию кнопка не выполняет никаких действий при активации. Однако вы можете добавить код JavaScript, чтобы сделать кнопку интерактивной, например, отображая оповещение, как в данном случае.

Нажмите кнопку Показать оповещение, чтобы увидеть всплывающее оповещение на экране. Для работы с примером необходимо включить интерактивный редактор.

ПРИМЕЧАНИЕ: Этот интерактивный пример использует JavaScript, но вам не нужно беспокоиться о понимании кода JavaScript. Вы изучите JavaScript в следующих модулях.

Пример кода
Файл index.html 
        
          <button type="button">Показать оповещение</button>
          <script src="index.js"></script>
Файл index.js 
        
          const btn = document.querySelector("button");
          btn.addEventListener("click", () => alert("Вы нажали на кнопку оповещения"));

Другим возможным значением атрибута type является значение submit. Ниже приведен пример использования элемента button с типом submit. 

    
      <form action="">
        <label for="email">Адрес электронной почты:</label>
        <input type="email" id="email" name="email" />
        <button type="submit">Отправить форму</button>
      </form>

Внутри этого элемента form есть label и input для ввода адреса электронной почты пользователя. Когда пользователь нажимает кнопку Отправить форму, его данные отправляются на сервер и обрабатываются. Третье возможное значение атрибута type — значение reset(сбросить). Вот пример элемента form с кнопками сброса и отправки.

Включите интерактивный редактор и взаимодействуйте с полем адреса электронной почты в окне предварительного просмотра, указав поддельный адрес. Затем нажмите кнопку сброса, и ваш адрес исчезнет из поля. 

    
      <form action="">
        <label for="email">Адрес электронной почты:</label>
        <input type="email" id="email" name="email" />
        <button type="reset">Сбросить форму</button>
        <button type="submit">Отправить форму</button>
      </form>

В этом модифицированном примере элементы label и input используются для сбора адреса электронной почты пользователя. Когда пользователь нажимает кнопку сброса, все введённые данные удаляются. Важно отметить, что кнопки сброса обычно не являются лучшим решением, поскольку они могут привести к случайному сбросу данных пользователем. Кроме того, большое количество кнопок в форме может загромождать пользовательский интерфейс.

Другой способ создания кнопок в HTML — использование элемента input. У элемента input также есть атрибут type с возможными значениями: submit, reset и button. Вот пример использования элемента input с типом button:

Пример кода
Файл index.html 
        
          <input class="start-btn" type="button" value="Начать игру" />
          <script src="index.js"></script>
Файл index.js 
        
          document.addEventListener("DOMContentLoaded", () => {
            const btn = document.querySelector(".start-btn");
            btn.addEventListener("click", () => {
              const paraEl = document.createElement("p");
              const bodyEl = document.querySelector("body");

              bodyEl.appendChild(paraEl);
              paraEl.textContent = "Игра началась!!!";
            });
          });

Атрибут value используется для отображения текста кнопки. В чём же разница между элементами input и button? Элементы input являются пустыми, то есть не могут иметь дочерних узлов, таких как текст, и могут иметь только начальный тег. С другой стороны, элемент button обеспечивает большую гибкость, поскольку в него можно вкладывать текст, изображения и значки.
