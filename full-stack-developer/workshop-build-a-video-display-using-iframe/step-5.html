Шаг 5

Один из атрибутов — allow. Это своего рода список разрешений, сообщающий браузеру, какие функции iframe может использовать.

Вот элемент iframe с атрибутом allow:

Пример кода: 
      
        <iframe
          allow="accelerometer autoplay clipboard-write encrypted-media gyroscope picture-in-picture web-share"
        ></iframe>

Добавьте атрибут allow со значениями accelerometer, autoplay и clipboard-write.

accelerometer позволяет iframe использовать датчики движения, чтобы он мог определять такие вещи, как наклон и вращение устройства. Функция autoplay позволяет автоматически начинать воспроизведение видео, а функция clipboard-write позволяет iframe записывать данные в буфер обмена пользователя.
