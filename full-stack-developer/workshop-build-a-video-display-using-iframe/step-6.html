Шаг 6

Добавьте encrypted-media, gyroscope и web-share к существующим значениям в атрибуте allow.

Эти три функции позволят использовать зашифрованные медиарасширения для защиты видео, позволят iframe переключаться в режим «картинка в картинке» при необходимости и позволят делиться содержимым iframe через собственные диалоговые окна общего доступа устройства.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Отображение видео в iframe</title>
          </head>
          <body>
            <h1>iframe Видеодисплей</h1>
            <iframe
              width="560"
              height="315"
              src="https://www.youtube.com/embed/I0_951_MPE0"
              allow="accelerometer autoplay clipboard-write"
            >
            
            </iframe>
          </body>
        </html>
