Шаг 8

И наконец, атрибут, который вы добавите, — это allowfullscreen. Как следует из названия, он позволяет просматривать видео в полноэкранном режиме.

На этом семинар завершен!

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Отображение видео в iframe</title>
          </head>
          <body>
            <h1>iframe Видеодисплей</h1>
            <iframe
              width="560"
              height="315"
              src="https://www.youtube.com/embed/I0_951_MPE0"
              allow="accelerometer autoplay clipboard-write encrypted-media gyroscope web-share"
              referrerpolicy="strict-origin-when-cross-origin"
              
            >
            
            </iframe>
          </body>
        </html>
