Шаг 7

Следующий атрибут, который вы добавите, — referrerpolicy. Это правило определяет, насколько подробной информацией вы делитесь при переходе с одной страницы на другую.

Добавьте атрибут referrerpolicy и установите для него значение strict-origin-when-cross-origin. Это позволит использовать полный адрес на одном сайте, только имя сайта на других сайтах и ​​ничего на незащищённых сайтах.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Отображение видео в iframe</title>
          </head>
          <body>
            <h1>iframe Видеодисплей</h1>
            <iframe
              width="560"
              height="315"
              src="https://www.youtube.com/embed/I0_951_MPE0"
              allow="accelerometer autoplay clipboard-write encrypted-media gyroscope web-share"
              
            >
            
            </iframe>
          </body>
        </html>
