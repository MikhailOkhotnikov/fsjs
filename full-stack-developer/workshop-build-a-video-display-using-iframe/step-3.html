Шаг 3

В первом уроке, посвящённом элементу iframe, вы узнали, что это заменяемый элемент, как и img. Это значит, что он также может использовать свойства width и height для определения своей высоты и ширины.

Задайте элементу iframe width равный 560 и height равный 315.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8" />
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <title>Отображение видео в iframe</title>
          </head>
          <body>
            <h1>iframe Видеодисплей</h1>
            <iframe>
            
            </iframe>
          </body>
        </html>
