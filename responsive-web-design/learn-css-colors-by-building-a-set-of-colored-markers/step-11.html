Пришло время добавить немного цвета в маркер. Помните, что один из способов добавить цвет в элемент — использовать ключевое слово цвета, например black, cyan, или yellow.

Напомню, что вот как можно нацелиться на класс freecodecamp:

Пример кода 

    
      .freecodecamp {
  
      }

Создайте новое правило CSS, нацеленное на класс marker, и задайте для его свойства background-color значение red.

Примечание: Вы не увидите никаких изменений после добавления CSS.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker"></div>
            </div>
          </body>
        </html>
