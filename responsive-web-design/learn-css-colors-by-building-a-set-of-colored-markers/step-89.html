Шаг 89

Но что, если вы хотите расширить тень ещё больше? Это можно сделать с помощью необязательного значения spreadRadius:

Пример кода

    
      box-shadow: offsetX offsetY blurRadius spreadRadius color;

Как и blurRadius, spreadRadius по умолчанию имеет значение 0, если он не указан.

Потренируйтесь, добавив 5-пиксельную тень непосредственно вокруг синего маркера.

В правиле CSS .blue добавьте свойство box-shadow со значениями 0 для offsetX, 0 для offsetY, 0 для blurRadius, 5px для spreadRadius и blue для color.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker red">
                <div class="cap"></div>
                <div class="sleeve"></div>
              </div>
              <div class="marker green">
                <div class="cap"></div>
                <div class="sleeve"></div>
              </div>
              <div class="marker blue">
                <div class="cap"></div>
                <div class="sleeve"></div>
              </div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .cap {
          width: 60px;
          height: 25px;
        }

        .sleeve {
          width: 110px;
          height: 25px;
          background-color: rgba(255, 255, 255, 0.5);
          border-left: 10px double rgba(0, 0, 0, 0.75);
        }

        .cap, .sleeve {
          display: inline-block;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .red {
          background: linear-gradient(rgb(122, 74, 14), rgb(245, 62, 113), rgb(162, 27, 27));
          box-shadow: -5px -5px red;
        }

        .green {
          background: linear-gradient(#55680D, #71F53E, #116C31);
          box-shadow: 5px 5px 5px green;
        }

        .blue {
          background: linear-gradient( hsl(186, 76%, 16%), hsl(223, 90%, 60%), hsl(240, 56%, 42%) );
        }
