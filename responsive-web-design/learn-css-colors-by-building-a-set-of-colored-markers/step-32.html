Шаг 32

Обратите внимание, что для получения оранжевого цвета вам нужно было увеличить интенсивность красного и уменьшить интенсивность зелёного rgb значений. Это связано с тем, что оранжевый — это сочетание красного и жёлтого.

Чтобы создать третичный цвет весенний зелёный, объедините голубой с зелёным. Измените функцию rgb в правиле CSS .two так, чтобы зелёный был максимальным значением, а синий — 127.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker one"></div>
              <div class="marker two"></div>
              <div class="marker three"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .one {
          background-color: rgb(255, 127, 0);
        }

        .two {
          background-color: rgb(0, 255, 255);
        }

        .three {
          background-color: rgb(255, 0, 255);
        }
