Шаг 16

Чтобы придать маркерам разные цвета, вам нужно будет добавить уникальный класс к каждому из них. Несколько классов можно добавить к элементу, перечислив их в атрибуте class и разделив пробелом. Например, следующий код добавляет классы animal и dog к элементу div.

Пример кода

    
      <div class="animal dog"></div>

Если вы добавляете несколько классов к HTML-элементу, стили первых перечисленных вами классов могут быть переопределены более поздними классами.

Для начала добавьте класс one к первому элементу div с классом marker.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker"></div>
              <div class="marker"></div>
              <div class="marker"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          background-color: red;
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }
