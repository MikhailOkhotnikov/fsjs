Шаг 34

Существуют еще три третичных цвета: шартрез-зеленый (зеленый + желтый), лазурный (синий + голубой) и розовый (красный + пурпурный).

Чтобы создать шартрезовый зеленый, обновите функцию rgb в правиле CSS .one так, чтобы красный цвет имел значение 127, и установите для зеленого максимальное значение.

Для лазурного обновите функцию rgb в правиле CSS .two так, чтобы зеленый цвет имел значение 127, а синий — максимальное значение.

А для розового цвета, который иногда называют ярко-розовым, обновите функцию rgb в правиле CSS .three так, чтобы синий цвет имел значение 127, а красный — максимальное значение.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker one"></div>
              <div class="marker two"></div>
              <div class="marker three"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .one {
          background-color: rgb(255, 127, 0);
        }

        .two {
          background-color: rgb(0, 255, 127);
        }

        .three {
          background-color: rgb(127, 0, 255);
        }
