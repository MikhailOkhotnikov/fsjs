Шаг 4

Атрибут charset определяет кодировку символов, используемую в документе. utf-8 (Unicode Transformation Format, формат преобразования Unicode – 8 бит) — стандарт кодировки символов, используемый для электронной связи.

Внутри элемента head разместите элемент meta с атрибутом charset, установленным на "utf-8".

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Цветные маркеры</title>
          </head>
          <body>
            
          </body>
        </html>
