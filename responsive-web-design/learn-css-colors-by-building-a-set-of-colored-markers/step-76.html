Шаг 76

Ваш колпачок выглядит хорошо, но он выглядел бы ещё лучше, если бы располагался ближе к правой стороне маркера. Один из способов сделать это — добавить ещё один элемент перед колпачком, чтобы сдвинуть его вправо.

Добавьте новый элемент div с классом cap перед div элементом колпачка.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker red">
                <div class="sleeve"></div>
              </div>
              <div class="marker green"></div>
              <div class="marker blue"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .sleeve {
          width: 110px;
          height: 25px;
          background-color: rgba(255, 255, 255, 0.5);
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .red {
          background: linear-gradient(rgb(122, 74, 14), rgb(245, 62, 113), rgb(162, 27, 27));
        }

        .green {
          background: linear-gradient(#55680D, #71F53E, #116C31);
        }

        .blue {
          background: linear-gradient( hsl(186, 76%, 16%), hsl(223, 90%, 60%), hsl(240, 56%, 42%) );
        }
