Шаг 3

Помните, что элемент title дает поисковым системам дополнительную информацию о странице. Он также отображает содержимое этого элемента title еще двумя способами:

  • в строке заголовка, когда страница открыта
  • на вкладке браузера для страницы при наведении на нее курсора. Даже если эта вкладка неактивна, как только вы наводите на нее курсор, отображается текст заголовка(title).

В элемент head вставьте элемент title с текстом Цветные маркеры.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>

          </head>
          <body>
            
          </body>
        </html>
