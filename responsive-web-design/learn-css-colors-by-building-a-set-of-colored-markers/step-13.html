Шаг 13

Ваш маркер выглядел бы лучше, если бы он был расположен по центру страницы. Простой способ сделать это — использовать свойство сокращения margin.

В последнем проекте вы устанавливали поля для элементов по отдельности с помощью таких свойств, как margin-top и margin-left. Сокращённое свойство margin позволяет легко устанавливать несколько полей одновременно.

Чтобы расположить маркер по центру страницы, задайте для его свойства margin значение auto. Это устанавливает значения margin-top, margin-right, margin-bottom и margin-left как auto.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          background-color: red;
          height: 25px;
          width: 200px;
        }
