Шаг 46

Очень распространённый способ применения цвета к элементу с помощью CSS — использование шестнадцатеричных(hexadecimal) значений. Хотя шестнадцатеричные значения кажутся сложными, на самом деле это просто ещё одна форма значений RGB.

Шестнадцатеричные значения цвета начинаются с символа # и состоят из шести символов от 0 до 9 и от A до F. Первая пара символов обозначает красный цвет, вторая пара — зелёный, а третья пара — синий. Например, #4B5320.

В селекторе классов .green задайте для свойства background-color шестнадцатеричный код цвета со значениями 00 для красного, FF для зелёного и 00 для синего.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker red"></div>
              <div class="marker green"></div>
              <div class="marker blue"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .red {
          background-color: rgb(255, 0, 0);
        }

        .green {
          background-color: rgb(0, 0, 0);
        }

        .blue {
          background-color: rgb(0, 0, 0);
        }
