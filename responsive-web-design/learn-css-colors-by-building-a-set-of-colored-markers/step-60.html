Шаг 60

Красный маркер выглядит гораздо реалистичнее. Теперь вы сделаете то же самое для зеленого маркера, используя комбинацию функции linear-gradient и шестнадцатеричных цветов.

В правиле CSS .green измените свойство background-color на background.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker red"></div>
              <div class="marker green"></div>
              <div class="marker blue"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .red {
          background: linear-gradient(180deg, rgb(122, 74, 14) 0%, rgb(245, 62, 113) 50%, rgb(162, 27, 27) 100%);
        }

        .green {
          background-color: #007F00;
        }

        .blue {
          background-color: hsl(240, 100%, 50%);
        }
