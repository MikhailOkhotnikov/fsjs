Шаг 37

Обратите внимание, что красный и голубой цвета очень яркие и расположены рядом друг с другом. Такой контраст может отвлекать, если он слишком часто используется на веб-сайте, и затруднять чтение текста, если он расположен на фоне контрастного цвета.

Лучше выбрать один цвет в качестве основного и использовать дополнительный цвет в качестве акцента, чтобы привлечь внимание к определённому контенту на странице.

Сначала в правиле h1 используйте функцию rgb для установки background-color цвета на голубой.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker one"></div>
              <div class="marker two"></div>
              <div class="marker three"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .one {
          background-color: rgb(255, 0, 0);
        }

        .two {
          background-color: rgb(0, 255, 255);
        }

        .three {
          background-color: rgb(0, 0, 0);
        }
