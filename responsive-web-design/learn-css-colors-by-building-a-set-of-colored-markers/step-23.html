Шаг 23

Обратите внимание, что background-color для вашего маркера по-прежнему красный. Это связано с тем, что вы установили максимальное значение rgb для функции красного цвета, равное 255, или 100% красного, а значения зелёного и синего установили на 0.

Теперь используйте функцию rgb, чтобы задать другие цвета.

В правиле CSS .two используйте функцию rgb для установки background-color максимального значения для зелёного цвета и 0 для других значений. А в правиле CSS .three используйте функцию rgb для установки background-color максимального значения для синего цвета и 0 для других значений.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker one"></div>
              <div class="marker two"></div>
              <div class="marker three"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .container {
          background-color: rgb(0, 0, 0);
        }

        .one {
          background-color: rgb(255, 0, 0);
        }

        .two {
          background-color: green;
        }

        .three {
          background-color: blue;
        }
