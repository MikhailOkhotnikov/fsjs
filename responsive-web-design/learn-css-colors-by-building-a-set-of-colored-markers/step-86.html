Последнее, что вам нужно сделать, — это добавить лёгкую тень к каждому маркеру, чтобы они выглядели ещё более реалистично.

Свойство box-shadow позволяет применить одну или несколько теней вокруг элемента. Вот базовый синтаксис:

Пример кода

    
      box-shadow: offsetX offsetY color;

Вот как работают значения offsetX и offsetY:

- оба offsetX и offsetY принимают числовые значения в px и других единицах CSS

- положительное значение offsetX перемещает тень вправо, а отрицательное значение - влево

- положительное значение offsetY перемещает тень вниз, а отрицательное - вверх

- если вам нужно нулевое значение (0) для любого или для обоих offsetX и offsetY, вам не нужно добавлять единицу измерения. Каждый браузер понимает, что нулевое значение означает отсутствие изменений.

Высота и ширина тени определяются высотой и шириной элемента, к которому она применяется. Вы также можете использовать необязательное значение spreadRadius, чтобы увеличить радиус действия тени. Подробнее об этом позже.

Начните с добавления простой тени к красному маркеру.

В правиле CSS .red добавьте свойство box-shadow со значениями 5px для offsetX, 5px для offsetY и red для color.
