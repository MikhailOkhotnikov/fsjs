Шаг 78

Похоже, ваш колпачёк исчез, но не волнуйтесь — он все еще там. Произошло следующее: ваш новый div cap занимает всю ширину маркера и смещает колпачёк вниз на следующую строку.

Это происходит потому, что свойство display по умолчанию для элементов div — block. Поэтому, когда два элемента block находятся рядом друг с другом, они накладываются друг на друга как настоящие блоки. Например, все ваши элементы marker накладываются друг на друга.

Чтобы расположить два элемента div на одной строке, установите для их свойств display значение inline-block.

Создайте новое правило, которое будет охватывать классы cap и sleeve, и установите для display значение inline-block.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker red">
                <div class="cap"></div>
                <div class="sleeve"></div>
              </div>
              <div class="marker green"></div>
              <div class="marker blue"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;

        }

        }

        }
