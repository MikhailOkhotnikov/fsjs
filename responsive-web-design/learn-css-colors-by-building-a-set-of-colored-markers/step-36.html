Шаг 36

Цветовой круг — это круг, в котором похожие цвета, или оттенки, расположены близко друг к другу, а разные — дальше друг от друга. Например, чистый красный цвет находится между оттенками розового и оранжевого.

Два цвета, расположенные напротив друг друга на цветовом круге, называются дополнительные цвета. Если смешать два дополнительных цвета, получится серый. Но если расположить их рядом, эти цвета создадут сильный визуальный контраст и будут казаться ярче.

В функции rgb для правила CSS .one установите максимальное значение 255 красного для получения чистого красного цвета. В функции rgb для правила CSS .two установите максимальное значение 255 зелёного и синего для получения голубого цвета.
