Шаг 12

Цвет фона был применён, но поскольку в элементе маркера div нет содержимого, по умолчанию он не имеет высоты.

В вашем CSS правиле .marker установите для свойства height значение 25px и для свойства width значение 200px

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          background-color: red;
        }
