Шаг 49

Вы узнали несколько способов задать плоские цвета в CSS, но вы также можете использовать цветовой переход или градиент для элемента.

Градиент — это когда один цвет плавно переходит в другой. Функция linear-gradient в CSS позволяет вам контролировать направление перехода вдоль линии и используемые цвета.

Следует помнить, что функция linear-gradient фактически создаёт элемент image и обычно используется в сочетании со свойством background, которое может принимать изображение в качестве значения.

В правиле CSS .red измените background-color свойство на background.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker red"></div>
              <div class="marker green"></div>
              <div class="marker blue"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .red {
          background-color: rgb(255, 0, 0);
        }

        .green {
          background-color: #007F00;
        }

        .blue {
          background-color: hsl(240, 100%, 50%);
        }
