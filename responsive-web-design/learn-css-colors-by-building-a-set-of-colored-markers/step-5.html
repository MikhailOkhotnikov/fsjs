Шаг 5

На веб-странице может быть несколько элементов meta. Каждый meta-элемент добавляет информацию о странице, которую нельзя выразить с помощью других HTML-элементов.

Добавьте ещё один элемент meta внутри head. Присвойте ему атрибут name со значением "viewport" и атрибут content со значением "width=device-width, initial-scale=1.0", чтобы ваша страница выглядела одинаково на всех устройствах.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            
          </body>
        </html>
