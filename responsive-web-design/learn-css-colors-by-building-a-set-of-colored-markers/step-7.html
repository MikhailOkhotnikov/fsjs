Шаг 7

В этом проекте вы будете работать с внешним файлом CSS для стилизации страницы. Вам необходимо создать файл styles.css. Но прежде чем вы сможете его использовать, вам нужно будет связать его со страницей.

Вложите элемент link в элемент head. Дайте ему атрибут rel, установленный на "stylesheet", и атрибут href, установленный на "styles.css".

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
          </body>
        </html>
