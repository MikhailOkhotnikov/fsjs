Шаг 64

Даже без цветовых переходов вы могли заметить, что цвета зелёного маркера меняются в тех же точках, что и у красного маркера. Первый цвет находится в начале (0%), второй — в середине (50%), а последний — в конце (100%) линии градиента.

Функция linear-gradient автоматически вычисляет эти значения и по умолчанию равномерно распределяет цвета вдоль линии градиента.

В правиле CSS .red удалите три цветовые градации из функции linear-gradient , чтобы немного очистить код.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker red"></div>
              <div class="marker green"></div>
              <div class="marker blue"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .red {
          background: linear-gradient(180deg, rgb(122, 74, 14) 0%, rgb(245, 62, 113) 50%, rgb(162, 27, 27) 100%);
        }

        .green {
          background: linear-gradient(180deg, #55680D, #71F53E, #116C31);
        }

        .blue {
          background-color: hsl(240, 100%, 50%);
        }
