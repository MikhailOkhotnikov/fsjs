Шаг 14

Теперь, когда вы расположили один маркер по центру и закрасили его, пришло время добавить остальные маркеры.

В container div добавьте ещё два элемента div и присвойте каждому из них класс marker.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          background-color: red;
          height: 25px;
          width: 200px;
          margin: auto;
        }
