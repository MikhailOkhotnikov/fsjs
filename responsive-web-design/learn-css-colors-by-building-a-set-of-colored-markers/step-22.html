Функция — это фрагмент кода, который может принимать входные данные и выполнять определённое действие. Функция CSS rgb принимает значения, или аргументы, для красного, зелёного и синего цветов и выдаёт цвет:

Пример кода

    
      rgb(red, green, blue);

Каждое значение красного, зелёного и синего — это число от 0 до 255. 0 означает, что этого цвета 0%, и он чёрный. 255 означает, что этого цвета 100%.

В правиле CSS .one замените ключевое слово цвета red на функцию rgb . Для функции rgb установите значение для красного на 255, значение для зелёного на 0 и значение для синего на 0.
