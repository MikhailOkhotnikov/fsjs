Шаг 75

Вы уже знакомы с функцией rgb для установки цветов. Чтобы добавить альфа-канал к цвету rgb, используйте функцию rgba.

Функция rgba работает так же, как функция rgb, но принимает ещё одно число от 0 до 1.0 для альфа-канала:

Пример кода

    
      rgba(redValue, greenValue, blueValue, alphaValue);

Вы также можете использовать альфа-канал с цветами hsl и hex. Скоро вы увидите, как это сделать.

В правиле .sleeve используйте функцию rgba для установки свойства background-color в значение «чистый белый» с непрозрачностью 50%.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker red">
                <div class="sleeve"></div>
              </div>
              <div class="marker green"></div>
              <div class="marker blue"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .sleeve {
          width: 110px;
          height: 25px;
          background-color: white;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .red {
          background: linear-gradient(rgb(122, 74, 14), rgb(245, 62, 113), rgb(162, 27, 27));
        }

        .green {
          background: linear-gradient(#55680D, #71F53E, #116C31);
        }

        .blue {
          background: linear-gradient( hsl(186, 76%, 16%), hsl(223, 90%, 60%), hsl(240, 56%, 42%) );
        }
