Шаг 8

Теперь, когда ваш внешний CSS-файл настроен, вы можете приступить к оформлению страницы.

Напомним, как выделить элемент абзаца и выровнять его по правому краю:

Пример кода 
    
      p {
        text-align: right;
      }

Создайте новое правило CSS, нацеленное на элемент h1 и задайте для его свойства text-align значение center.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
          </body>
        </html>
