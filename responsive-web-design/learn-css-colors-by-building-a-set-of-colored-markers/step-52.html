Шаг 52

Вы пока не увидите градиент, потому что для работы функции linear-gradient требуется как минимум два цветовых аргумента.

В той же функции linear-gradient используйте функцию rgb для установки второго аргумента цвета в чистый зелёный цвет.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker red"></div>
              <div class="marker green"></div>
              <div class="marker blue"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .red {
          background: linear-gradient(90deg, rgb(255, 0, 0));
        }

        .green {
          background-color: #007F00;
        }

        .blue {
          background-color: hsl(240, 100%, 50%);
        }
