Шаг 88

Обратите внимание, что края тени четкие. Это связано с тем, что для свойства blurRadius есть необязательное box-shadow значение:

Пример кода

    
      box-shadow: offsetX offsetY blurRadius color;

Если значение blurRadius не указано, по умолчанию используется 0 и получаются чёткие края. Чем выше значение blurRadius, тем сильнее эффект размытия.

В правиле CSS .green добавьте свойство box-shadow со значениями 5px для offsetX, 5px для offsetY, 5px для blurRadius и green для color.
