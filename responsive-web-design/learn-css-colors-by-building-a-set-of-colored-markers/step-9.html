Шаг 9

Теперь вы добавите несколько элементов, которые в конечном итоге превратятся в цветные маркеры.

Сначала внутри элемента body добавьте элемент div и присвойте его атрибуту class значение container. Убедитесь, что элемент div находится ниже элемента h1.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }
