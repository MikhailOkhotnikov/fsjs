Шаг 50

Функция linear-gradient очень гибкая — вот основной синтаксис, который вы будете использовать в этом руководстве:

Пример кода 

    
      linear-gradient(gradientDirection, color1, color2, ...);

gradientDirection — это направление линии, используемой для перехода. color1 и color2 — это аргументы цвета, которые будут использоваться в самом переходе. Это может быть любой цвет, включая ключевые слова цвета, шестнадцатеричные значения, rgb, или hsl.

Теперь применим градиент от красного к зеленому вдоль линии под углом 90 градусов к первому маркеру.

Сначала в правиле CSS .red задайте для свойства background значение linear-gradient() и передайте ему значение 90deg в качестве gradientDirection.
