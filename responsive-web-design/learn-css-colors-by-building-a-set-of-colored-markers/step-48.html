Шаг 48

Цветовая модель HSL, или оттенок(hue), насыщенность(saturation) и яркость(lightness), — это ещё один способ представления цветов.

Функция CSS hsl принимает 3 значения: число от 0 до 360 для оттенка, процент от 0 до 100 для насыщенности и процент от 0 до 100 для яркости.

Если представить себе цветовой круг, то красный цвет находится под углом 0 градусов, зелёный — под углом 120 градусов, а синий — под углом 240 градусов.

Насыщенность — это интенсивность цвета от 0%, или серого, до 100% для чистого цвета. К значениям насыщенности и яркости необходимо добавить знак процента %.

Яркость — это то, насколько ярким кажется цвет: от 0% (полностью черный) до 100% (полностью белый), при этом 50% — нейтральный цвет.

В правиле CSS .blue используйте функцию hsl для изменения свойства background-color на чистый синий цвет. Задайте оттенок 240, насыщенность 100% и яркость 50%.
