Шаг 39

Вы заметили, что ваш взгляд естественным образом притягивается к красному цвету в центре? При разработке сайта вы можете использовать этот эффект, чтобы привлечь внимание к важным заголовкам, кнопкам или ссылкам.

Помимо дополнительных цветов, существует несколько других важных цветовых сочетаний, но о них вы узнаете чуть позже.

А пока используйте функцию rgb в правиле CSS .two, чтобы сделать background-color чёрным.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker one"></div>
              <div class="marker two"></div>
              <div class="marker three"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
          background-color: rgb(0, 255, 255);
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .one {
          background-color: rgb(0, 0, 0);
        }

        .two {
          background-color: rgb(255, 0, 0);
        }

        .three {
          background-color: rgb(0, 0, 0);
        }
