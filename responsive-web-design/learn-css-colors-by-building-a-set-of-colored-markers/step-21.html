Шаг 21

Существуют две основные цветовые модели: аддитивная модель RGB (red-красный, green-зеленый, blue-синий), используемая в электронных устройствах, и субтрактивная модель CMYK (cyan-голубой, magenta-пурпурный, yellow-желтый, black-черный), используемая в печати.

В этом проекте вы будете работать с моделью RGB. Это означает, что цвета начинаются с черного и меняются по мере введения различных уровней красного, зеленого и синего. Простой способ увидеть это — использовать функцию CSS rgb.

Создайте новое правило CSS, нацеленное на класс container и установите для него background-color чёрный цвет с помощью rgb(0, 0, 0).

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker one"></div>
              <div class="marker two"></div>
              <div class="marker three"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .one {
          background-color: red;
        }

        .two {
          background-color: green;
        }

        .three {
          background-color: blue;
        }
