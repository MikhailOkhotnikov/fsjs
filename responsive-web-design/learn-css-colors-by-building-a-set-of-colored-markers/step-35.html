Шаг 35

Теперь, когда вы изучили все основные, вторичные и третичные цвета на цветовом круге, вам будет проще понять другие концепции теории цвета и то, как они влияют на дизайн.

Во-первых, в правилах CSS .one, .two и .three отрегулируйте значения в rgb функции так, чтобы background-color для каждого элемента был установлен чистый черный цвет. Помните, что функция rgb использует аддитивную цветовую модель, где цвета начинаются с черного и изменяются по мере увеличения значений красного, зеленого и синего.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <link rel="stylesheet" href="styles.css">
            <title>Цветные маркеры</title>      
          </head>
          <body>
            <h1>Цветные маркеры CSS</h1>
            <div class="container">
              <div class="marker one"></div>
              <div class="marker two"></div>
              <div class="marker three"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1 {
          text-align: center;
        }

        .marker {
          height: 25px;
          width: 200px;
          margin: 10px auto;
        }

        .container {
          background-color: rgb(255, 255, 255);
          padding: 10px 0;
        }

        .one {
          background-color: rgb(127, 255, 0);
        }

        .two {
          background-color: rgb(0, 127, 255);
        }

        .three {
          background-color: rgb(255, 0, 127);
        }
