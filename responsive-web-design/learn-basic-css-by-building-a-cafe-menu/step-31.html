Шаг 31

Ниже существующей пары «кофе/цена» добавьте следующие виды кофе и цены, используя article-элементы с двумя вложенными p-элементами внутри каждого. Как и прежде, текст первого p-элемента должен содержать название сорта кофе, а текст второго p-элемента — цену.

Пример кода

    
      Макиато с карамелью 375
      Тыквенная приправа 350
      Фундук 400
      Мокко 450
Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Меню кафе</title>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div class="menu">
              <main>
              <h1>КЕМПЕР КАФЕ</h1>
              <p>Основано в 2020</p>
              <section>
                <h2>Кофе</h2>
                <article>
                  <p>Французская ваниль</p>
                  <p>300</p>
                </article>
              </section>
              </main>
            </div>
          </body>
        </html>
