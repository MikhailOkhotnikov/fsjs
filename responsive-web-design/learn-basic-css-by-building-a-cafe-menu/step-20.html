Шаг 20

Элемент div используется в основном для оформления дизайна, в отличие от других элементов контента, которые вы использовали до сих пор. Добавьте элемент div внутрь элемента body, а затем переместите все остальные элементы внутрь нового div.

Внутри открывающего тега div добавьте атрибут id со значением menu.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Меню кафе</title>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <main>
              <h1>КЕМПЕР КАФЕ</h1>
              <p>Основано в 2020</p>
              <section>
                <h2>Кофе</h2>
              </section>
            </main>
          </body>
        </html>
Файл styles.css 
      
        h1, h2, p {
          text-align: center;
        }
        body {
          background-color: burlywood;
        }
