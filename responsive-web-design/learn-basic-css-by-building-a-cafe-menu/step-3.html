Шаг 3

meta — это один из нескольких элементов, которые предоставляют дополнительную информацию, не отображаемую на веб-странице, но полезную для поисковых систем или для отображения страницы.

Внутри элемента head вложите элемент meta с атрибутом charset со значением utf-8, чтобы сообщить браузеру, как кодировать символы на странице. 

    
      <!DOCTYPE html>
      <html lang="ru">
        <head>
          <title>Меню кафе</title>
        </head>
      </html>
