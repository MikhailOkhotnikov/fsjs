Шаг 1

В этом проекте вы познакомитесь с основами CSS (каскадных таблиц стилей), создав меню для кафе. CSS — это язык, используемый для оформления HTML-документов. Он описывает, как HTML-элементы должны отображаться на экране.

Как вы узнали из последних нескольких шагов в приложении «Фото с кошкой», для создания веб-страницы необходима базовая структура. В каждом HTML-документе должно быть DOCTYPE объявление и html элемент.DOCTYPE сообщает браузеру, в какой версии HTML находится документ. А html элемент представляет собой корневой элемент, содержащий все остальные элементы.

Пример кода

    
      <!DOCTYPE html>
      <html lang="en">
      <!--все остальные элементы находятся здесь-->
      </html>

Добавьте тег <!DOCTYPE html> и элемент html с атрибутом lang со значением ru.
