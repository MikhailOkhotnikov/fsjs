Теперь наша цель — сделать так, чтобы div не занимал всю ширину страницы. Для этого идеально подходит свойство CSS width.

Вы можете использовать id селектор для таргетинга на определенный элемент с id атрибутом. Селектор id определяется путем размещения символа хэша # непосредственно перед id значением элемента. Например, если элемент id имеет значение cat, то вы должны настроить таргетинг на этот элемент следующим образом:

Пример кода

    
      #cat {
        width: 250px;
      }

Используйте #menu селектор, чтобы придать вашему элементу ширину 300px.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Меню кафе</title>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div id="menu">
              <main>
              <h1>КЕМПЕР КАФЕ</h1>
              <p>Основано в 2020</p>
              <section>
                <h2>Кофе</h2>
              </section>
              </main>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1, h2, p {
          text-align: center;
        }
        body {
          background-color: burlywood;
        }
