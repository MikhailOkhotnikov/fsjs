Шаг 14

Вы стилизовали три элемента, написав CSS внутри тегов style. Это работает, но, поскольку стилей будет гораздо больше, лучше поместить все стили в отдельный файл и ссылаться на него.

Мы создали для вас отдельный файл styles.css и переключили редактор на этот файл. Вы можете переключаться между файлами с помощью вкладок в верхней части редактора.

Для начала перепишите созданные вами стили в файл styles.css. Не забудьте исключить открывающие и закрывающие теги style. 

    
      <!DOCTYPE html>
      <html lang="ru">
        <head>
          <title>Меню кафе</title>
          <meta charset="utf-8">
          <style>
            h1, h2, p {
              text-align: center;
            }
          </style>
        </head>
        <body>
          <main>
            <h1>КЕМПЕР КАФЕ</h1>
            <p>Основано в 2020</p>
            <section>
              <h2>Кофе</h2>
            </section>
          </main>
        </body>
      </html>
