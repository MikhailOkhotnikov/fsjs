Шаг 12

На предыдущем шаге вы использовали селектор типа для оформления элемента h1. Отцентрируйте содержимое элементов h2 и p, добавив новый селектор типа для каждого из них к существующему элементу style. 

    
      <!DOCTYPE html>
      <html lang="ru">
        <head>
          <title>Меню кафе</title>
          <meta charset="utf-8">
          <style>
            h1 {
              text-align: center;
            }
          </style>
        </head>
        <body>
          <main>
            <h1>КЕМПЕР КАФЕ</h1>
            <p>Основано в 2020</p>
            <section>
              <h2>Кофе</h2>
            </section>
          </main>
        </body>
      </html>
