Текст Меню кафе в меню занимает больше места сверху, чем адрес в нижней части меню. Это связано с тем, что в браузере по умолчанию есть отступ сверху для элемента h1.

Измените верхний отступ элемента h1 на 0, чтобы убрать весь верхний отступ.
