Шаг 15

Теперь, когда у вас есть CSS в файле styles.css, удалите элемент style и всё его содержимое. После этого текст, который был центрирован, сместится влево.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Меню кафе</title>
            <meta charset="utf-8">
            <style>
              h1, h2, p {
                text-align: center;
              }
            </style>
          </head>
          <body>
            <main>
              <h1>КЕМПЕР КАФЕ</h1>
              <p>Основано в 2020</p>
              <section>
                <h2>Кофе</h2>
              </section>
            </main>
          </body>
        </html>
Файл styles.css 
      
        h1, h2, p {
          text-align: center;
        }
