article элементы обычно содержат несколько элементов со связанной информацией. В данном случае это будет название сорта кофе и цена за этот сорт. Разместите два p элемента внутри article элемента. Текст первого элемента должен быть Французская ваниль, а текст второго 300.00.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Меню кафе</title>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div class="menu">
              <main>
              <h1>КЕМПЕР КАФЕ</h1>
              <p>Основано в 2020</p>
              <section>
                <h2>Кофе</h2>
                <article>

                </article>
              </section>
              </main>
            </div>
          </body>
        </html>
