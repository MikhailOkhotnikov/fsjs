Чтобы сообщить посетителям, что кафе было основано в 2020 году, добавьте элемент p под элементом h1 с текстом Основано в 2020. 

    
      <!DOCTYPE html>
      <html lang="ru">
        <head>
          <title>Меню кафе</title>
          <meta charset="utf-8">
        </head>
        <body>
          <main>
            <h1>КЕМПЕР КАФЕ</h1>
          </main>
        </body>
      </html>
