Шаг 16

Теперь вам нужно связать файл styles.css со стилями, чтобы они применялись снова. Внутри элемента head добавьте элемент link. Присвойте ему атрибут rel со значением "stylesheet" и атрибут href со значением "styles.css".

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Меню кафе</title>
            <meta charset="utf-8">
          </head>
          <body>
            <main>
              <h1>КЕМПЕР КАФЕ</h1>
              <p>Основано в 2020</p>
              <section>
                <h2>Кофе</h2>
              </section>
            </main>
          </body>
        </html>
Файл styles.css 
      
        h1, h2, p {
          text-align: center;
        }
