Шаг 17

Чтобы стиль страницы на мобильном устройстве выглядел так же, как на компьютере или ноутбуке, вам нужно добавить элемент meta со специальным атрибутом content.

Добавьте следующее в head элемент:

Пример кода

    
      <meta name="viewport" content="width=device-width, initial-scale=1.0" />
Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Меню кафе</title>
            <meta charset="utf-8">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <main>
              <h1>КЕМПЕР КАФЕ</h1>
              <p>Основано в 2020</p>
              <section>
                <h2>Кофе</h2>
              </section>
            </main>
          </body>
        </html>
Файл styles.css 
      
        h1, h2, p {
          text-align: center;
        }
