Вы можете добавить стиль к элементу, указав его в элементе style и задав для него свойство следующим образом:

Пример кода

    
      element {
        property: value;
      }

Центрируйте содержимое элемента h1 с помощью свойства text-align со значением center.

    
      <!DOCTYPE html>
      <html lang="ru">
        <head>
          <title>Меню кафе</title>
          <meta charset="utf-8">
          <style>

          </style>
        </head>
        <body>
          <main>
            <h1>КЕМПЕР КАФЕ</h1>
            <p>Основано в 2020</p>
            <section>
              <h2>Кофе</h2>
            </section>
          </main>
        </body>
      </html>
