Что ж, это не сработало. При оформлении элементов p как inline-block и размещении их на отдельных строках в коде справа от первого элемента p появляется дополнительное пространство, из-за чего второй элемент смещается на следующую строку. Эта проблема может возникать и из-за пробелов.

Один из способов исправить это — сделать ширину каждого элемента p немного меньше, чем 50%. Измените значение width на 49% для каждого класса и посмотрите, что произойдёт.

          </head>
          <body>

Файл styles.css 
      
        h1, h2, p {
          text-align: center;
        }

        body {
          background-image: url(https://cdn.freecodecamp.org/curriculum/css-cafe/beans.jpg);
        }

        .menu {
          width: 80%;
          background-color: burlywood;
          margin-left: auto;
          margin-right: auto;
        }

        .flavor {
          text-align: left;
          width: 50%;
        }
        
        .price {
          text-align: right;
          width: 50%;
        }

        .item p {

        }
