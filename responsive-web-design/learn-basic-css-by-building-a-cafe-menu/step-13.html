Шаг 13

Теперь у вас есть три селектора типов с одинаковым оформлением. Вы можете добавить одну и ту же группу стилей ко многим элементам, создав список селекторов. Каждый селектор отделяется от другого запятыми, например:

Пример кода

    
      selector1, selector2 {
        property: value;
      }

Удалите три существующих селектора типов и замените их одним списком селекторов, который центрирует текст для элементов h1, h2, и p. 

    
      <!DOCTYPE html>
      <html lang="ru">
        <head>
          <title>Меню кафе</title>
          <meta charset="utf-8">
          <style>
            h1 {
              text-align: center;
            }
            h2 {
              text-align: center;
            }
            p {
              text-align: center;
            }
          </style>
        </head>
        <body>
          <main>
            <h1>КЕМПЕР КАФЕ</h1>
            <p>Основано в 2020</p>
            <section>
              <h2>Кофе</h2>
            </section>
          </main>
        </body>
      </html>
