Шаг 36

Это примерно то, что вам нужно, но теперь было бы неплохо, если бы вкус и цена располагались на одной строке. p элементы являются блочными элементами, поэтому они занимают всю ширину родительского элемента.

Чтобы они оказались на одной строке, нужно применить некоторые стили к элементам p так, чтобы они больше походили на inline-элементы. Для этого начните с добавления атрибута class со значением item к первому элементу article под заголовком Кофе.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Меню кафе</title>
            <meta charset="utf-8">
            <meta name="viewport" content="width=device-width, initial-scale=1.0" />
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div class="menu">
              <main>
              <h1>КЕМПЕР КАФЕ</h1>
              <p>Основано в 2020</p>
              <section>
                <h2>Кофе</h2>
                <article>
                  <p class="flavor">Французская ваниль</p>
                  <p class="price">300</p>
                </article>
                <article>
                  <p>Макиато с карамелью</p>
                  <p>375</p>
                </article>
                <article>
                  <p>Тыквенная приправа</p>
                  <p>350</p>
                </article>
                <article>
                  <p>Фундук</p>
                  <p>400</p>
                </article>
                <article>
                  <p>Мокко</p>
                  <p>450</p>
                </article>
              </section>
              </main>
            </div>
          </body>
        </html>
Файл styles.css 
      
        h1, h2, p {
          text-align: center;
        }

        body {
          background-image: url(https://cdn.freecodecamp.org/curriculum/css-cafe/beans.jpg);
        }

        .menu {
          width: 80%;
          background-color: burlywood;
          margin-left: auto;
          margin-right: auto;
        }

        .flavor {
          text-align: left;
        }
        
        .price {
          text-align: right;
        }
