Теперь .one центрирован по горизонтали, но его верхнее поле выходит за пределы холста и попадает на границу фрейма, смещая весь холст на 20 пикселей вниз.

Добавьте внутренний отступ(padding) в 1px к элементу .canvas, чтобы дать элементу .one что-то твердое, от чего можно было бы отталкиваться.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
            <title>Картина Ротко</title>
          </head>
          <body>
            <div class="frame">
              <div class="canvas">
                <div class="one"></div>
              </div>
            </div>
          </body>
        </html>
Файл styles.css 
    
      .canvas {
        width: 500px;
        height: 600px;
        background-color: #4d0f00;
      }
      .frame {
        border: solid black 50px;
        width: 500px;
        padding: 50px;
        margin: 20px auto;
      }
      .one {
        width: 425px;
        height: 150px;
        background-color: #efb762;
        margin: 20px auto;
      }
