Шаг 14

Используйте внутренние отступы для регулировки интервала внутри элемента.

В .frame используйте свойство сокращения padding, чтобы увеличить пространство между элементами .frame и .canvas на 50px. Сокращение увеличит пространство сверху, снизу, слева и справа от границы элемента и холста внутри.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
            <title>Картина Ротко</title>
          </head>
          <body>
            <div class="frame">
              <div class="canvas"></div>
            </div>
          </body>
        </html>
Файл styles.css 
      
        .canvas {
          width: 500px;
          height: 600px;
          background-color: #4d0f00;
        }
        .frame {
          border: solid black 50px;
          width: 50px;
        }
