Шаг 15

Используйте margin для регулировки отступов снаружи элемента.

Используя свойство margin, задайте элементу .frame вертикальное поле 20px и горизонтальное поле auto. Это сместит фрейм вниз на 20 пикселей и горизонтально отцентрирует его на странице.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
            <title>Картина Ротко</title>
          </head>
          <body>
            <div class="frame">
              <div class="canvas"></div>
            </div>
          </body>
        </html>
Файл styles.css 
    
      .canvas {
        width: 500px;
        height: 600px;
        background-color: #4d0f00;
      }
      .frame {
        border: solid black 50px;
        width: 50px;
        padding: 50px;
      }
