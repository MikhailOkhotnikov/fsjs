Пришло время для CSS.

Даже если в вашем div нет текста, он всё равно обрабатывается как блок с содержимым. Напишите правило CSS, использующее селектор класса .canvas и устанавливающее его width на 500 пикселей. Вот правило CSS, устанавливающее ширину класса card на 300 пикселей:

Пример кода

    
      .card {
        width: 300px;
      }
Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
            <title>Картина Ротко</title>
          </head>
          <body>
            <div class="canvas"></div>
          </body>
        </html>
