Внешний отступ(margin) — это область за пределами блока, и ее можно использовать для управления пространством между другими блоками или элементами.

На изображении нижний элемент имеет больший верхний внешний отступ(margin), что сдвигает его дальше вниз по странице.

Теперь, когда вы понимаете блочную модель CSS, давайте начнем с картины Ротко.

Удалите элемент img.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <title>Картина Ротко</title>
          </head>
          <body>
            <img src="https://cdn.freecodecamp.org/curriculum/css-box-model/diagram-3.png">
          </body>
        </html>
