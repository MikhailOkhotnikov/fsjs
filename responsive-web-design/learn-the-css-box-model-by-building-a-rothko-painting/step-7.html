Прежде, чем вы сможете приступить к оформлению добавленных вами div элементов, вам нужно связать CSS с HTML.

Добавьте элемент link для ссылки на ваш файл styles.css. Установите href равным styles.css и не забудьте установить атрибут rel равным stylesheet.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <title>Картина Ротко</title>
          </head>
          <body>
            <div class="canvas"></div>
          </body>
        </html>
