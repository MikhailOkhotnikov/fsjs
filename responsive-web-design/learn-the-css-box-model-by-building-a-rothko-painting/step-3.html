Шаг 3

В блочной модели CSS каждый элемент HTML рассматривается как блок с четырьмя областями.

Представьте, что вы получаете коробку от вашего любимого интернет-магазина. Содержимое — это товар в коробке или, в нашем случае, заголовок, абзац или элемент изображения.

Измените атрибут src в img с https://cdn.freecodecamp.org/curriculum/css-box-model/diagram-1.png на https://cdn.freecodecamp.org/curriculum/css-box-model/diagram-2.png.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <title>Картина Ротко</title>
          </head>
          <body>
            <img src="https://cdn.freecodecamp.org/curriculum/css-box-model/diagram-1.png">
          </body>
        </html>
