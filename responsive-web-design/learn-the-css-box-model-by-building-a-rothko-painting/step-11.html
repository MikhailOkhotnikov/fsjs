Шаг 11

Каждой картине нужна рамка.

Оберните элемент .canvas в другой div. Присвойте этому div класс frame.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
            <title>Картина Ротко</title>
          </head>
          <body>
            <div class="canvas"></div>
          </body>
        </html>
Файл styles.css 
      
        .canvas {
          width: 500px;
          height: 600px;
          background-color: #4d0f00;
        }
