Вы можете создать ссылку на другую страницу с помощью элемента anchor (a).

Вот пример ссылки на https://www.freecodecamp.org:

Пример кода

    <a href="https://www.freecodecamp.org"></a>

Добавьте элемент anchor после абзаца, со ссылкой на https://freecatphotoapp.com. На этом этапе ссылка не будет отображаться в предварительном просмотре. 

    
      <html>
        <body>
          <main>
            <h1>CatPhotoApp</h1>
            <h2>Фотографии кошек</h2>
            <!-- Задача: Добавить ссылку на фотографии кошек -->
            <p>Смотрите больше фотографий кошек в нашей галерее.</p>
            <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Симпатичный оранжевый кот, лежащий на спине">         
          </main>
        </body>
      </html>
