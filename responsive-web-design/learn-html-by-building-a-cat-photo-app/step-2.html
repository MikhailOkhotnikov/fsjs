Шаг 2

Элементы заголовков от h1 до h6 используются для обозначения важности контента под ними. Чем меньше число, тем выше важность, поэтому h2 элементы имеют меньшую важность, чем h1 элементы.

Пример кода

<h1>самый важный элемент заголовка</h1>
<h2>второй по важности элемент заголовка</h2>
<h3>третий по важности элемент заголовка</h3>
<h4>четвертый по важности элемент заголовка</h4>
<h5>пятый по важности элемент заголовка</h5>
<h6>наименее важный элемент заголовка</h6>

Используйте только один h1 элемент на странице и размещайте заголовки меньшей важности под заголовками большей важности.

Под h1 элементом добавьте h2 элемент с таким текстом:

Cat Photos 
    
      <html></html>
        <body>
          <h1>CatPhotoApp</h1>
        </body>
      </html>
