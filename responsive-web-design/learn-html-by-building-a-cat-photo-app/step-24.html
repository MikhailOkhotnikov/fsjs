Элемент подписи к рисунку (figcaption) используется для добавления подписи, описывающей изображение, содержащееся в элементе figure.

Вот пример figcaption элемента с подписью Милый котик:

Пример кода

    
      <figure>
        <img src="image.jpg" alt="Описание изображения">
        <figcaption>Милый котик</figcaption>
      </figure>

После изображения, вложенного в элемент figure, добавьте элемент figcaption с текстом:
