Шаг 15

На предыдущих этапах вы использовали элемент привязки, чтобы превратить текст в ссылку. Другие типы контента также можно превратить в ссылки, заключив их в теги привязки.

Вот пример превращения изображения в ссылку:

Пример кода

    
      <a href="example-link">
        <img src="image-link.jpg" alt="Фотография кошки.">
      </a>

Превратите изображение в ссылку, окружив его необходимыми тегами элементов. Используйте https://freecatphotoapp.com в качестве значения атрибута href ссылки. 

    
      <html>
        <body>
          <main>
            <h1>CatPhotoApp</h1>
            <h2>Фотографии кошек</h2>
            <!-- Задача: Добавить ссылку на фотографии кошек -->
            <p>Смотрите больше <a target="_blank" href="https://freecatphotoapp.com">фотографий кошек</a> в нашей галерее.</p>
            <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Симпатичный оранжевый кот, лежащий на спине">         
          </main>
        </body>
      </html>
