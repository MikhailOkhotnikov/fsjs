Изучите HTML, создав приложение для фотографий кошек

HTML-теги придают веб-странице ее структуру. Вы можете использовать HTML-теги для добавления фотографий, кнопок и других элементов на вашу веб-страницу.

На этом курсе вы изучите наиболее распространенные HTML-теги, создав собственное приложение для фотографий кошек.
