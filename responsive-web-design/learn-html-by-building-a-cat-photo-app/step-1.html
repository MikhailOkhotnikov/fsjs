Шаг 1

HTML-элементы содержат открывающий и закрывающий тег с содержимым между ними.

Вот базовый синтаксис:

Пример кода

<openingTag>Контент</closingTag>

Первый элемент, о котором вы узнаете, — это элемент h1. Элемент h1 — это элемент заголовка, который используется для основного заголовка веб-страницы.

Пример кода

<h1>Это основной заголовок</h1>

Измените текст элемента h1 ниже с Hello World на CatPhotoApp и посмотрите, как он изменится в режиме предварительного просмотра в браузере. 

    
      <html></html>
        <body>
          <h1>Hello World</h1>
        </body>
      </html>
