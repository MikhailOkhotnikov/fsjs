Чтобы открывать ссылки на новой вкладке, вы можете использовать атрибут target элемента anchor (a).

Атрибут target указывает, где открывать связанный документ. target="_blank" открывает связанный документ в новой вкладке или окне.

Вот базовый синтаксис для a элемента с target атрибутом:

Пример кода

    <a href="https://www.freecodecamp.org" target="_blank">freeCodeCamp</a>

Добавьте атрибут target со значением _blank к открывающему тегу элемента anchor (a), чтобы ссылка открывалась на новой вкладке. 

    
      <html>
        <body>
          <main>
            <h1>CatPhotoApp</h1>
            <h2>Фотографии кошек</h2>
            <!-- Задача: Добавить ссылку на фотографии кошек -->
            <p>Смотрите больше <a href="https://freecatphotoapp.com">фотографий кошек</a> в нашей галерее.</p>
            <img src="https://cdn.freecodecamp.org/curriculum/cat-photo-app/relaxing-cat.jpg" alt="Симпатичный оранжевый кот, лежащий на спине">         
          </main>
        </body>
      </html>
