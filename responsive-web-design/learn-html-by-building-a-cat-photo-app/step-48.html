Шаг 48

Обратите внимание, что обе кнопки-переключателя могут быть выбраны одновременно. Чтобы при выборе одной кнопки-переключателя другая автоматически отключалась, обе кнопки должны иметь атрибут name с одинаковым значением.

Вот пример двух переключателей с одинаковым name атрибутом:

Пример кода

    
      <input type="radio" name="meal"> Завтрак
      <input type="radio" name="meal"> Обед

Добавьте name атрибут со значением indoor-outdoor к обоим переключателям.
