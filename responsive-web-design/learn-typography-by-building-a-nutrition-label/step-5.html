Шаг 5

Создайте селектор body и задайте для него font-family равным Open Sans с резервным вариантом sans-serif.

Помните, что шрифты, в названии которых есть пробелы, необходимо заключать в кавычки для CSS.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <title>Пищевая этикетка</title>
            <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Open+Sans:400,700,800">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <h1>Пищевая ценность</h1>
            <p>8 порций в контейнере</p>
            <p>Размер порции 2/3 стакана (55 г)</p>
          </body>
        </html>
