Шаг 20

Ваш новый класс пока не имеет никакого стиля. Создайте селектор .bold и задайте ему свойство font-weight, установленное на 800, чтобы сделать текст жирным.

Продолжайте и удалите свойство font-weight из селектора h1.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <title>Пищевая этикетка</title>
            <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Open+Sans:400,700,800">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div class="label">
              <h1>Пищевая ценность</h1>
              <div class="divider"></div>
              <p>8 порций в контейнере</p>
              <p class="bold">Размер порции 2/3 стакана (55 г)</p>
            </div>
          </body>
        </html>
Файл styles.css 
      
        * {
          box-sizing: border-box;
        }

        html {
          font-size: 16px;
        }

        body {
          font-family: "Open Sans", sans-serif;
        }

        .label {
          border: 2px solid black;
          width: 270px;
          margin: 20px auto;
          padding: 0 7px;
        }

        h1 {
          font-weight: 800;
          text-align: center;
          margin: -4px 0;
          letter-spacing: 0.15px;
        }

        p {
          margin: 0;
        }

        .divider {
          border-bottom: 1px solid #888989;
          margin: 2px 0;
        }
