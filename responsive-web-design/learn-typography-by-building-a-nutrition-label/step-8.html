Шаг 8

Границы можно использовать для группировки и расстановки приоритетов контента.

Создайте селектор .label и задайте для него border равной 2px solid black.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <title>Пищевая этикетка</title>
            <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Open+Sans:400,700,800">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div class="label">
              <h1>Пищевая ценность</h1>
              <p>8 порций в контейнере</p>
              <p>Размер порции 2/3 стакана (55 г)</p>
            </div>
          </body>
        </html>
Файл styles.css 
      
        html {
          font-size: 16px;
        }

        body {
          font-family: "Open Sans", sans-serif;
        }
