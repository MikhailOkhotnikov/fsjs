Изучите типографику, создав этикетку для продуктов питания

Типографика — это искусство оформления текста таким образом, чтобы он был легко читаемым и соответствовал своему назначению.

В этом курсе вы будете использовать типографику для создания веб-страницы с этикетками продуктов питания. Вы узнаете, как стилизовать текст, настроить высоту строки и позиционировать текст с помощью CSS.
