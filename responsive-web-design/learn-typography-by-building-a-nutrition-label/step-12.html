Шаг 12

Помните, что использование тегов h1, h2 и подобных определяет семантическую структуру вашего HTML. Однако вы можете настроить CSS этих элементов, чтобы контролировать визуальный поток и иерархию.

Создайте правило h1 и установите свойство font-weight на 800. Это сделает текст h1 более жирным.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <title>Пищевая этикетка</title>
            <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Open+Sans:400,700,800">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div class="label">
              <h1>Пищевая ценность</h1>
              <p>8 порций в контейнере</p>
              <p>Размер порции 2/3 стакана (55 г)</p>
            </div>
          </body>
        </html>
Файл styles.css 
      
        * {
          box-sizing: border-box;
        }

        html {
          font-size: 16px;
        }

        body {
          font-family: "Open Sans", sans-serif;
        }

        .label {
          border: 2px solid black;
          width: 270px;
          margin: 20px auto;
          padding: 0 7px;
        }
