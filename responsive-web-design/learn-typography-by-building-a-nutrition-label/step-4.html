Шаг 4

В элементе head добавьте элемент link с атрибутом rel, установленным на stylesheet, и атрибутом href, установленным на https://fonts.googleapis.com/css?family=Open+Sans:400,700,800.

Это позволит импортировать семейство шрифтов Open Sans со значениями насыщенности шрифта 400, 700 и 800.

Также добавьте элемент link для ссылки на ваш файл styles.css.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <title>Пищевая этикетка</title>
          </head>
          <body>
            <h1>Пищевая ценность</h1>
            <p>8 порций в контейнере</p>
            <p>Размер порции 2/3 стакана (55 г)</p>
          </body>
        </html>
