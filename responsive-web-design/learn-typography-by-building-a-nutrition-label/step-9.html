Шаг 9

Грамотное использование пустого пространства может привлечь внимание к важным элементам вашей страницы и помочь направить взгляд пользователя по тексту.

Задайте для селектора .label свойство width, равное 270px.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <meta charset="UTF-8">
            <title>Пищевая этикетка</title>
            <link rel="stylesheet" href="https://fonts.googleapis.com/css?family=Open+Sans:400,700,800">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <div class="label">
              <h1>Пищевая ценность</h1>
              <p>8 порций в контейнере</p>
              <p>Размер порции 2/3 стакана (55 г)</p>
            </div>
          </body>
        </html>
Файл styles.css 
      
        html {
          font-size: 16px;
        }

        body {
          font-family: "Open Sans", sans-serif;
        }

        .label {
          border: 2px solid black;
        }
