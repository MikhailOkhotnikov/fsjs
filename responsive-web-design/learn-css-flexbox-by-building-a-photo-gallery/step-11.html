Шаг 11

Теперь ваши изображения слишком большие.

Создайте селектор .gallery img для их нацеливания. Дайте им всю width в 100% и max-width 350px.

Также установите свойство height на 300px, чтобы ваши изображения были единообразными размерами.

          </head>
