Шаг 24

Атрибут alt элемента image должен описывать содержимое изображения. Программы чтения с экрана объявляют альтернативный текст вместо изображений. Если изображение не может быть загружено, этот текст отображается вместо изображения.

Чтобы завершить проект, добавьте атрибут alt ко всем девяти вашим изображениям кошек, чтобы описать их. Используйте значение длиной не менее пяти символов для каждого.
