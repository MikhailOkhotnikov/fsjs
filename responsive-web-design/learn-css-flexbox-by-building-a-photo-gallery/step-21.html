Между изображениями должно быть немного места.

Сокращенное свойство CSS gap задает зазоры, также известные как желоба, между строками и столбцами. Свойство gap и его подсвойства row-gap и column-gap предоставляют эту функциональность для гибкой, сетчатой ​​и многоколоночной компоновки. Вы применяете свойство к элементу-контейнеру.

Задайте для вашего гибкого контейнера .gallery свойство gap со значением 16px.
