Псевдоэлемент ::after создает элемент, который является последним дочерним элементом выбранного элемента. Вы можете использовать его для добавления пустого элемента после последнего изображения. Если вы зададите ему ту же ширину(width), что и изображения, он сдвинет последнее изображение влево, когда галерея находится в двухколоночном макете. Сейчас он находится в центре, потому что вы установили justify-content: center для flex-контейнера.

Пример:

    
      .container::after {
        content: "";
        width: 860px;
      }
