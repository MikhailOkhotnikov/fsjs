Шаг 8

Обратите внимание, как синяя граница изображения выходит за пределы красной границы галереи. Это связано с тем, как браузеры вычисляют размер элементов контейнера.

Для установки этого поведения используется свойство box-sizing. По умолчанию используется модель content-box. При использовании этой модели, когда элемент имеет определенную ширину, эта ширина вычисляется только на основе содержимого элемента. Значения padding и border добавляются к общей ширине, поэтому элемент увеличивается, чтобы вместить эти значения.

Попробуйте явно задать box-sizing для content-box с помощью глобального селектора *. На этом этапе вы не увидите никаких изменений, поскольку используете значение по умолчанию.
