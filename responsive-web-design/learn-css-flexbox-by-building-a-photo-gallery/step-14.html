Шаг 14

Flexbox — это одномерный CSS-макет, который позволяет управлять размещением и выравниванием элементов внутри контейнера.

Чтобы использовать его, задайте элементу свойство display равное flex. Это сделает элемент flex-контейнером. Любые прямые потомки flex-контейнера называются flex-элементами.

Создайте селектор .gallery и сделайте его flex-контейнером.

        <!DOCTYPE html>

            <title>Фотогалерея</title>
          </head>
