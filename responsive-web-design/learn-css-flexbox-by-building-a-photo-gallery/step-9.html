Шаг 9

Модель размера border-box делает противоположность content-box. Общая ширина элемента, включая padding и border, будет явно заданной шириной. Содержимое элемента сожмется, чтобы освободить место для padding и border.

Измените свойство box-sizing на border-box. Обратите внимание, как теперь синие границы изображения вписываются в красную границу галереи.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="en">
          <head>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
            <meta name="viewport" content="width=device-width, initial-scale=1.0">
            <title>Фотогалерея</title>
          </head>
          <body>
            <header class="header">
              <h1>фотогалерея css flexbox</h1>

Файл styles.css 
      
        * {
          box-sizing: content-box;
        }
        .gallery {
          width: 50%;
          border: 5px solid red;
        }
        img {
          width: 100%;
          padding: 5px;
          border: 5 px solid blue;
        }
