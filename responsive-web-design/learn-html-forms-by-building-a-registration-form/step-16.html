Единица измерения rem относится к корневому элементу em и зависит от размера шрифта элемента html.

Поскольку элементы label по умолчанию отображаются в одну строку, они все отображаются рядом друг с другом на одной строке, что затрудняет чтение текста. Чтобы они отображались на отдельных строках, добавьте display: block к элементу label и добавьте margin 0.5rem 0, чтобы отделить их друг от друга.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Registration Form</title>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <h1>Форма регистрации</h1>
            <p>Пожалуйста, заполните эту форму необходимой информацией</p>
            <form action="https://register-demo.freecodecamp.org" method="post">
              <fieldset>
                <label>Введите свое имя:</label>
                <label>Введите свою фамилию:</label>
                <label>Введите свой Email:</label>
                <label>Создайте новый пароль:</label>
              </fieldset>
              <fieldset></fieldset>
              <fieldset></fieldset>
            </form>
          </body>
        </html>
Файл styles.css 
      
        body {
          width: 100%;
          height: 100vh;
          margin: 0;
          background-color: #1b1b32;
          color: #f5f6f7;
        }
