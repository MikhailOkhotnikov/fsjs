Шаг 12

Атрибут method указывает, как отправлять данные формы по URL-адресу, указанному в action атрибуте. Данные формы могут быть отправлены через GET запрос в виде параметров URL (с помощью method="get") или через POST запрос в виде данных в теле запроса (с помощью method="post").

Установите method атрибут для отправки данных вашей формы через POST запрос.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Registration Form</title>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <h1>Форма регистрации</h1>
            <p>Пожалуйста, заполните эту форму необходимой информацией</p>
            <form action="https://register-demo.freecodecamp.org"></form>
          </body>
        </html>
Файл styles.css 
      
        body {
          width: 100%;
          height: 100vh;
          margin: 0;
          background-color: #1b1b32;
          color: #f5f6f7;
        }
