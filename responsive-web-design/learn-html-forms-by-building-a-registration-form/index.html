Изучите HTML-формы, создав регистрационную форму.

Вы можете использовать HTML-формы для сбора информации от людей, посещающих вашу веб-страницу.

В этом курсе вы изучите HTML-формы, создав страницу регистрации. Вы узнаете, как контролировать типы данных, которые люди могут вводить в вашу форму, а также некоторые новые инструменты CSS для стилизации вашей страницы.
