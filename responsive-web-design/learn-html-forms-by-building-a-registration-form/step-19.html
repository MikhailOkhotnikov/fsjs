Шаг 19

Указание атрибута type элемента формы важно для того, чтобы браузер знал, какие данные он должен ожидать. Если type не указан, браузер по умолчанию использует text.

Присвойте первым двум элементам input атрибут type с значением text, третьему - атрибут type с значением email, а четвертому - атрибут type с значением password.

Тип email позволяет использовать только электронные письма с @ и . в домене. Тип password скрывает ввод и предупреждает, если сайт не использует HTTPS.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Registration Form</title>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <h1>Форма регистрации</h1>
            <p>Пожалуйста, заполните эту форму необходимой информацией</p>
            <form action="https://register-demo.freecodecamp.org" method="post">
              <fieldset>
                <label for="first-name">Введите свое имя: <input id="first-name"/></label>
                <label for="last-name">Введите свою фамилию: <input id="last-name"/></label>
                <label for="email">Введите свой Email: <input id="email"/></label>
                <label for="new-password">Создайте новый пароль: <input id="new-password"/></label>
              </fieldset>
              <fieldset></fieldset>
              <fieldset></fieldset>
            </form>
          </body>
        </html>
