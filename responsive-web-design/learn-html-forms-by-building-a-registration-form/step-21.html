Шаг 21

На этом этапе вы должны иметь возможность отправить форму. Однако вы можете заметить, что ничего не происходит.

Чтобы сделать форму более интерактивной, добавьте атрибут required к элементам input в первом fieldset.

Теперь, если вы попытаетесь отправить форму, не заполнив обязательные поля, вы увидите сообщение об ошибке.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Registration Form</title>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <h1>Форма регистрации</h1>
            <p>Пожалуйста, заполните эту форму необходимой информацией</p>
            <form action="https://register-demo.freecodecamp.org" method="post">
              <fieldset>
                <label for="first-name">Введите свое имя: <input id="first-name" type="text" /></label>
                <label for="last-name">Введите свою фамилию: <input id="last-name" type="text" /></label>
                <label for="email">Введите свой Email: <input id="email" type="email" /></label>
                <label for="new-password">Создайте новый пароль: <input id="new-password" type="password" /></label>
              </fieldset>
              <fieldset></fieldset>
              <fieldset></fieldset>
              <input type="submit" value="Отправить" />
            </form>
          </body>
        </html>

        body {
