Шаг 27

Вы хотите, чтобы одновременно можно было выбрать только один радиокнопку. Однако форма не знает, что радиокнопки связаны между собой.

Чтобы связать переключатели, присвойте им один и тот же атрибут name со значением account-type. Теперь нельзя будет одновременно выбрать оба переключателя.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Registration Form</title>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <h1>Форма регистрации</h1>
            <p>Пожалуйста, заполните эту форму необходимой информацией</p>
            <form action="https://register-demo.freecodecamp.org" method="post">
              <fieldset>
                <label for="first-name">Введите свое имя: <input id="first-name" type="text" required /></label>
                <label for="last-name">Введите свою фамилию: <input id="last-name" type="text" required /></label>
                <label for="email">Введите свой Email: <input id="email" type="email" required /></label>
                <label for="new-password">Создайте новый пароль: <input id="new-password" type="password" required pattern="[a-z0-5]{8,}" /></label>
              </fieldset>
              <fieldset>
                <label><input type="radio" /> Личный</label>
                <label><input type="radio" /> Деловой</label>
              </fieldset>
              <fieldset></fieldset>
              <input type="submit" value="Отправить" />
            </form>
          </body>
        </html>
