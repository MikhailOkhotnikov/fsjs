С помощью type="password" вы можете использовать атрибут pattern для определения регулярного выражения, которому пароль должен соответствовать, чтобы считаться действительным.

Добавьте pattern атрибут к input элементу password, чтобы требовать совпадения входных данных: [a-z0-5]{8,}

Выше приведено регулярное выражение, которое соответствует восьми или более строчным буквам или цифрам от 0 до 5. Затем удалите атрибут minlength и попробуйте.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Registration Form</title>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <h1>Форма регистрации</h1>
            <p>Пожалуйста, заполните эту форму необходимой информацией</p>
            <form action="https://register-demo.freecodecamp.org" method="post">
              <fieldset>
                <label for="first-name">Введите свое имя: <input id="first-name" type="text" required /></label>
                <label for="last-name">Введите свою фамилию: <input id="last-name" type="text" required /></label>
                <label for="email">Введите свой Email: <input id="email" type="email" required /></label>
                <label for="new-password">Создайте новый пароль: <input id="new-password" type="password" required minlength="8" /></label>
              </fieldset>
              <fieldset></fieldset>
              <fieldset></fieldset>
              <input type="submit" value="Отправить" />
            </form>
          </body>
        </html>
