Шаг 38

Элемент textarea действует как элемент input типа text, но обладает дополнительным преимуществом: он может принимать многострочный текст и задавать начальное количество строк и столбцов.

Пользователи смогут зарегистрироваться, указав биографию. Добавьте label с текстом Предоставьте биографию: в конце fieldset. Добавьте элемент textarea внутри элемента label. Обратите внимание, что textarea требует закрывающего тега.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Registration Form</title>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <h1>Форма регистрации</h1>
            <p>Пожалуйста, заполните эту форму необходимой информацией</p>
            <form action="https://register-demo.freecodecamp.org" method="post">
              <fieldset>
                <label for="first-name">Введите свое имя: <input id="first-name" type="text" required /></label>
                <label for="last-name">Введите свою фамилию: <input id="last-name" type="text" required /></label>
                <label for="email">Введите свой Email: <input id="email" type="email" required /></label>
                <label for="new-password">Создайте новый пароль: <input id="new-password" type="password" required pattern="[a-z0-5]{8,}" /></label>
              </fieldset>
              <fieldset>
                <legend>Тип учетной записи (обязательно)</legend>
                <label for="personal-account"><input type="radio" name="account-type" id="personal-account" checked /> Личный</label>
                <label for="business-account"><input type="radio" name="account-type" id="business-account" /> Деловой</label>
              </fieldset>
              <fieldset>
                <label>Загрузите фотографию профиля: <input type="file" /></label>
                <label>Введите свой возраст (лет): <input type="number" min="13" max="120" /></label>
                <label>Как вы узнали о нас?
                  <select>
                    <option value="">(выберите один)</option>
                    <option value="1">Новости freeCodeCamp</option>
                    <option value="2">Канал  freeCodeCamp на YouTube</option>
                    <option value="3">Форум freeCodeCamp</option>
                    <option value="4">Другой</option>
                  </select>
                </label>
              </fieldset>
              <label for="terms-and-conditions"><input type="checkbox" id="terms-and-conditions" required />Я принимаю <a href="https://www.freecodecamp.org/news/terms-of-service/">правила и условия</a></label>
              <input type="submit" value="Отправить" />
            </form>
          </body>
        </html>
Файл styles.css 
      
        body {
          width: 100%;
          height: 100vh;
          margin: 0;
          background-color: #1b1b32;
          color: #f5f6f7;
        }

        label {
          display: block;
          margin: 0.5rem 0;
        }
