Шаг 11

Как следует из заголовка, вы создаете форму. Поэтому после элемента p вставьте form с атрибутом action, нацеленным на https://register-demo.freecodecamp.org.

Файл index.html 
      
        <!DOCTYPE html>
        <html lang="ru">
          <head>
            <title>Registration Form</title>
            <meta charset="UTF-8">
            <link rel="stylesheet" href="styles.css">
          </head>
          <body>
            <h1>Форма регистрации</h1>
            <p>Пожалуйста, заполните эту форму необходимой информацией</p>
          </body>
        </html>
Файл styles.css 
      
        body {
          width: 100%;
          height: 100vh;
          margin: 0;
          background-color: #1b1b32;
          color: #f5f6f7;
        }
